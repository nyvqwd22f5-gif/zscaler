# API Response Codes and Error Messages
Source: https://help.zscaler.com/easm/api-response-codes-and-error-messages
PDF: https://help.zscaler.com/pdf/com/en/1533591.pdf

The following table lists the HTTP status codes returned by the API.
| Code | Description |
| ---- | ----------- |
| 200 | Successful |
| 400 | Bad Request (e.g., incorrect filter keys are used in the URL) |
| 401 | Unauthorized (e.g., an incorrect token is used) |
| 404 | Resource Not Found |
| 500 | Internal Server Error |
