# Editing Network Connector Provisioning Keys
Source: https://help.zscaler.com/unified/editing-network-connector-provisioning-keys
PDF: https://help.zscaler.com/pdf/com/en/1532376.pdf

This article describes how to edit a Network Connector provisioning key. To learn about configuring a Network Connector group and the available options, see [About Network Connector Provisioning Keys](/unified/about-network-connector-provisioning-keys).
To edit a configured Network Connector provisioning key:
- Go to **Infrastructure **>** Private Access **>** Component **>** VPN (For Legacy Apps) **>** Network Connector Provisioning Keys**.
-
In the table, locate the Network Connector provisioning key you want to modify and click the **Edit** icon.
The **Edit Network Connector Provisioning Key **drawer appears.
Only keys that can be viewed or exported after creation appear in the table.
-
In the **Edit Network Connector** **Provisioning Key **drawer, you can modify any field. To learn more, see [About Network Connector Provisioning Keys](/unified/about-network-connector-provisioning-keys).
![Edit Network Connector Provisioning Key Window in the Admin Portal](/downloads/zpa/vpn-legacy-apps/network-connector-management/network-connector-provisioning-keys/editing-network-connector-provisioning-keys/zpa-edit-network-connector-provkey.png)
- Click **Save**.
