# Manually Updating Private Cloud Controller Software
Source: https://help.zscaler.com/unified/manually-updating-private-cloud-controller-software
PDF: https://help.zscaler.com/pdf/com/en/1528706.pdf

After a Private Cloud Controller is scheduled for an update, you can't [change the periodic software update time](/unified/scheduling-periodic-software-updates-private-cloud-controller-group) for that particular Private Cloud Controller. However, you can update the Private Cloud Controller's software manually.
To manually update software for a Private Cloud Controller:
- Go to **Infrastructure** > **Private Access** > **Business Continuity **> **Private Cloud Controllers**.
- In the table, locate the Private Cloud Controller you want to upgrade.
- In the **Upgrade Status** column, verify that the status is **Scheduled**. If the status is not **Scheduled**, then you can't manually upgrade the Private Cloud Controller.
- Click the **Update Now** icon.
![The Update Now Icon in the Private Cloud Controller Page](/downloads/zpa/manually-upgrading-private-cloud-controller-software/zpa-private-cloud-controller-update-now.jpg)
The **Confirm Update Action **window appears.
- In the **Confirm Update Action** window, click the checkbox to confirm.
![Confirming the Update Action on the Private Cloud Controller Page](/downloads/zpa/manually-upgrading-private-cloud-controller-software/zpa-private-cloud-controller-confirm-update-action.jpg)
- Click **Update Now**.
