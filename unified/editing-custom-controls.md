# Editing Custom Controls
Source: https://help.zscaler.com/unified/editing-custom-controls
PDF: https://help.zscaler.com/pdf/com/en/1493036.pdf

All [custom controls](/unified/about-custom-controls) are editable, even ones currently in use by an [AppProtection profile](/unified/about-appprotection-profiles).
To edit a custom control:
- Go to **Policies > Cybersecurity > Inline Security >** **Protection Controls**.
- Click **Custom Controls**.
- In the table, locate the custom control you want to modify and click the **Edit** icon.
The **Edit Custom Control** window appears.
![Edit Custom Control window in the Admin Portal](/downloads/zpa/inspection-management/edit-custom-controls/zpa-editcustomcontrols.png)
- On the **Select Control Type** tab, modify fields as necessary. To learn more about each field, see [Configuring a Custom Control](/unified/configuring-custom-controls#type).
- Click **Next** to view the **Set Control Definition** tab and modify fields as necessary. To learn more about each field, see [Configuring a Custom Control](/unified/configuring-custom-controls#definition).
- Click **Next** to view the **Set Severity and Action** tab and modify fields as necessary. To learn more about each field, see [Configuring a Custom Control](/unified/configuring-custom-controls#action).
- Click **Next** to view the **Review** tab and to review your custom control changes.
- Click **Save**.
