# Configuring Network Service Groups for Internet & SaaS
Source: https://help.zscaler.com/unified/configuring-network-service-groups-internet-saas
PDF: https://help.zscaler.com/pdf/com/en/1494146.pdf

To create a network services group:
- Go to **Policies **>** Access Control **>** Firewall **>** Network Services**.
- In the **Service Groups** tab, click **Add Network Service Group**.
- Enter a **Name** for the network services group. It can include any character and spaces.
- Click the down arrow beside **Services**.
- Choose any number of custom and predefined services you want to include in the group.
- (Optional) Type in any additional notes or information about the group. The description cannot exceed 10,240 characters. To learn more, see [Ranges & Limitations](/unified/ranges-limitations#internet-saas).
- Click **Save** and activate the change.
