# Recommended FTP Control Policy
Source: https://help.zscaler.com/unified/recommended-ftp-control-policy
PDF: https://help.zscaler.com/pdf/com/en/1494121.pdf

Zscaler recommends that you configure the following [FTP Control](/unified/about-ftp-control) policy:
- Enable the **Allow FTP over HTTP **option
- Enable the **Allow Native FTP **option
- Disable the** Allow Any URL Category **option
- Set **Allowed URL Categories** to **None**
