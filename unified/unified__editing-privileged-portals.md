# Editing Privileged Portals
Source: https://help.zscaler.com/unified/editing-privileged-portals
PDF: https://help.zscaler.com/pdf/com/en/1492581.pdf

To edit a privileged portal:
- Go to **Policies **>** Access Control **>** Clientless **>** Privileged Portals**.
- In the table, locate the privileged portal you want to modify and click the **Edit **icon.
- In the **Edit Portal **window, modify fields as necessary. To learn more about each field, see [Configuring Privileged Portals](/unified/configuring-privileged-portals).
![Edit Portal window within the ZPA Admin Portal](/downloads/zpa/about-privileged-consoles/Editing%20Portal.png)
- Click **Save**.
