# Configuring Quarantine Tombstone Notification Templates
Source: https://help.zscaler.com/unified/configuring-quarantine-tombstone-notification-templates
PDF: https://help.zscaler.com/pdf/com/en/1493301.pdf

You can create templates for the tombstone file created when a file is quarantined, which is configured through the SaaS Security API Quarantine policy rules. To learn more, see [About SaaS Security API Malware Detection](/unified/about-saas-security-api-malware-detection) and [Configuring the SaaS Security API Malware Detection Policy](/unified/configuring-saas-security-api-control-policy).
To add a quarantine tombstone notification template:
- Go to **Policies > Common Configuration > Out-of-Band CASB > Tombstone Template**.
- Click **Add Tombstone File Template**.
- Under **Name**, enter a name for the template.
[See image.](#Quar_1)
- Under **Description**, enter the text you want the tombstone file to contain.
- Click **Save** and activate the change.
