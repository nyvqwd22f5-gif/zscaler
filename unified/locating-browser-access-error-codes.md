# Locating Browser Access Error Codes
Source: https://help.zscaler.com/zpa/locating-browser-access-error-codes
PDF: https://help.zscaler.com/pdf/com/en/1484186.pdf

If your users encounter an issue when attempting to access a Browser Access enabled application, the end user notification (EUN) includes the error code. For example:
![ZPA End User Notification (EUN) showing Browser Access-related error code](/downloads/zpa/documentation-knowledgebase/browser-access/understanding-browser-access-error-codes/zpa-browseraccess-eunmsg.png)
The EUN heading (e.g., ZPA BA: Proxy loop detected) is a hyperlink which leads your users to the Zscaler Help Portal for more information and the HTTP status code is always displayed. This information also appears within the [Diagnostics](/zpa/documentation-knowledgebase/dashboard-diagnostics) page.
To learn more regarding ZPA error and policy block codes, including possible resolutions, see [ZPA Session Status Codes](/zpa/zpa-session-status-codes).
