# Allowing Non-Administrator Users Access to Zscaler Client Connector Log Files
Source: https://help.zscaler.com/unified/allowing-non-administrator-users-access-zscaler-client-connector-log-files
PDF: https://help.zscaler.com/pdf/com/en/1496106.pdf

This feature enables Admin Portal administrators to grant Zscaler Client Connector log file access to users who donâ€™t have administrator privileges. This allows non-administrator users to send specific log files to admins instead of having to send the entire log folder in a ZIP file, making the troubleshooting process more efficient.
To allow non-admin users access to the Zscaler program data folder to access log files:
- In the Admin Portal, go to **Infrastructure > Connectors > Client > User Privacy**.
- On the **User Privacy** tab, select** Allow access to Client Connector logs folder for non-administrators**.
![Allow non admin access to the Zscaler Program Data folder log files](/downloads/client-connector-allow-access-logs-non-admins.png)
- Click **Save**.
