# Scheduling Periodic Software Updates for a Private Service Edge Group
Source: https://help.zscaler.com/unified/scheduling-periodic-software-updates-private-service-edge-group
PDF: https://help.zscaler.com/pdf/com/en/1496791.pdf

To schedule a periodic software update for a Private Service Edge group:
- Go to the **Private Service Edges **page (Infrastructure > Private Access > Components > Private Service Edges).
- In the table, locate the Private Service Edge group and click the **Edit** icon.
- In the **Edit Private Service Edge Group** window, under **Private Service Edge Software Update Schedule**, choose the day of the week and start time for the periodic update.
![Modifying the Private Service Edge Software Update Schedule for a Private Service Edge group within ZPA](/downloads/zpa/private-service-edge-management/private-service-edge-software-updates/scheduling-periodic-software-updates-zpa-service-edge-group/Edit%20PSE%20Group%20window.png)
- Click **Save**.
To confirm that the update was scheduled, go to the [Private Service Edge Groups](/unified/about-private-service-edge-groups-private-applications) page and verify that the proper day and time is displayed in the **Next Periodic Software Update** column.
