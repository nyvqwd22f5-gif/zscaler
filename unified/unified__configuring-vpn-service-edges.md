# Configuring VPN Service Edges
Source: https://help.zscaler.com/unified/configuring-vpn-service-edges
PDF: https://help.zscaler.com/pdf/com/en/1532358.pdf

Within the Admin Portal, you can create [VPN Service Edges](/unified/about-vpn-service-edges). For a complete list of ranges and limits, see [Ranges & Limitations](/unified/ranges-limitations).
To add a new VPN Service Edge:
- Go to **Infrastructure** > **Private Access** > **Component** > **VPN (for Legacy Apps)** > **VPN Service Edges**.
- Click **Add VPN Service Edges**.
The **Add VPN Service Edge** window appears.
- In the **Add VPN Service Edge** window:
- **Name**: Enter the name for the VPN Service Edge.
- **Region**: Select the region that is designated to the VPN Service Edge.
- **Default UDP Listen Port**: View the default UDP port.
- **Client IP Pool**: Define the IP pool assigned to the VPN Service Edge. This IP pool is used to assign the IP address to the Zscaler Client Connector. Click **Add once** you have defined the IP pool.
[See image.](#image)
- Click **Save**.
