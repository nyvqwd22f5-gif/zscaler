# Searching in the Admin Portal
Source: https://help.zscaler.com/unified/searching-admin-portal
PDF: https://help.zscaler.com/pdf/com/en/1503801.pdf

To help you navigate faster, you can search for pages in the Admin Portal. Search results include Admin Portal pages (except for items in the Help menu) and do not include:
- Tabs
- Field names
- Toggle names
- Tooltips or other text
To search the Admin Portal:
-
Click the **Search **icon in the navigation menu to display the search box, or press `Ctrl`+`K`.
[See image.](#search-icon)
-
Enter your search text in the search box to display a list of menus that match your search term.
[See image.](#search-results)
- Click any search result to navigate to that page.
The search provides auto-complete suggestions as you type, and keeps a history of your recent searches.
