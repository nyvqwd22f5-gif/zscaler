# About the Endpoint Integration Page
Source: https://help.zscaler.com/unified/about-endpoint-integration-page
PDF: https://help.zscaler.com/pdf/com/en/1496121.pdf

In the Admin Portal, you can configure how Zscaler Client Connector interacts with applications installed on the device.
Zscaler Client Connector Endpoint Integration settings provide the following benefits and allow you to:
- Control whether Zscaler Client Connector can overwrite Zscaler settings for macOS and Windows devices or create new settings to configure Firefox.
- Avoid port conflicts with other applications by configuring the port for Zscaler Client Connector to listen on.
- Add your VPN vendor to ensure Zscaler Client Connector detects it as a VPN Trusted Network.
- Configure the Zscaler Client Connector synthetic IP range for your organization's Private Applications.
- Enable Zscaler Client Connector to block non-Zscaler packets destined for the synthetic IP range.
