# Adding an Index Tool Configuration
Source: https://help.zscaler.com/unified/adding-index-tool-configuration
PDF: https://help.zscaler.com/pdf/com/en/1493271.pdf

To add an Index Tool Configuration:
- Go to **Policies > Data Protection > Common Resources > Index Tool**.
- Click **Add Index Tool Configuration**.
The **Add Index Tool Configuration** window appears.
-
In the **Add Index Tool Configuration** window:
- **VM Name**: Enter a unique name for the virtual machine (VM)
- **Status**: Make sure that the VM is **Enabled**.
- **Enable Single Sign-On**: Select **Enabled **if you are configuring single sign-on (SSO) for the Index Tool. To learn more, see [Configuring Single Sign-On for the Index Tool](/unified/configuring-single-sign-index-tool).
[See image.](#add-index-tool)
-
Click **Save** and [activate the change](/unified/saving-and-activating-changes-admin-portal).
After you save, the **SSL Certificate** for the configuration can be downloaded from the [Index Tool](/unified/about-index-tool) page or from the [Edit Index Tool Configuration](/unified/modifying-index-tool-configuration) window.
