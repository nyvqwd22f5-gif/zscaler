# About the Gen AI Security Report
Source: https://help.zscaler.com/unified/about-gen-ai-security-report
PDF: https://help.zscaler.com/pdf/com/en/1506861.pdf

The Gen AI Security Report gives visibility and insight into the generative AI (Gen AI) applications accessed within your organization.
The Gen AI Security Report provides the following benefits and enables you to:
- Gain visibility and control over Gen AI application usage within your organization, enabling you to innovate with AI while protecting sensitive data.
- Permit the use of only appropriate AI applications based on risk, with granular usage controls, down to individual departments and users.
- Identify who is making use of AI in the organization and gain complete access to all prompts and queries in the AI applications.
- Get visibility into the type of prompts being entered, by classifying them into categories using small language models.
About the Gen AI Security Report Page
On the Gen AI Security Report page (Analytics > Internet & SaaS > Analytics > Gen AI Security Report), you can do the following:
- Filter the report for the **Last 1 Day**, **Last 7 Days**, **Last 15 Days**, **Last Month**, or **Last Quarter**.
- Gain a comprehensive view of the Gen AI application activities in your organization, such as:
-
**Gen AI Applications**: View the number of Gen AI applications used in the organization, categorized by sanctioned and unsanctioned applications. Click the value displayed to further analyze the application on the [Web Insights Logs](/unified/about-insights-logs) page.
[See image.](#web_insights)
- **Transactions to Gen AI**: View the total number of transactions for the Gen AI applications in your organization.
- **Sensitive Data to Gen A****I**: View the total number of sensitive content items, based on DLP Engine and DLP Dictionaries, discovered per application. Click the value displayed to further analyze the data in the **Analyze More** page with content type **DLP Engine** selected.
- **Accessed by Users**: View the number of users who accessed the Gen AI applications.
- **Total Files Uploaded**: View the total number of files uploaded.
- **View Prompts**: You are redirected to the [Web Insights Logs](/unified/about-insights-logs) page, which shows the prompts entered by each user for the AI applications. Click **Back to Original** to go to the **Gen AI Security Report** page. The prompt filter **Not Null** is applied on redirection. To view the **View Prompts** tab, you must enable the prompt settings for individual AI applications in **URL & Cloud App Control> Advanced Policy Settings > Gen AI Prompt Configuration**. To learn more, see [Configuring Advanced Policy Settings](/unified/configuring-advanced-policy-settings).
- **Analyze More**: Further analyze the transactions in a detailed view.
- [Detailed View](#analyzemore_drilldown)
- **Gen AI Application Usage**: View the top 6 Gen AI applications used in the organization with policy actions (Allowed, Blocked, or Isolated) distribution for each of the applications in the form of a bar chart. Hover over the bars to view the number of transactions generated by the users. You can filter data for the Gen AI applications by **Total Bytes**, **Transactions**, or **Users, **and by the **Status **of the application (**All**, **Sanctioned**, or **Unsanctioned**). Policy action distribution is displayed only when you select **Total Bytes** or **Transactions **filters. You can choose to view the graph for a specific policy action from the bottom of the graph.
- **Gen AI Usage Trends**: View the Gen AI application usage trends of the top 6 applications in the form of a line chart. Hover over a date to view the Gen AI application usage trend for that date. You can choose to view the graph for any specific Gen AI application from the bottom of the graph. By default, all the Gen AI applications are selected. You can filter the data by **Transactions**, **Users**, or **Total Bytes**. You can choose to view the graph for a specific Gen AI application from the bottom of the graph.
- **Sensitive Data Transactions**: View the sensitive content, based on DLP Engine, discovered per application in the form of a donut chart. Hover over the donut chart to view the sensitive data transactions triggered by the DLP dictionaries. You can filter the data by total **Transactions **or **Total Bytes**. Click **Analyze More** to further drill down on data with the content type **DLP Engine** selected.
- **Gen AI Usage by Department**: View the Gen AI application usage drilled down to organization departments or groups in the form of a stacked bar chart. Hover over the chart to view the Gen AI application usage by department. You can click the particular department to drill down further. Click **Analyze More** to further drill down for data about selected departments.
- **Prompt Classification & Document Classification**:
- **Prompt Classification**: View the Gen AI applications usage trends based on the classification of prompts made using the ML model. Hover over the block chart to view the percentage-wise distribution of the prompt classification and the corresponding Gen AI application that was used for the prompt. You can filter the data by the top AI applications or individual Gen AI applications. Hover over the individual blocks to view the percentage-wise distribution of the application by the prompt classification. Click **Analyze More** to further drill down on the data with the content type **Prompt Classification** selected.  To learn more, see [Web Insights Logs: Filters](/unified/web-insights-logs-filters) and [Web Insights Logs: Columns](/unified/web-insights-logs-columns).
- **Document Classification**: View the trends of prompts being entered for Gen AI document type usage across the Gen AI applications. Hover over the line chart to view the prompt themes used for the document type. You can filter data by all AI applications or a specific AI application. You can choose to view the graph for any specific document type from the bottom of the graph. By default, all document type usage patterns are displayed. Click **Analyze More** to further drill down data with the content type **Document Type** selected.
- **Top Users**: View the top users and the corresponding Gen AI applications used in the organization. Hover over the line severity bars to view the data distribution across the Gen AI applications for all the users. You can filter by all AI applications or select a specific Gen AI application.

[]The detailed view is divided into three columns: **Applications**, **Content Type**, and **Users**. You can click the **Column Menu** icon to customize the columns. You can drag and drop the labels to interchange the order of columns. To drill down for specific data:
- Select the time period for which you want to see the data. You can select for the **Last 1 Day**, **Last 7 Days**, **Last 15 Days**, **Last Month**, or **Last Quarter**.
-
Filter the data for specific Policy Actions taken (**All**, **Allowed**, **Blocked**, or **Isolated**), Application Status (**All**, **Sanctioned,** or **Unsanctioned**), or Content Type (**DLP Dictionaries**, **DLP Engines**, **Document Classification**, or **Prompt Classification**)
You can choose to show or hide filters from the top right of the page.
The **Applications** column populates the data determined by your selections. Each application tile shows the application status generated by them and their percentage contribution to the total incidents generated by the application.
-
Select the application status for which you want to view the data.
The Content Type (**DLP Dictionaries**,** DLP Engines**, or **ML Categories**)** **column is updated for the selected application. Each content type tile shows the number of transactions discovered by them and their percentage contribution to the total transactions discovered by the content type.
-
Select the content type for which you want to further drill down the data.
The **Users** column is updated for the selected content type. Each user tile shows the number of incidents by the user and the percentage contribution to the total transactions generated by all the users.
-
Click the **Actions **drop down menu in the **Applications**, **Content Type**, or **Users **tile to do the following:
- **View Logs**: You are redirected to the Insights Logs page. Click **Back to Original** to go back to the **Gen AI Security Report**. To learn more, see [Web Insights Logs: Filters](/unified/web-insights-logs-filters) and [Web Insights Logs: Columns](/unified/web-insights-logs-columns).
- **Create Policy**: You are redirected to the Add AI & ML Rule window where you can set the Cloud App Control Role, Criteria, Rule Expiration, and Action.
- **View Prompts**: To view prompts for that particular application. You are redirected to the Insights Logs page where you can view the prompts entered by the user which triggered the DLP Engine. The prompt filter for **Not Null** is applied on redirection. To view the **View Prompts** tab, you must enable the prompt settings for individual AI applications in **URL & Cloud App Control > Advanced Policy Settings > Gen AI Prompt Configuration**. To learn more, see [Configuring Advanced Policy Settings](/unified/configuring-advanced-policy-settings).
Additionally, you can click the **Download** icon (![Download icon in the Drill Down View](/downloads/zia/documentation-knowledgebase/analytics/dashboards-reports-and-logs/reports/about-endpoint-dlp-report-incidents/image%20(19).png)
) to export any column data to a comma-separated value (CSV) file. When you select an item in a column, the option to export as a CSV file is no longer available for that column.
