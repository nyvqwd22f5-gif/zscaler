# Editing Network Segments
Source: https://help.zscaler.com/unified/editing-network-segments
PDF: https://help.zscaler.com/pdf/com/en/1532362.pdf

This article describes how to edit a Network segment. To learn about configuring Network segments and the available options, see [About Network Segments](/unified/about-network-segments).
To edit a Network segment:
- Go to **Infrastructure** >** Private Access **>** Component > VPN (For Legacy Apps) **>** Network Segments**.
-
In the table, locate the network segment you want to modify and click the **Edit **icon.
The **Edit Network Segment** drawer appears.
- In the **Edit Network Segment** drawer, you can modify any field. To learn more, see [About Network Segments](/unified/about-network-segments).
![Image of Edit Network Segments Drawer](/downloads/zpa/editing-network-segments/zpa-vpn-network-segments-edit_0.png)
- Click **Save**.
