# Accessing and Using the Executive Insights App
Source: https://help.zscaler.com/unified/accessing-and-using-executive-insights-app
PDF: https://help.zscaler.com/pdf/com/en/1520566.pdf

The Executive Insights App is designed to provide an executive summary of the key data points and valuable insights into your enterprise security environment on a mobile device. It provides you with a dynamic view of your operational performance and security posture.
To get started with the Executive Insights App:
To access the Executive Insights App, you need the Internet & SaaS subscription.
- [1. Authorize the executives to access and use the Executive Insights App.](#authorize-executives)
You might already have the role enabled for you. Try to install and access the app, and if you can log in to the app, skip this step.
- [2. Install the Executive Insights App on your mobile device.](#new-install-EIA)
- [3. Log in to Executive Insights App.](#new-log-into-EIA)
To learn about the common errors that might occur in the Executive Insights App and the troubleshooting tips, see [Executive Insights App Errors and Troubleshooting](/unified/executive-insights-app-errors-and-troubleshooting).
[]Contact your IT admin or Network Security team to add role-based access control (RBAC) roles using the [Admin Portal](/unified/adding-admin-roles) or [ZIdentity Admin Portal](/zidentity/assigning-cxo-insight-role).
Administrators can authorize executives to view the data of multiple organizations.
After you are authorized by an administrator to access the Executive Insights App, you can download the app, sign in, and view your organization's summary.
[]Select the appropriate installation instructions for your device:
- [Android](#android-instruction)
- [iOS](#iOS-instruction)

-
On the **Login** screen, in the **Email Address** field, enter your corporate email address associated with the Zscaler account.
[See image.](#new-EIA-login-screen)
-
Review the Zscaler's End User Subscription Agreement and Privacy Policy.
To accept the Zscaler's End User Subscription Agreement and Privacy Policy and proceed to using the app, select the checkbox below the **Email Address** field.
-
Tap **Login**.
The login fails if the email address is not authorized.
You receive an email from `no-reply@zscaler.com` with the verification code. If you donâ€™t see this email within 30 to 60 seconds in your inbox, please check your spam folder.
[See image.](#EIA-one-time-verification-code)
The **Verification** screen appears.
To resend the verification code, tap **Resend Code**.
-
On the **Verification** screen, enter the verification code.
[See image.](#EIA-verification-screen)
-
Tap **Next**.
The **Select Organization** screen appears.
-
On the **Select Organization** screen, select a tenant that you would like to access.
You can switch tenants in the **User Profile** section.
[See image.](#new-EIA-select-org-screen)
-
Tap **Next**.
The **Welcome** screen appears. On the **Welcome** screen, you can browse the Executive Insights App content.
You can tap **Skip** to skip the Executive Insights App preview.
[See image.](#new-EIA-welcome-screen)
