# Deleting an Isolation Profile for Private Applications
Source: https://help.zscaler.com/unified/deleting-isolation-profile-private-applications
PDF: https://help.zscaler.com/pdf/com/en/1500361.pdf

After [creating a Private Applications isolation profile](/unified/creating-isolation-profiles-private-applications), you can delete the profile at any time.
To delete your Private Applications isolation profile:
- Go to **Policies** > **Access Control** > **Clientless** > **Profiles**.
- From the **Isolation Profiles** page, click the **Delete** icon for the isolation profile that you want to delete.
[See image.](#Delete-icon)
- A confirmation message asks you to confirm deletion of the profile. Click **Delete**.
[See image.](#Delete-confirmation-message)
The Isolation Profile page refreshes and removes the deleted profile from the list.
