# Editing SAML Attributes
Source: https://help.zscaler.com/unified/editing-saml-attributes
PDF: https://help.zscaler.com/pdf/com/en/1491336.pdf

To edit a SAML attribute:
- Go to **Administration** > **Identity** > **Private Access **> **SAML Attributes**.
- In the table, locate the SAML attribute you want to modify and click the **Edit icon**.
- In the **Edit SAML Attribute **window, modify fields as necessary. You cannot modify the selected **IdP Configuration**. If you need to adjust this attribute for another [IdP configuration](/unified/about-idp-configuration), you must import or add it again for that IdP. To learn more, see [Importing SAML Attributes](/unified/importing-saml-attributes) and [Manually Adding SAML Attributes](/unified/manually-adding-saml-attributes).
![Edit SAML Attribute window ](/downloads/zpa/documentation-knowledgebase/saml-attributes/midp-about-samlattribute-edit/zpa-editsamlattribute.png)
- Click **Save**.
