# Managing Chrome Posture Profiles
Source: https://help.zscaler.com/unified/managing-chrome-posture-profiles
PDF: https://help.zscaler.com/pdf/com/en/1529464.pdf

Zscaler recommends reviewing your Chrome posture profiles periodically to ensure security evaluation is required for your devices using the Chrome Enterprise browser.
[]Editing a Chrome Posture Profile
To edit a Chrome posture profile:
- Go to** Policies** > **Access Control **> **Clientless** > **Chrome Posture Profile**.
- Click the **Edit** icon for the posture profile you want to modify.
-
In the **Edit Chrome Posture Profile** window, modify the fields as necessary. To learn more, see [Configuring Chrome Posture Profiles](/unified/configuring-chrome-posture-profiles).
[See image.](#img_edit)
- Click **Save**.
[]Deleting a Chrome Posture Profile
To delete a Chrome posture profile:
- Go to** Configuration & Control** > **Chrome Enterprise Browser** > **Chrome Posture Profile**.
-
Click the **Delete** icon for the Chrome posture profile you want to delete.
![Confirm deletion of Chrome posture profile](/downloads/zpa/browser-access/maintaining-chrome-posture-profiles/ZPA-Chrome-Device-Posture-delete.png)
- Click **Delete** to confirm the deletion of the selected Chrome posture profile.
