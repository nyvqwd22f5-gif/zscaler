# Recommended Browser Control Policy
Source: https://help.zscaler.com/unified/recommended-browser-control-policy
PDF: https://help.zscaler.com/pdf/com/en/1494396.pdf

Zscaler recommends that you configure the following [Browser Control](/unified/configuring-browser-control-policy) policy:
- Disable **Enable Checks & User Notification**
- Disable **Allow All Browsers**
