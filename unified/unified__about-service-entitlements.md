# About Service Entitlements
Source: https://help.zscaler.com/unified/about-service-entitlements
PDF: https://help.zscaler.com/pdf/com/en/1505796.pdf

The Service Entitlements page shows the list of Zscaler services that your organization is subscribed to. You can assign end users to these Zscaler services.
This page is available only if your organization is provisioned with ZIdentity for end users.
Service Entitlements provide the following benefits and enable you to:
- View your enrolled Zscaler services.
- Assign users and user groups to the subscribed Zscaler services.
- Control which users or user groups within your organization are assigned to a Zscaler service.
About the Service Entitlements Page
On the Service Entitlements page (Administration > Entitlements > End User Entitlements), you can do the following:
- View a list of subscribed Zscaler services. For each service, you can view:
- **Name**: The name of the Zscaler service.
- **Cloud Name**: The Zscaler cloud name.
- **Organization Name**: The name of your organization.
-
Select a service to view or [assign users and user groups](/unified/assigning-entitlements-users#service-entitlements) to that service.
