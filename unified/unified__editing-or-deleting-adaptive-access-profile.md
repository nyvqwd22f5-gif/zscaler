# Editing or Deleting an Adaptive Access Profile
Source: https://help.zscaler.com/unified/editing-or-deleting-adaptive-access-profile
PDF: https://help.zscaler.com/pdf/com/en/1515906.pdf

You can review the Adaptive Access profiles over a period of time for their effectiveness, and modify or delete the profiles as required.
Editing an Adaptive Access Profile
To edit an Adaptive Access profile:
- Go to **Policies** > **Common Configuration** > **Adaptive Access** > **Profiles**.
-
On the **Profiles** page, click the **Edit** icon for the required profile.
[See image.](#aae-clickediticon)
-
In the **Edit Profiles** window, modify the values as required.
[See image.](#aae-editprofiledetails)
- Click **Update**.
Deleting an Adaptive Access Profile
To delete an Adaptive Access profile:
- Go to **Policies** > **Common Configuration** > **Adaptive Access** > **Profiles**.
-
On the **Profiles** page, click the **Delete** icon for the required profile.
[See image.](#aae-deleteicon)
-
In the **Confirmation** window, enter `YES`, then click **Delete**.
[See image.](#aae-deleteprofile)
