# About Administrative Entitlements
Source: https://help.zscaler.com/unified/about-administrative-entitlements
PDF: https://help.zscaler.com/pdf/com/en/1505791.pdf

The Administrative Entitlements page shows the list of Zscaler services that your organization is subscribed to. You can assign ZIdentity users as service admins to these Zscaler services and with a role that is created in the Admin Portal. Admins can view only the assigned services on the Admin Portal. To learn more, see [Managing Entitlements](/unified/managing-entitlements).
Administrative Entitlement provides the following benefits and enables you to:
- View your subscribed Zscaler services for which you can assign admins.
- Assign users or user groups with admin roles to your subscribed Zscaler services.
- Control which user within your organization has access to the Admin Portal.
- View all the users and user groups assigned as service admins for a Zscaler service.
About the Administrative Entitlements Page
On the Administrative Entitlements page (Administration > Admin Management > Role Based Access Control > Administrative Entitlements), you can do the following:
- View a list of subscribed Zscaler services. For each service, you can view:
- **Name**: The name of the Zscaler service.
- **Cloud Name**: The Zscaler cloud name.
- **Organization Name**: The name of your organization.
-
Select a Zscaler service to view or [assign individual users or user groups](/unified/assigning-entitlements-users#admin-entitlements) as service admins for that service.
