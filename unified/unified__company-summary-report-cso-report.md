# Company Summary Report (CSO Report)
Source: https://help.zscaler.com/unified/company-summary-report-cso-report
PDF: https://help.zscaler.com/pdf/com/en/1497501.pdf

The Company Summary Report for Chief Security Officers (CSO Report) is an interactive report (Analytics > Internet & SaaS > Analytics > Interactive Reports) that provides information tailored for audiences like the CSO of your organization.
The report covers the following information for your organization:
- Overall Internet Transactions
- Total Bytes
- Total Mobile Transactions
- Total Mobile Bytes
- Overall Threat Posture
- Top Blocked Viruses and Spyware
- Top Blocked Advanced Threats
- Top Blocked Sandbox Threats
- Top Blocked Malware
- Top Locations with Highest Threats
- Top Blocked Threat Super Categories for Road Warriors
- DLP Engine Violations
- DLP Dictionary Violations
- Top Categories for Liability Exposure Allowed
- Bandwidth Loss Trend
- Top Users Infected by Threats
About the CSO Report Page
As with other [interactive reports](/unified/about-interactive-reports), the CSO Report supports real-time interactive analysis and contains [widgets](/unified/about-widgets) that present information in different graphical formats. You can drill down to the logs for each chart by rolling over the chart, clicking, and selecting one of the following options:
- Click** View Logs **to see its associated logs. See [About Insights Logs](/unified/about-insights-logs).
- Click **Analyze Chart **to see the chart in Web Insights and to drill down to the logs. See [Analyzing Traffic Using Insights](/unified/analyzing-traffic-using-insights).
You can also:
- [Add custom notes to the report](/unified/adding-notes-dashboards-and-interactive-reports)
- Add it your list of favorite reports
- [Copy the report](/unified/creating-or-copying-report)
- [Schedule regular email deliveries of the report](/unified/about-scheduled-reports)
- [Print the report](/unified/printing-reports)
