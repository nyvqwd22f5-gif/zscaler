# About Alert Rules
Source: https://help.zscaler.com/unified/about-alert-rules
PDF: https://help.zscaler.com/pdf/com/en/1530726.pdf

The Alert Rules page provides a graphical representation of the top 5 Security and UEBA alerts by the rule type and the number of rules that have been enabled or disabled.
The Alert Rules page provides the following benefits and enables you to:
- Easily assess the top 5 Security and UEBA alerts using detailed widgets.
- Enable or disable preconfigured alerts for ease of use.
The Alert Rules page also lists the preconfigured rules in a table where you can enable or disable the alert. Click the Edit icon to customize the alert rules for the preconfigured alerts.
About the Alert Rules Page
On the Alert Rules page (Administration > Alerts > Security & UEBA Alerts > Alert Rules), you can do the following:
- View the top 5 Security alerts by rule name.
- Select the time period to display the top 5 alerts.
- View the top 5 UEBA alerts by rule name.
- Select the time period to display the top 5 alerts.
- View the total number of enabled or disabled alert rules.
- [Add a new alert rule](/unified/configuring-alert-rule-internet-saas).
- [Edit an existing alert rule](/unified/configuring-alert-rule-internet-saas).
- Delete an existing alert rule.
