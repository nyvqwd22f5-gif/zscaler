# Editing Network Connector Groups
Source: https://help.zscaler.com/unified/editing-network-connector-groups
PDF: https://help.zscaler.com/pdf/com/en/1532375.pdf

This article describes how to edit a Network Connector group. To learn about configuring a Network Connector group and the available options, see [About Network Connector Groups](/unified/about-network-connector-groups).
To edit a configured Network Connector group:
- Go to **Infrastructure **>** Private Access **>** Component **>** VPN (for Legacy Apps) **>** Network Connector Groups**.
-
In the table, locate the Network Connector group you want to modify and click the **Edit** icon.
The **Edit Network Connector Group** drawer appears.
-
In the **Edit Network Connector Group** drawer, you can modify any field. To learn more, see [Configuring Network Connectors](/unified/configuring-network-connectors).
![Editing Network Connector groups in the Admin Portal](/downloads/zpa/editing-network-connector-groups/zpa-edit-network-connector-group_1.png)
- Click **Save**.
