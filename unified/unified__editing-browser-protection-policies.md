# Editing Browser Protection Policies
Source: https://help.zscaler.com/unified/editing-browser-protection-policies
PDF: https://help.zscaler.com/pdf/com/en/1493701.pdf

To edit a Browser Protection policy:
- Go to **Policies **>** Cybersecurity **>** Inline Security **>** Private AppProtection **> **Protection Policies**.
- In the table, locate the policy you want to modify and click the **Edit** icon.
The **Edit Browser Protection Policy **window appears.
![Edit Browser Protection Policy window in the Admin Portal](/downloads/zpa/policies/editing-browser-protection-policies/Browser%20Protection%20Policy%20edit%20window.png)
If you only want to change a Browser Protection policy rule's place in the Rule Order, you can manually enter in a new value on the [Browser Protection Policy page](/unified/about-browser-protection-policy).
- In the **Edit Browser Protection Policy** window, modify fields as necessary. For more information about each field, see [Configuring Browser Protection Policies](/unified/configuring-browser-protection-policies).
- Click **Save**.
