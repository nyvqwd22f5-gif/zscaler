# Configuring a Default Global Log Mode
Source: https://help.zscaler.com/unified/configuring-default-global-log-mode
PDF: https://help.zscaler.com/pdf/com/en/1494476.pdf

You can configure a default global log mode for new profiles. This saves you from having to configure the log mode for each new profile in [App Profiles](/unified/configuring-zscaler-client-connector-app-profiles). You can override this setting for individual profiles in App Profiles.
To configure the default global log level:
- In the Admin Portal, go to **Infrastructure > Connectors > Client**.
- Under Platform Settings, select the operating system and click the **Platform Settings** tab.
- From the **Log Mode** drop-down menu, select a mode. For a description of each log mode, see [Configuring Zscaler Client Connector App Profiles](/unified/configuring-zscaler-client-connector-app-profiles).
![Configure global log mode in Platform Settings](/downloads/client-connector/zscaler-client-connector-profile-management/configuring-global-default-log-level/Platform_Settings.png)
- Click **Save**.
