# Updating the Control Severity
Source: https://help.zscaler.com/unified/updating-control-severity
PDF: https://help.zscaler.com/pdf/com/en/1516266.pdf

You can view and change the severity level of a control on the [Posture page](/unified/about-posture) or the [Control Panel](/unified/about-control-panel) header.
- [Update the Control Severity on the Posture Page](#Severity-Posture-Page)
- [Update the Control Severity on the Control Panel Header](#Severity-Control-Panel)
[]To update the control severity:
- On the Posture page, hover over the control severity you want to update and click it.
-
From the drop-down menu, select the severity level for the control.
[See image.](#Control-Severity-Posture-Page-Drop-Down)
The **Update Control Severity Level **window appears.
- (Optional) Enter the reason for changing the severity level and click **Save**.
[See image.](#Update-Control-Severity-Window)
The severity level is updated successfully, and the update is recorded on the [Audit Log](/unified/about-control-panel#Audit-Log) tab.
[]To update the control severity:
- Select a control.
The Control Panel opens.
- On the Control Panel header, click the control severity to update it.
-
From the drop-down menu, select the severity level for the control.
[See image.](#Control-Severity-Control-Panel-Drop-Down)
The **Update Control Severity Level** window appears.
- (Optional) Enter the reason for changing the severity level and click **Save**.
The severity level is updated successfully, and the update is recorded on the [Audit Log ](/unified/about-control-panel#Audit-Log)tab.
The defined control severity will override any future system recalculations. You can revert to the default severity.
To revert to the default severity from the Control Panel header or the Posture page:
-
Click the **Refresh** icon next to the severity.
[See image.](#Refresh-Icon)
A confirmation window appears.
-
Click **Yes** to proceed.
[See image.](#Revert-Severity-Confirmation-Window)
