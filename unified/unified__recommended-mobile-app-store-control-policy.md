# Recommended Mobile App Store Control Policy
Source: https://help.zscaler.com/unified/recommended-mobile-app-store-control-policy
PDF: https://help.zscaler.com/pdf/com/en/1495966.pdf

Zscaler recommends that you [configure](/unified/configuring-mobile-app-store-control-policy) the following [Mobile App Store Control](/unified/about-mobile-app-store-control) policy.
- **Admin Rank**: 1
- **Rule Status**: Enabled
- **App Stores**: Any
- **Users**: Any
- **Groups**: Any
- **Departments**: Any
- **Locations**: Any
- **Location Groups**: Any
- **Time**: Always
- **Application Download**: Allow
