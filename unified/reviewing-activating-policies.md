# Reviewing and Activating Policies
Source: https://help.zscaler.com/unified/reviewing-activating-policies
PDF: https://help.zscaler.com/pdf/com/en/1490911.pdf

The final onboarding step displays all the policy configurations you have selected. To make changes to any set of policies, click the **Edit **icon to return to that section. You can also make changes later in the Admin Portal.
Click **Activate and Launch **to activate Zscaler with the policies you have selected and launch the [Networking dashboard](/unified/viewing-networking-dashboard) in the Admin Portal. From the Admin Portal, you have access to all Zscaler capabilities.
To learn more, see [Navigating within the Admin Portal](/unified/signing-admin-portal#navigating-admin-portal).
