# Adding Cloud NSS Feeds for Cloud & Branch Connector
Source: https://help.zscaler.com/unified/adding-cloud-nss-feeds-cloud-branch-connector
PDF: https://help.zscaler.com/pdf/com/en/1519541.pdf

A Cloud NSS feed specifies the data from the logs that the Cloud NSS sends to the security information and event management (SIEM) system. You can filter the data, so you send only the data you need to the SIEM. You can add one or more fields for the logs and one field for alerts. To learn more about how to configure each feed, see the following articles:
