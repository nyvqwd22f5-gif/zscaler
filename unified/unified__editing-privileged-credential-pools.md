# Editing Privileged Credential Pools
Source: https://help.zscaler.com/unified/editing-privileged-credential-pools
PDF: https://help.zscaler.com/pdf/com/en/1524966.pdf

To edit a privileged credential pool:
- Go to **Policies > Access Control **>** Clientless **>** Privileged Credentials **>** Credential Pools**.
- In the table, locate the privileged credential pool you want to modify and click the **Edit** icon.
The **Edit Privileged Credential** **Pool** window appears.
- In the **Edit Privileged Credential Pool** window, modify fields as necessary. To learn more about each field, see [Configuring Privileged Credential Pools](/unified/configuring-privileged-credential-pools).
