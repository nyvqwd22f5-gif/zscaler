# Searching for a Zscaler Client Connector App Profile
Source: https://help.zscaler.com/unified/searching-zscaler-client-connector-app-profile
PDF: https://help.zscaler.com/pdf/com/en/1494441.pdf

You can quickly find a specific Zscaler Client Connector app [profile](/unified/configuring-zscaler-client-connector-app-profiles) using the Search** **field by selecting different criteria to narrow down your search results.
To find a Zscaler Client Connector app profile using the Search field:
- In the Admin Portal, go to **Infrastructure > Connectors > Client**.
- Under Platform Settings, select the operating system.
- In the **Search** field, select a search option from the drop-down menu. You can search using the following criteria:
- Policy Name
- Group Name
- Policy Token
- Enter search words and press `Enter` to display your search results.
To clear the **Search **field, delete the search term you typed and press `Enter`.
