# Editing Privileged Credentials Policies
Source: https://help.zscaler.com/unified/editing-privileged-credentials-policies
PDF: https://help.zscaler.com/pdf/com/en/1493856.pdf

To edit a privileged credentials policy rule:
- Go to **Policies** > **Access Control **> **Clientless** > **Privileged Policy**.
- Click **Privileged Credentials Policy**.
- Locate the privileged credential policy rule you want to modify and click the **Edit** icon.
The **Edit Privileged Credentials Policy** window appears.
If you only want to change a privileged credentials policy rule's place in the Rule Order, you can manually enter a new value on the [Privileged Credentials Policy](/unified/about-privileged-credentials-policy) page.
- In the** Edit Privileged Credentials Policy** window, modify fields as necessary. For more information about each field, see [Configuring Privileged Credentials Policies](/unified/configuring-privileged-credentials-policies).
![Edit Privileged Credentials Policy window in the Admin Portal](/downloads/Editing%20window.png)
- Click **Save**.
