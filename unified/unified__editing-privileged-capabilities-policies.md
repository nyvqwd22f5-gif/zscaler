# Editing Privileged Capabilities Policies
Source: https://help.zscaler.com/unified/editing-privileged-capabilities-policies
PDF: https://help.zscaler.com/pdf/com/en/1494541.pdf

To edit a privileged capabilities policy rule:
- Go to **Policies **>** Access Control **>** Clientless **>** Privileged Policy**.
- Locate the privileged capabilities policy rule you want to modify and click the **Edit** icon.
If you only want to change a privileged capabilities policy rule's place in the Rule Order, you can manually enter a new value on the [Privileged Capabilities Policy page](/unified/about-privileged-capabilities-policy).
- In the **Edit Privileged Capabilities Policy** window, modify fields as necessary. For more information about each field, see [Configuring Privileged Capabilities Policies](/unified/configuring-privileged-capabilities-policies).
![Edit Privileged Capabilities Policy window in the Admin Portal](/downloads/zpa/policies/privileged-remote-access-policy/editing-privileged-capabilities-policies/Edit%20Privileged%20Capabilities%20Policy%20window.png)
- Click **Save**.
