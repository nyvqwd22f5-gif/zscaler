# About Root Certificates for Isolation in Private Applications
Source: https://help.zscaler.com/unified/about-root-certificates-isolation-private-applications
PDF: https://help.zscaler.com/pdf/com/en/1500406.pdf

Root certificates allow admins to have more control over isolation profile authentication. Admins can manage root certificates for their organization and associate them with specific isolation profiles. To learn more, see [Creating Isolation Profiles for Private Applications](/unified/creating-isolation-profiles-private-applications).
About the Root Certificates for Isolation Page
On the Root Certificates for Isolation page (Policies > Access Control > Clientless > Root Certificates), you can do the following:
- [Add](/unified/adding-root-certificates-isolation-private-applications) a new root certificate.
- Expand details for all root certificates in the list.
- View a list of all root certificates for your organization.
- [Edit](/unified/editing-root-certificate-isolation-private-applications) any custom root certificates that are not the default.
- [Delete](/unified/deleting-root-certificate-isolation-private-applications) any custom root certificates that are not the default and are not associated with any isolation profile.
