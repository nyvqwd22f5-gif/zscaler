# About Authentication Default Settings
Source: https://help.zscaler.com/zia/about-authentication-default-settings
PDF: https://help.zscaler.com/pdf/com/en/1400051.pdf

[Watch a video about Authentication Default Settings.](https://fast.wistia.net/embed/iframe/dfw76cozw7)
Zscaler supports different types of [provisioning and authentication methods](/zia/choosing-provisioning-and-authentication-methods). You can configure the default authentication profile according to the authentication method you choose. To learn more about authenticating users, see [About Provisioning and Authenticating Users](/zia/provisioning-and-authenticating-users).
The Authentication Default Settings provide the following benefits and enable you to:
- Force all users in your organization to reauthenticate with a single click.
- Configure Kerberos authentication.
- Configure a default authentication profile for your organization.
The Default Settings page displays limited options if you're subscribed to the ZIdentity service. The unavailable settings are managed in the ZIdentity Admin Portal. To learn more, see [What Is ZIdentity?](/zidentity/what-zidentity)
[See image.](#default-settings-zidentity)
About the Default Settings Page
On the Default Settings page (Administration > Authentication Settings > Default Settings), you can do the following:
- [Configure the default authentication profile](/zia/configuring-default-authentication-profile).
- [Configure Kerberos authentication](/zia/deploying-kerberos-authentication).
- [Force reauthentication for all users](/zia/managing-forced-reauthentication).
- Save and [activate your changes](/zia/saving-and-activating-changes-zia-admin-portal) or cancel.
- Go to the [Identity Providers page](/zia/about-identity-providers).
- Go to the [Authentication Bridges page](/zia/about-zscaler-authentication-bridge).
- Go to the [Authentication Profiles page](/zia/about-authentication-profiles).
Authentication Profiles are a feature in limited availability. To access this feature, contact your Zscaler Account team.
