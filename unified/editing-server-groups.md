# Editing Server Groups
Source: https://help.zscaler.com/zpa/editing-server-groups
PDF: https://help.zscaler.com/pdf/com/en/1483676.pdf

To edit a configured server group:
- Go to **Configuration & Control** > **Private Infrastructure** > **App Connector Management** > **Server Groups**.
- Click the **Server Groups** tab.
- In the table, locate the server group you want to modify and click the **Edit **icon.
- In the** Edit Server Group **window, modify fields as necessary. You can only modify which servers are in the server group if **Dynamic Server Discovery** is disabled. To learn more about each field, see [Configuring Server Groups](/zpa/configuring-server-groups).
If there are required fields missing, the incomplete configuration icon appears next to the missing item.
![Incomplete configuration warning when editing a Server Group](/downloads/zpa/documentation-knowledgebase/servers/server-groups/editing-server-groups/zpa_EditServerGroups_ICWservers.png)
[See image.](#editservergroup)
- Click **Save**.
