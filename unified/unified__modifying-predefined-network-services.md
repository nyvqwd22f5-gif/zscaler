# Modifying Predefined Network Services
Source: https://help.zscaler.com/unified/modifying-predefined-network-services
PDF: https://help.zscaler.com/pdf/com/en/1494136.pdf

You can add, modify, or delete ports in all predefined services except ICMP_any, UDP_any, TCP_any, and OTHER.
To modify the ports of a predefined [network service](/unified/about-network-services):
- Go to **Policies **> **Access Control** > **Firewall** >** Network Services**.
-
Find the predefined network service you want to change and click the **Edit** icon.
![Screenshot of Network Services page highlighting the edit icon used to manage difference ports](/downloads/zia/documentation-knowledgebase/creating-and-managing-policies/firewall/firewall-policies-resources/how-do-i-modify-predefined-network-services/network_services_edit_screenshot.png)
- You can change any of the **TCP Source Ports**, **TCP Destination Ports, UDP Source Ports**, or** UDP Destination Ports**.
- Click **Save** and activate the change.
