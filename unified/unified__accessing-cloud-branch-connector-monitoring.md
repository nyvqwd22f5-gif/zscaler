# Accessing Cloud & Branch Connector Monitoring
Source: https://help.zscaler.com/unified/accessing-cloud-branch-connector-monitoring
PDF: https://help.zscaler.com/pdf/com/en/1516831.pdf

The Cloud Connector Monitoring or Branch Connector Monitoring page provides information on the name, group, location, geolocation, and status of your Cloud or Branch Connector virtual machines (VMs) deployed in your cloud or branch accounts.
Accessing the Cloud Connector Monitoring Page
On the Cloud Connector Monitoring page (Infrastructure > Connector > Cloud > Cloud Connector Monitoring), you can view the Cloud Connector monitoring table. The Cloud Connector table displays:
- **Name**: The name of the Cloud Connector.
- **Type**: The cloud provider type.
- **Group**: The name of the group to which the Cloud Connector VM is assigned.
- **Location**: The location of the Cloud Connector group.
- **Geo Location**: The physical location of the Cloud Connector when deployed.
- **Status**: The status of the Cloud Connector (i.e., **Active** or **Inactive**).
- **HA Status**: The high availability (HA) status of the Branch Connector. This column applies to Branch Connector only.
- **VM Size**: The size of the virtual machine (i.e., **Small** or **Medium**).
Click the **View** icon to access the [Cloud Connector Details](/unified/analyzing-cloud-connector-details) page, where the general, management, and traffic forwarding information for the selected Cloud Connector is available.
Accessing the Branch Connector Monitoring Page
On the Branch Connector Monitoring page (Infrastructure > Connector > Edge > Branch Connector Monitoring), you can view the Branch Connector monitoring table. The Branch Connector table displays:
- **Name**: The name of the Branch Connector.
- **Type**: The hypervisor host or hardware device.
- **Group**: The name of the group to which the Branch Connector VM is assigned.
- **Location**: The location of the Branch Connector group.
- **Geo Location**: The physical location of the Branch Connector when deployed.
- **Status**: The status of the Branch Connector (i.e., **Active** or **Inactive**).
- **HA Status**: The high availability (HA) status of the Branch Connector.
- **VM Size**: The size of the virtual machine (i.e., **Small** or **Medium**).
Click the **View** icon to access the [Branch Connector Details](/unified/analyzing-branch-connector-details) page, where the general, management, and traffic forwarding information for the selected Branch Connector is available.
