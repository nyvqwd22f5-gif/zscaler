# Searching for an Internet & SaaS Posture Profile
Source: https://help.zscaler.com/unified/searching-internet-saas-posture-profile
PDF: https://help.zscaler.com/pdf/com/en/1493066.pdf

You can quickly locate a specific an Internet & SaaS posture profile using the Search field.
To find an Internet & SaaS Posture Profile using the Search field:
- In the Admin Portal, go to **Policies > Common Configuration > Resources > Internet & SaaS Posture Profiles**.
- Select a platform.
- In the **Search **field, enter search words and press `Enter` to display your search results.
To clear the **Search **field, delete the search term you typed and press `Enter`.
