# Adding an ICAP Receiver for DLP
Source: https://help.zscaler.com/unified/adding-icap-receiver-dlp
PDF: https://help.zscaler.com/pdf/com/en/1493401.pdf

You can forward information about transactions that violate the DLP policy to the [Internet Content Adaptation Protocol (ICAP) receivers](/unified/about-icap-receivers-dlp) youâ€™ve defined in the Admin Portal.
To add a secure or unencrypted ICAP receiver, you must define them on the DLP Incident Receiver page by providing the public IP address of your DLP server with the port number on which your network firewall initially accepts the secure ICAP traffic sent by the Zscaler service. To learn more, see [][Enabling Secure ICAP](/unified/enabling-secure-icap) and [][Enabling Unencrypted ICAP](/unified/enabling-unencrypted-icap).
