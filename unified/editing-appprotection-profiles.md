# Editing AppProtection Profiles
Source: https://help.zscaler.com/unified/editing-appprotection-profiles
PDF: https://help.zscaler.com/pdf/com/en/1493151.pdf

To edit an AppProtection profile:
- Go to **Policies **>** Cybersecurity **> **Inline Security **> **Protection Profiles** > **AppProtection**.
- In the table, locate the AppProtection profile you want to modify and click the **Edit** icon.
The **Edit AppProtection Profile** window appears.
![Edit AppProtection Profile window in the Admin Portal](/downloads/zpa/appprotection-private-application-traffic-formerly-inspection/appprotection-private-application-traffic-profiles-formerly-inspection/edit-inspection-profiles/Edit%20AppProtection%20Profile.png)
- On the **Profile Details** tab, modify fields as necessary. To learn more about each field, see [Configuring AppProtection Profiles](/unified/configuring-appprotection-profiles).
- Click **Next** to view the **Select Control** tab and modify fields as necessary. To learn more about each field, see [Configuring AppProtection Profiles](/unified/configuring-appprotection-profiles#controls#controls).
- Click **Next** to view the **Paranoia Level and Execution Order** tab and modify fields as necessary. To learn more about each field, see [Configuring AppProtection Profiles](/unified/configuring-appprotection-profiles#controls#paranoia-execution).
- Click **Next** to view the **Override Controls** tab and modify fields as necessary. To learn more about each field, see [Configuring AppProtection Profiles](/unified/configuring-appprotection-profiles#controls#overrides).
- Click **Next** to view the **Review** tab and review your AppProtection profile changes.
- Click **Save**.
