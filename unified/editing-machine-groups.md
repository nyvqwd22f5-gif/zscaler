# Editing Machine Groups
Source: https://help.zscaler.com/zpa/editing-machine-groups
PDF: https://help.zscaler.com/pdf/com/en/1484676.pdf

To edit a configured Machine group:
- Go to **Authentication **>** Device Authentication **>** Machine Groups**.
- In the table, locate the machine group you want to modify and click the **Edit** icon.
The **Edit Machine Group** window appears.
- In the **Edit Machine Group window**, you can edit any field.
![Edit Machine Group window within the ZPA Admin Portal](/downloads/zpa/edit-machine-groups/zpa-editmachinegroup_0.png)
- Click **Save**.
