# About the User Panel
Source: https://help.zscaler.com/unified/about-user-panel
PDF: https://help.zscaler.com/pdf/com/en/1516116.pdf

The User Panel provides insights into a user's details such as status, findings, roles, and permissions granted to the user in 3rd-Party App Governance.
The User Panel provides the following benefits and allows you to:
- View details and generate a report on all user activities performed on the associated apps.
- Detect and investigate suspicious activity, and track unauthorized access to apps.
- View and manage user findings.
The User Panel consists of the following:
- [User Panel Header](#User-Panel-Header)
- [Findings Tab](#Findings-Tab)
- [Apps Tab](#Apps-Tab)
- [Files Tab](#Files-Tab)
- [Activities Tab](#Activities-Tab)
- [Details Tab](#Details-Tab)
[]On the User Panel header, you can view the user's basic information such as name, email, platform, and status.
![User Panel Header showing user details](/downloads/tech-pubs-drafts/zia-draft-articles/about-user-panel/User_Panel_Header.png)
[]On the **Findings** tab, you can do the following:
- View findings associated with the user. Click each finding for more information. You can also change a finding's status. To learn more, see[ Updating the App Finding Status](/unified/updating-app-finding-status).
- View information about the user's sign-in activities.
![User Panel showing Findings Tab](/downloads/tech-pubs-drafts/zia-draft-articles/about-user-panel-draft-doc-57657/User_Panel_FindingsTab.png)
[]On the **Apps** tab, you can do the following:
- View information about top apps and user's permissions. Click each app name for more information.
- View the user groups.
![User Panel showing Apps Tab](/downloads/tech-pubs-drafts/zia-draft-articles/about-user-panel-draft-doc-57657/User_Panel_AppsTab.png)
[]On the **Files** tab, you can do the following:
- View information about the Data Loss Prevention (DLP) violations for the user's files over a selected period of time. The DLP violations are classified based on the document type.
- View information about the top files with DLP violations.
This feature is in limited availability. To upgrade your license to access this feature, contact your Zscaler Account team. To view the information on the **Files** tab, you must have a license for SaaS Security Data at Rest Scanning DLP and the DLP scan must be previously configured.
![User Panel showing Files Tab](/downloads/tech-pubs-drafts/zia-draft-articles/about-user-panel-draft-doc-54997/Userpanel_Files_Tab1.png)
[]On the **Activities** tab, you can do the following:
- Search for a specific activity performed by the user.
- Filter your search by **User Activity**, **Security Event**, and** Findings**.
- Export your search results to a CSV file.
- View the user activities performed on the associated apps.
![User Panel showing Activities Tab](/downloads/tech-pubs-drafts/zia-draft-articles/about-user-panel-draft-doc-54997/User_Panel_ActivitiesTab.png)
[]On the **Details** tab, you can view general information about the user, such as email, display name, role, etc.
![User Panel showing Details Tab](/downloads/tech-pubs-drafts/zia-draft-articles/about-user-panel-draft-doc-57657/User_Panel_DetailsTab.png)
