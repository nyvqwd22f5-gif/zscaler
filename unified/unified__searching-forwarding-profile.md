# Searching for a Forwarding Profile
Source: https://help.zscaler.com/unified/searching-forwarding-profile
PDF: https://help.zscaler.com/pdf/com/en/1494961.pdf

You can quickly locate a specific forwarding profile using the **Search **field.
To find a Forwarding Profile using the Search** **field:
- In the Admin Portal, go to **Infrastructure > Connectors > Client > Forwarding Profile for Platforms**.
- In the **Search** field, enter search words and press `Enter` to display your search results.
To clear the **Search **field, delete the search term you typed and press `Enter`.
