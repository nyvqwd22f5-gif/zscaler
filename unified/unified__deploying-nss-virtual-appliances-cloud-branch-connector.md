# Deploying NSS Virtual Appliances for Cloud & Branch Connector
Source: https://help.zscaler.com/unified/deploying-nss-virtual-appliances-cloud-branch-connector
PDF: https://help.zscaler.com/pdf/com/en/1519491.pdf

To deploy your NSS virtual appliance, the Zscaler service needs to compute the appropriate resources for your NSS. To learn more, see the deployment guide linked below for your platform:
You must be subscribed to NSS for Firewall, Cloud Connector, and Branch Connector to manage your Cloud & Branch Connector NSS.
- [NSS Deployment Guide for Microsoft Azure](/unified/nss-deployment-guide-microsoft-azure)
- [NSS Deployment Guide for Amazon Web Services](/unified/nss-deployment-guide-amazon-web-services)
- [NSS Deployment Guide for VMware vSphere](/unified/nss-deployment-guide-vmware-vsphere)
For Cloud & Branch Connector, consider only the** **NSS for Firewall configurations in the deployment guides and ignore the NSS for Web configurations as they are not applicable.
