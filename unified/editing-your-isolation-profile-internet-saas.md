# Editing Your Isolation Profile for Internet & SaaS
Source: https://help.zscaler.com/unified/editing-your-isolation-profile-internet-saas
PDF: https://help.zscaler.com/pdf/com/en/1492781.pdf

After [creating an isolation profile for Internet & SaaS](/unified/creating-isolation-profiles-internet-saas), you can edit the profile at any time.
To edit your Internet & SaaS isolation profile:
- Go to **Policies **> **Common Configuration **> **Resources **> **Browser Isolation**.
- On the **Isolation Profiles** page, click the **Edit** icon for the isolation profile.
The **Edit Isolation Profile** window appears.
[See image.](#Edit-icon)
- In the **Edit Isolation Profile **window, modify the information as necessary. To learn more, see [Creating Isolation Profiles for Internet & SaaS](/unified/creating-isolation-profiles-internet-saas).
You can use this isolation profile to create policies in Internet & SaaS integrated with Isolation. To learn more, see [Configuring Internet & SaaS for Isolation](/unified/configuring-internet-saas-isolation).
