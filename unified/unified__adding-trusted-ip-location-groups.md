# Adding Trusted IP Location Groups
Source: https://help.zscaler.com/unified/adding-trusted-ip-location-groups
PDF: https://help.zscaler.com/pdf/com/en/1505201.pdf

This article describes how to add IP location groups. You can add up to 4,096 IP location groups. You can also use a [CSV file to import multiple location groups](/unified/importing-trusted-ip-location-groups-csv-file).
To add a location group:
- Go to **Infrastructure **> **Locations **> **Trusted IP Location Groups**.
- Click **Add Location Group**.
The **Add Location** **Group **window appears.
- In the **Add Location** window:
- **Name**: Enter a name for the location group.
- **Locations**: Select the locations that you want to assign to this group. Click the checkbox against the location to add them to the group. Click the **X **icon to remove a location or **Clear All** at any time. You can add up to 2,048 locations.
- **Description**: Enter a description or notes for the location group.
[See image.](#add-groups)
- Click **Save**.
