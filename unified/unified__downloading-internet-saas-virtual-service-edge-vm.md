# Downloading an Internet & SaaS Virtual Service Edge VM
Source: https://help.zscaler.com/unified/downloading-internet-saas-virtual-service-edge-vm
PDF: https://help.zscaler.com/pdf/com/en/1495701.pdf

Downloading an Internet & SaaS Virtual Service Edge virtual machine (VM) is one of the tasks you must complete when deploying Virtual Service Edge clusters for production. To learn more, see [Configuring Internet & SaaS Virtual Service Edge Clusters](/unified/configuring-internet-saas-virtual-service-edge-clusters).
To download a Virtual Service Edge VM for each Virtual Service Edge in a cluster:
- Go to **Infrastructure **>** Internet & SaaS **>** Traffic Forwarding **>** Virtual Service Edges**.
-
Click **Download Virtual Service Edge VM**.
The **Download Virtual Service Edge VM** window appears and displays the technical specifications of the VM.
-
Click **Download Virtual Service Edge VM**.
