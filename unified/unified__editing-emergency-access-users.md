# Editing Emergency Access Users
Source: https://help.zscaler.com/unified/editing-emergency-access-users
PDF: https://help.zscaler.com/pdf/com/en/1491436.pdf

To edit an emergency user:
- Go to **Administration** > **Identity** > **Private Access** > **Emergency Access Users**.
- In the table, locate the Emergency Access user you want to modify and click **Edit**.
The **Edit Emergency Access User** window appears.
- In the **Edit Emergency Access User** window, modify fields as necessary.
![The Edit Emergency Access window ](/downloads/tech-pubs-drafts/zpa-draft-articles/editing-emergency-access-users-doc-46058/zpa-editing-an-emergency-access-user.png)
You can only edit the **First Name** and **Last Name** fields.
- Click **Save**.
