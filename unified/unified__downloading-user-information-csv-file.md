# Downloading User Information to a CSV File
Source: https://help.zscaler.com/unified/downloading-user-information-csv-file
PDF: https://help.zscaler.com/pdf/com/en/1491821.pdf

To download a CSV file of users and their information:
- Go to **Administration** > **Identity** > **Internet & SaaS** > **User Management**.
- Ensure you're in the** Users **tab.
-
Click **Download**.
[See image](#Image1).
[]![Download Tab](/downloads/unified/administration/internet-saas/identity/user-management-authentication-settings/user-management/users/downloading-user-information-csv-file/Unified-User-Management-Users-Page-Download.png)
This CSV file can't be used to [import users](/zslogin/importing-user-details-csv-file) because it uses a different format. If you need to import a CSV file to make modifications to users, use the same format as the **Download** **Sample** file provided by Zscaler.
