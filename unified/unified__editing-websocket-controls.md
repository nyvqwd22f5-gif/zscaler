# Editing WebSocket Controls
Source: https://help.zscaler.com/unified/editing-websocket-controls
PDF: https://help.zscaler.com/pdf/com/en/1492986.pdf

All [WebSocket custom controls](/unified/about-websocket-controls) are editable, even ones currently in use by an [AppProtection profile](/unified/about-appprotection-profiles).
To edit a custom control:
- Go to **Policies **> **Cybersecurity **>** Inline Security **>** Protection Controls** > **WebSocket Controls**.
- In the table, locate the WebSocket custom control you want to modify and click the **Edit** icon.
The **Edit WebSocket Custom Control** window appears.
![Edit WebSocket Custom Control window in the ZPA Admin Portal](/downloads/zpa/inspection-management/edit-custom-controls/EditWScontrol.png)
- On the **Control Details** tab, modify fields as necessary. To learn more about each field, see [Configuring WebSocket Controls](/unified/configuring-websocket-controls).
- Click **Next** to view the **Control Definition** tab and modify fields as necessary. To learn more about each field, see [Configuring WebSocket Controls](/unified/configuring-websocket-controls).
- Click **Next** to view the **Severity** tab and modify fields as necessary. To learn more about each field, see [Configuring WebSocket Controls](/unified/configuring-websocket-controls).
- Click **Next** to view the **Review** tab and review your WebSocket custom control changes.
- Click **Save**.
