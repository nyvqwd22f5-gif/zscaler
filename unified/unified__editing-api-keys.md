# Editing API Keys
Source: https://help.zscaler.com/unified/editing-api-keys
PDF: https://help.zscaler.com/pdf/com/en/1491546.pdf

To edit an API key:
- Go to **Administration** > **API Configuration** > **Legacy API** > **Private Access API**.
- In the table, locate the key you want to modify and click the **Edit** icon.
-
In the **Edit API Keys **window, modify fields as necessary. To learn more about each field, see [Adding an API Key](/unified/adding-api-keys).
![Editing an API Key](/downloads/unified/administration/private-applications/api-configuration/editing-api-keys/experience-center-edit-api-key.png)
- Click **Save**.
