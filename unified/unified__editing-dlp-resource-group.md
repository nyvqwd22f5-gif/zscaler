# Editing a DLP Resource Group
Source: https://help.zscaler.com/unified/editing-dlp-resource-group
PDF: https://help.zscaler.com/pdf/com/en/1493226.pdf

[]You can edit a [DLP resource group](/unified/adding-dlp-resource-group) to change basic information, or add and remove resources from the group.
To edit a DLP resource group:
- Go to **Policies **>** Data Protection **>** Policy **>** Endpoint DLP Resources**.
- On the **DLP Resources** page:
- [Edit a network share group](#network-shares-group-edit)
- [Edit a network printers group](#printers-group-edit)
- [Edit a removable storage device group](#removable-storage-group-edit)
- [Activate](/unified/saving-and-activating-changes-admin-portal) your changes.
[]
- Click **Network Shares**.
- Click the **Network Share Groups** tab, then locate the group in the table and click **Edit**.
The **Edit Group - Network Shares** window is displayed.
- In the **Edit Group - Network Shares** window:
- Update the following **Basic information** as needed:
- **Name**: The name of the network share group
- (Optional) **Description**: A description of the network share group
- Click **Next**.
[See image.](#edit-network-share-group-01)
- Select the network shares you want to add or remove, then click **Save**.
[See image.](#edit-network-share-group-02)
You receive a confirmation message and return to the **Network Shares** page.

- Click **Printers**.
- Click the **Printer Groups** tab, then locate the group in the table and click **Edit**.
The **Edit Group - Printers** window is displayed.
- In the **Edit Group - Printers** window:
- Update the following **Basic information** as needed:
- **Name**: The name of the network printer group
- (Optional) **Description**: A description of the network printer group
- Click **Next**.
[See image.](#edit-printer-group-01)
- Select the network printers you want to add or remove, then click **Save**.
[See image.](#edit-printer-group-02)
You receive a confirmation message and return to the **Printers** page.

- Click **Removable Storage Devices**.
- Click the **Removable Storage Device Groups** tab, then locate the group in the table and click **Edit**.
The **Edit Group - Removable Storage Devices** window is displayed.
- In the **Edit Group - Removable Storage Devices** window:
- Update the following **Basic information** as needed:
- **Name**: The name of the removable storage device group
- (Optional) **Description**: A description of the removable storage device group
- Click **Next**.
[See image.](#edit-removable-storage-group-01)
- Select the removable storage devices you want to add or remove, then click **Save**.
[See image.](#edit-removable-storage-group-02)
You receive a confirmation message and return to the **Removable Storage Devices** page.
