# Downloading Group Information to a CSV File
Source: https://help.zscaler.com/unified/downloading-group-information-csv-file
PDF: https://help.zscaler.com/pdf/com/en/1491846.pdf

To download a CSV file of groups and their information:
- Navigate to **Administration** > **Identity** > **Internet & SaaS** > **User Management**.
- Click the** Groups **tab.
- Click **Download**.
This CSV file can't be used to [import groups](/zslogin/importing-group-details-csv-file) because it uses a different format. If you need to import a CSV file to make modifications to groups, use the same format as the **Download** **Sample **file provided by Zscaler.
