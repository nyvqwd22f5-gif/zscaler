# Editing Browser Protection Profiles
Source: https://help.zscaler.com/unified/editing-browser-protection-profiles
PDF: https://help.zscaler.com/pdf/com/en/1493171.pdf

To edit a Browser Protection profile:
- Go to **Policies **>** Cybersecurity **>** Inline Security **>** Protection Profiles**.
- In the table, locate the Browser Protection profile you want to modify and click the **Edit** icon.
The **Edit Browser Protection Profile** window appears.
![Edit Browser Protection Profile window within the ZPA Admin Portal](/downloads/zpa/appprotection-private-application-traffic-formerly-inspection/browser-protection-profiles/editing-browser-protection-profiles/Editing%20Browser%20Protection%20window.png)
- On the** Details** tab, modify fields as necessary. To learn more about each field, see [Configuring Browser Protection Profiles](/unified/configuring-browser-protection-profiles).
- Click **Next** to view the **Criteria** tab and modify fields as necessary. To learn more about each field, see [Configuring Browser Protection Profiles](/unified/configuring-browser-protection-profiles).
- Click **Next** to view the** Review** tab and review your Browser Protection profile changes.
- Click **Save**.
