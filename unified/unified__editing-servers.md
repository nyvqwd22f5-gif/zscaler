# Editing Servers
Source: https://help.zscaler.com/unified/editing-servers
PDF: https://help.zscaler.com/pdf/com/en/1492261.pdf

To edit a server:
- Go to the **Servers **page (Policies > Access Control > Private Applications > Servers).
- In the table, locate the server you want to modify and click the **Edit icon**.
-
In the** Edit Server **window, modify fields as necessary. To learn more about each field, see [Explicitly Defining Servers](/unified/explicitly-defining-servers).
[See Image.](#edit-server-window)
If there are required fields missing, the incomplete configuration icon appears next to the missing item.
![Incomplete configuration warning when editing a Server](/downloads/zpa/documentation-knowledgebase/servers/servers/editing-servers/zpa_EditServers_ICWservergrp_0.png)
- Click **Save**.
[]![Edit Server window within the ZPA Admin Portal](/downloads/zpa/documentation-knowledgebase/servers/servers/editing-servers/zpa-editserver.png)
