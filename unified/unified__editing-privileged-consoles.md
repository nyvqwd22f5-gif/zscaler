# Editing Privileged Consoles
Source: https://help.zscaler.com/unified/editing-privileged-consoles
PDF: https://help.zscaler.com/pdf/com/en/1492541.pdf

To edit a privileged console:
- Go to **Policies **>** Clientless **>** Privileged Consoles**.
- In the table, locate the privileged console you want to modify and click the **Edit **icon.
- In the **Edit Console **window, modify fields as necessary. To learn more about each field, see [Configuring Privileged Consoles](/unified/configuring-privileged-consoles).
![Edit Console window within the ZPA Admin Portal](/downloads/zpa/about-privileged-consoles/Editing%20Console.png)
- Click **Preview Console** to see the console link preview.
- Click **Save**.
