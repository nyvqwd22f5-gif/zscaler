# Company Summary Report (CIO Report)
Source: https://help.zscaler.com/unified/company-summary-report-cio-report
PDF: https://help.zscaler.com/pdf/com/en/1497496.pdf

The Company Summary Report for Chief Information Officers (CIO Report) is an interactive report (Analytics > Internet & SaaS > Analytics > Interactive Reports) that provides information tailored for audiences like the CIO of your organization.
The report covers the following information for your organization:
- Peak Transactions
- Blocked Transactions Trend
- Overall Traffic
- Top Bandwidth Locations
- Cloud Applications
- Social Networking Applications
- Streaming Media Applications
- Top Blocked URL Categories
- Top Blocked URL Classes
- Viruses & Spyware
- Advanced Threats
- Sandbox
- Mobile Traffic by Device Type
- Mobile Traffic by Location
- Mobile Traffic by Apps
About the CIO Report Page
As with other [interactive reports](/unified/about-interactive-reports), the CIO Report supports real-time interactive analysis and contains [widgets](/unified/about-widgets) that present information in different graphical formats. You can drill down to the logs for each chart by rolling over the chart, clicking, and selecting one of the following options:
- Click** View Logs **to see its associated logs. See [About Insights Logs](/unified/about-insights-logs).
- Click **Analyze Chart **to see the chart in Web Insights and to drill down to the logs. See [Analyzing Traffic Using Insights](/unified/analyzing-traffic-using-insights).
You can also:
- [Add custom notes to the report](/unified/adding-notes-dashboards-and-interactive-reports)
- Add it your list of favorite reports
- [Copy the report](/unified/creating-or-copying-report)
- [Schedule regular email deliveries of the report](/unified/about-scheduled-reports)
- [Print the report](/unified/printing-reports)
