# Editing a Deployed Network Connector
Source: https://help.zscaler.com/zpa/editing-deployed-network-connector
PDF: https://help.zscaler.com/pdf/com/en/1529058.pdf

To edit a deployed Network Connector:
- Go to **VPN (for Legacy Apps) **> **Network Connectors** > **Network Connectors**.
-
In the table, locate the Network Connector you want to modify and click the **Edit** icon.
The **Edit Network Connector** window appears.
- In the **Edit Network Connector** drawer, you can only modify the following fields:
- **Name**: Update the name of the group. The name cannot contain special characters, with the exception of periods (.), hyphens (-), and underscores ( _)
- **Description**: (Optional) Enter a description of the Network Connector.
- **Status**: Enable or disable the Network Connector.
- **Signing Certificate**: Click the name of the certificate to edit the certificate.
- **Network Connector Group**: Click the name of the group to edit the group.
![Image of editing a Network Connector in the ZPA Admin Portal](/downloads/zpa/editing-deployed-network-connector/zpa-edit-network-conn.png)
- Click **Save**.
