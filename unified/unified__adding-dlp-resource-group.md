# Adding a DLP Resource Group
Source: https://help.zscaler.com/unified/adding-dlp-resource-group
PDF: https://help.zscaler.com/pdf/com/en/1493221.pdf

[]Grouping [Data Loss Prevention (DLP) resources](/unified/about-dlp-resources) allows you to apply Endpoint Data Loss Prevention (DLP) policy rules to multiple resources at the same time. After you create a DLP resource group, you add that group to the [Endpoint DLP policy rules that you configure](/unified/configuring-endpoint-dlp-policy-rules). For example, suppose that you work at a company with corporate headquarters in the United States, but you want to apply a country-specific set of endpoint policy rules to the network printers in your Canadian branch office. In that case, you can create a DLP resource group that includes all network printers in the branch office, and then you apply the policy rules to the resource group you created.
To add a DLP resource group:
- Go to **Policies **>** Data Protection **>** Policy **>** Endpoint DLP Resources**.
- On the **DLP Resources** page:
- [Create a network share group](#network-shares-group)
- [Create a network printers group](#printers-group)
- [Create a removable storage device group](#removable-storage-group)
- [Activate](/unified/saving-and-activating-changes-admin-portal) your changes.
[]On the **Network Shares** page:
- Click the **Network Share Groups** tab, then click **Add Group**.
The **Add Group - Network Shares** window is displayed.
- In the **Add Group - Network Shares** window:
- Enter the following **Basic information**:
- **Name**: The name of the network share group
- (Optional) **Description**: A description of the network share group
- Click **Next**.
[See image.](#add-network-share-group-01)
- Select the network shares you want to add to the group, then click **Add Group**.
[See image.](#add-network-share-group-02)
The new group is displayed in the list of network share groups on the **Network Shares** page.
[See image.](#add-network-share-group-03)
