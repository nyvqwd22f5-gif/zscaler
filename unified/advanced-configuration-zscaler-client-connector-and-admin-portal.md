# Advanced Configuration for Zscaler Client Connector and Admin Portal
Source: https://help.zscaler.com/unified/advanced-configuration-zscaler-client-connector-and-admin-portal
PDF: https://help.zscaler.com/pdf/com/en/1496306.pdf

In the Admin Portal, you can configure advanced settings for the portal itself and Zscaler Client Connector.
About the Advanced Configuration Page
On the Advanced Configuration page ((Infrastructure > Connectors > Client > Directory Sync & Custom Root Cert), you can do the following:
- [Check the next sync time and manually sync groups between Internet & SaaS and Zscaler Client Connector.](/unified/syncing-directory-groups-between-internet-saas-and-zscaler-client-connector)
- [Upload your organization's custom SSL certificate for Zscaler to automatically use with Zscaler Client Connector.](/unified/uploading-custom-ssl-certificate-zscaler-client-connector)
