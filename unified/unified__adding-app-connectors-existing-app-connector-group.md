# Adding App Connectors to an Existing App Connector Group
Source: https://help.zscaler.com/unified/adding-app-connectors-existing-app-connector-group
PDF: https://help.zscaler.com/pdf/com/en/1496071.pdf

To add an App Connector to an existing App Connector group:
- Go to **App Connectors **page (Infrastructure > Private Access > Component > App Connectors).
- Click **Add Connector**.
The **Add Connector** window appears.
- In the **Choose Key** tab:
- Select **Choose an existing provisioning key**.
- Select an existing provisioning key from the drop-down menu. Ensure that the key you select is associated with the appropriate App Connector group. You can click **Clear Selection** to remove any selections.
[See image.](#seeimage8)
- Click **Next**.
- In the **Deploy Connector** tab:
- **Copy Provisioning Key**: Copy the App Connector provisioning key. You will need to enter this key when you deploy the App Connector to a platform.
- **Choose Platform**: Select the platform you want to deploy your App Connector on and follow the instructions that appear below. To learn more, see the [App Connector Deployment Guides for Supported Platforms](/unified/networking/private-applications/app-connector-management/app-connector-deployment-guides-supported-platforms).
[See image.](#seeimage7)
- Deploy your App Connector on the chosen platform.
- Click **Done**.
