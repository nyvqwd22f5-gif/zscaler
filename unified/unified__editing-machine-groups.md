# Editing Machine Groups
Source: https://help.zscaler.com/unified/editing-machine-groups
PDF: https://help.zscaler.com/pdf/com/en/1491381.pdf

To edit a configured Machine group:
- Go to **Administration** > **Identity** > **Private Applications **>** Machine Groups**.
- In the table, locate the machine group you want to modify and click the **Edit** icon.
The **Edit Machine Group** window appears.
- In the **Edit Machine Group window**, you can edit any field.
![Edit Machine Group window ](/downloads/zpa/edit-machine-groups/zpa-editmachinegroup_0.png)
- Click **Save**.
