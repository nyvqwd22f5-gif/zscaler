# End-of-Life for Zscaler Authentication Bridge Hardware
Source: https://help.zscaler.com/eos-eol/end-life-zab-hardware
PDF: https://help.zscaler.com/pdf/com/en/1283181.pdf

Zscaler will no longer support the Zscaler Authentication Bridge (ZAB) hardware as of March 1, 2019.
Support for the ZAB hardware will be permanently terminated after March 31, 2019.
The ZAB functionality is now available via software. To continue using ZAB, migrate to the ZAB software before March 1, 2019.
If you need help with the migration or if you have additional questions or concerns, contact Zscaler Support via the Support link in the Admin Portal.
Announcement date: December 20, 2017
