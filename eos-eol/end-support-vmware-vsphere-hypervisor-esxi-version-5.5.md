# End-of-Support for VMware vSphere Hypervisor (ESXi) Version 5.5
Source: https://help.zscaler.com/eos-eol/end-support-vmware-vsphere-hypervisor-esxi-version-5.5
PDF: https://help.zscaler.com/pdf/com/en/1474366.pdf

Zscaler will no longer support ESXi version 5.5 after December 24, 2024. Customers currently using ESXi version 5.5 will need to use ESXi version 7.0 Update 2 or later after this date.
If you need help upgrading to ESXi 7.0 Update 2 or later, or if you have additional questions or concerns, contact Zscaler Support.
Announcement date: January 26, 2024
