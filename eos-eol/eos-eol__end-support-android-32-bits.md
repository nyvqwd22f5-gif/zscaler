# End-of-Support for Android 32-Bit
Source: https://help.zscaler.com/eos-eol/end-support-android-32-bits
PDF: https://help.zscaler.com/pdf/com/en/1380776.pdf

Zscaler Client Connector version 1.8 and later for the Android platform will not support 32-bit devices. Support for 32-bit is still available with previous Zscaler Client Connector versions of the Android platform.
Announcement date: July 15, 2021
