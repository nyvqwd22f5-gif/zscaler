# End-of-Support for macOS Catalina
Source: https://help.zscaler.com/eos-eol/end-support-macos-catalina
PDF: https://help.zscaler.com/pdf/com/en/1457621.pdf

As Apple announced the end of support for macOS Catalina, Zscaler Client Connector version 3.9 and later no longer supports macOS Catalina. This means that any fundamental issues with macOS Catalina itself cannot be addressed by Zscaler, and you might be required to upgrade to a later version.
Announcement date: July 12, 2023
