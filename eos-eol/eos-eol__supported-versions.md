Zscaler Client Connector Supported Versions
Zscaler supports the latest software version and the two previous versions for each software product. For Zscaler Client Connector, Zscaler also supports the current Long-Term Support (LTS) version. In some cases, Zscaler might designate another version as a supported version. To learn more about LTS, see [Zscaler Client Connector Long-Term Support](/eos-eol/zscaler-client-connector-long-term-support).
