# Revoking API Keys
Source: https://help.zscaler.com/zpc/revoking-api-keys
PDF: https://help.zscaler.com/pdf/com/en/1455831.pdf

To revoke an API key:
- Go to **Administration **> **Authentication & Authorization** > **API Keys**.
- In the table, locate the key you want to revoke, click the **Menu **icon, and then click **Revoke**.
The **Revoke API Key** window appears.
[See image.](#deleteAPIKey)
- In the **Revoke API Key** window, confirm you want to revoke the API key, and then click **Revoke**.
Revoking an API key cannot be undone. If you revoke an API key, that API key no longer has access to authenticate.
