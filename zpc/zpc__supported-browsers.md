# Supported Browsers
Source: https://help.zscaler.com/zpc/supported-browsers
PDF: https://help.zscaler.com/pdf/com/en/1392431.pdf

Zscaler actively tests the operating system and browser compatibility with the Zscaler Posture Control (ZPC) Admin Portal.
Windows 10 and 11
ZPC supports the following browsers on Windows 10 and 11:
- Google Chrome (two most recent versions)
- Mozilla Firefox (two most recent versions)
- Microsoft Edge (two most recent versions)
Older versions of these browsers should be compatible, but are no longer actively tested.
Other browsers should be compatible, but are not actively tested.
Mac OS X & macOS
ZPC supports the following Apple operating systems:
- OS X 9
- OS X 10
- macOS 10
ZPC supports the following browsers on OS X and macOS:
- Google Chrome (two most recent versions)
- Mozilla Firefox (two most recent versions)
- Safari (two most recent versions)
Older versions of these browsers should be compatible, but are no longer actively tested.
Other browsers should be compatible, but are not actively tested.
