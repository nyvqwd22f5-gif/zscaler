# Supported OS Versions for IaC Scanning
Source: https://help.zscaler.com/zpc/supported-os-versions-iac-scanning
PDF: https://help.zscaler.com/pdf/com/en/1452496.pdf

ZPC provides IaC scan support on various operating systems (OS) and versions, so you can seamlessly use the Zscaler IaC scan tool on different OS to scan the IaC resources. To learn more, see [About IaC Integrations](/zpc/about-iac-integrations).
The following OS versions are supported for CI/CD tools and CLI Scanner:
- [CI/CD Tools](#oscicd)
- [CLI Scanner](#oscli)
