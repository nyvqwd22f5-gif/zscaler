# Editing or Deleting Slack Integrations
Source: https://help.zscaler.com/zpc/editing-or-deleting-slack-integrations
PDF: https://help.zscaler.com/pdf/com/en/1456761.pdf

From the Zscaler Posture Control (ZPC) Admin Portal, you can edit or delete any Slack integration as required.
Editing a Slack Integration
To edit a Slack integration:
- Go to **Administration **> **Integrations**.
- Under **ChatOps (Slack)**, click the **Edit** icon for the Slack channel that you want to edit.
[See image.](#edit-icon)
- On the **Integration Details** page:
- Change the **Integration Name**.
- Click **Continue Editing** if you want to add more channels.
You cannot edit the existing channel or Webhook details.
[See image.](#edit-slack)
- Click **Next**.
- Review the integration details.
[See image.](#reviewslack)
- Click **Finish**.
Deleting a Slack Integration
To delete a Slack integration:
- Go to **Administration **>** Integrations**.
- Under **ChatOps (Slack)**, click the **Delete** icon for the Slack integration that you want to delete.
[See image.](#delete-icon)
- In the **Delete Integration** window, read the message related to alert rules.
You cannot delete an integration if it is associated with alert rules. You must first move the alert rules to another integration.
[See image.](#deleteslack)
- After you've moved the alert rules, click **Delete** to delete the Slack integration.
