# Understanding the ZPC API
Source: https://help.zscaler.com/zpc/understanding-zpc-api
PDF: https://help.zscaler.com/pdf/com/en/1455891.pdf

Zscaler Posture Control (ZPC) API gives you programmatic access to retrieve cloud alerts for ZPC. To learn more, see:
- [Reference Guide > Alerts API](/zpc/alerts-api)
- [About Alerts](/zpc/about-alerts)
- [About IaC Alerts](/zpc/about-iac-alerts)
Zscaler recommends walking through [Getting Started](/zpc/getting-started-zpc-api) for information regarding authentication and making API calls. For detailed information on all available API calls, endpoints, and parameters, see the reference guide. For a table summarizing all available API calls, endpoints, and rate limits, see [Understanding Rate Limiting](/zpc/understanding-rate-limiting).
