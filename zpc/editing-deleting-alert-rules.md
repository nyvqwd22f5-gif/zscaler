# Editing or Deleting Alert Rules
Source: https://help.zscaler.com/zpc/editing-deleting-alert-rules
PDF: https://help.zscaler.com/pdf/com/en/1396891.pdf

You can edit or delete any cloud alert rule or IaC alert rule you've configured.
Editing an Alert Rule
To edit an alert rule:
- On the Alerts page, select the **Notifications** tab.
The list of all the alert rules is displayed.
- Click the **Edit** icon for the alert rule that you want to edit.
[See image.](#alert-rule-edit)
- On the **Edit Alert Rule** page, change the **Alert Rule Name** or **Alert Rule Status** as required, then click **Next**.
[See image.](#edit)
- Update the details on the **Resource Scope** page as required, then click **Next**.
- Update the details on the **Notifications** page as required, then click **Next**.
[See image.](#notifications-tab)
- Review the changes. Click the **Edit** icon if you want to change any value again.
[See image.](#finish)
- Click **Finish**.
A message appears indicating that the alert rule has been updated successfully.
Deleting an Alert Rule
To delete an alert rule:
- On the Alerts page, select the **Notifications** tab.
The list of all alert rules configured is displayed.
- Click the **Delete** icon for the alert rule that you want to delete.
- A confirmation message appears. Read the message, then click **Delete**.
[See image.](#delete)
A message appears indicating that the alert rule has been deleted successfully.
