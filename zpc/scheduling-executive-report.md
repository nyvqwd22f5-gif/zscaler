# Scheduling Executive Reports
Source: https://help.zscaler.com/zpc/scheduling-executive-report
PDF: https://help.zscaler.com/pdf/com/en/1462831.pdf

You can schedule executive summary reports for regular distribution to specified recipients.
To schedule an executive report:
- Go to **Analytics** > **Executive Report**.
- Click the **Schedule Report** icon in the top right-hand corner.
[See image.](#schedule-report-icon)
- In the **Add Schedule Report** window:
- **Schedule Name**: Enter a unique name for the report.
- **Report**: Select the report for scheduling. By default, the default Executive Report page is selected.
- **Recipients**: Enter the email addresses of the recipients.
- **Status**: Select the status (**Enabled** or **Disabled**) to enable or disable scheduling.
- **Frequency**: Select a schedule as follows:
- Select **Daily** and select the **time** and **timezone**.
- Select **Weekly** and select the **day**, **time**, and **timezone**.
- Select **Monthly** and select the **date**, **time**, and **timezone**.
[See image.](#add-schedule-window)
- Click **Save**.
A message appears indicating that the schedule report is created successfully and can be viewed under [Scheduled Reports](/zpc/managing-scheduled-executive-reports).
