# Onboarding Cloud Accounts
Source: https://help.zscaler.com/zpc/onboarding-cloud-accounts
PDF: https://help.zscaler.com/pdf/com/en/1395551.pdf

Onboard your cloud accounts on Zscaler Posture Control (ZPC) to establish integrations required by ZPC to collect configuration and identity metadata. When your cloud accounts are onboarded, ZPC provides you with actionable insights on your cloud deployment's security posture. You can onboard multiple cloud accounts from any cloud service provider into your ZPC service. To learn how to onboard cloud accounts, see:
- [Onboard an Amazon Web Services (AWS) Account](/zpc/onboarding-amazon-web-services-account)
- [Onboard a Microsoft Azure Account](/zpc/onboarding-microsoft-azure-account)
- [Onboard a Google Cloud Provider (GCP) Account](/zpc/onboarding-google-cloud-platform-account)
- [Onboarding an Oracle Cloud Infrastructure (OCI) Tenant](/zpc/onboarding-oracle-cloud-infrastructure-oci-tenant)
