# Enabling or Disabling Alert Ignore Filters
Source: https://help.zscaler.com/zpc/enabling-or-disabling-alert-ignore-filters
PDF: https://help.zscaler.com/pdf/com/en/1440706.pdf

You can enable or disable the ignore filters you've set up for specific IaC or cloud alerts.
To enable or disable the ignore filter:
- In the left-side navigation, select **Alerts**.
- On the **Alerts** page, click **Ignore Filters**.
- Click the toggle under the **State** column to enable or disable an ignore filter.
[See image.](#ignorefilterspage)
- A confirmation message appears indicating that the filter is going to be ignored. Click **Enable** or **Disable** depending on the action. For example, the following message is displayed when you disable the ignore filter.
[See image.](#message)
