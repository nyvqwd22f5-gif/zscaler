# Editing or Deleting an ITSM Integration
Source: https://help.zscaler.com/zpc/editing-or-deleting-itsm-integration
PDF: https://help.zscaler.com/pdf/com/en/1402936.pdf

You can edit or delete any ITSM integration that you've configured.
Editing an ITSM Integration
To edit an ITSM integration:
- Go to **Administration **> **Integrations**.
- On the **Integrations** page, click the **Edit** icon for the ITSM integration that you wish to edit.
-
In the **Edit Integration** window:
- Change the **ServiceName**.
- Click **Reset Validation** to validate the ServiceNow hostname and credentials if required.
[See image.](#edit)
- Click **Save**.
Deleting and ITSM Integration
To delete an ITSM integration:
- Go to **Administration **>** Integrations**.
- On the **Integrations** page, click the **Delete** icon for the ITSM integration that you wish to delete.
-
A confirmation message appears. Click **Delete**.
[See image.](#delete)
You cannot delete an integration if it is associated with alert rules. You need to move the alert rules to another integration, then you can delete this integration. To learn more, see [About Alert Rules](/zpc/about-alert-rules).
