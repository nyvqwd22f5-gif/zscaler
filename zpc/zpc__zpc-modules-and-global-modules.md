# ZPC Modules and Global Modules
Source: https://help.zscaler.com/zpc/zpc-modules-and-global-modules
PDF: https://help.zscaler.com/pdf/com/en/1461636.pdf

The features available in the ZPC Admin Portal are classified as modules and global modules. Users can access data in these modules depending on their roles and permissions. To learn more, see [About Roles](/zpc/about-roles).
- **Global Modules**: Data is not segregated by cloud accounts and [business units](/zpc/about-business-units) are not applicable for these modules.
- **Modules**: Data can be filtered by cloud accounts that are associated with business units. The business unit filters are applicable for these modules.
