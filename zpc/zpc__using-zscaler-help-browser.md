# Using the Zscaler Help Browser
Source: https://help.zscaler.com/zpc/using-zscaler-help-browser
PDF: https://help.zscaler.com/pdf/com/en/1392411.pdf

The Zscaler Help Browser is available for all pages within the Zscaler Posture Control (ZPC) Admin Portal. Through the Help Browser, you can view the Zscaler Help Portal articles in context with the page you're on, without leaving the ZPC Admin Portal.
To open the Help Browser:
- Go to the page where you want to view an article.
- Click the **Open Help Browser** icon in the lower-right corner.
[See image.](#help)
The Help Browser displays the article relevant to the page you are viewing.
[See image.](#article)
About the Help Browser
The Help Browser is also adjustable. There are different features to help you customize the browser and navigate within it.
Customizing the Help Browser
You can do the following to customize the Help Browser:
- [Move the Help Browser](#move)
- [Resize the Help Browser](#resize)
- [Change the font size](#font)
[]You can move the Help Browser anywhere within the menu bar and footer. When you hover over the browser header, a **Move** cursor appears. Click and drag the Help Browser to the desired location.
![Move the help browser](/downloads/zpc/getting-started/admin-portal/using-zscaler-help-browser/zpc-move-browser.png)
[]You can resize the Help Browser to expand videos, pictures, and content. When you hover to the top-left corner of the browser, an **Expand** cursor appears. Click and drag the corner to resize the window.
![Resize the help browser window](/downloads/zpc/getting-started/admin-portal/using-zscaler-help-browser/zpc-resize-help.png)
[]There are several font sizes for the Help Browser. You can click the **Font** icons to increase or decrease the text size.
![Change the font size](/downloads/zpc/getting-started/admin-portal/using-zscaler-help-browser/zpc-help-font.png)
Zscaler saves your customization until you log out or the session expires.
Navigating within the Help Browser
To navigate within the Help Browser:
- View the article relevant to the page you're viewing in the ZPC Admin Portal.
- Navigate back and forth across articles you've viewed.
- Search for an article.
- Copy the URL of an article.
- Open the article in a new browser tab.
![Navigating within the help browser](/downloads/zpc/getting-started/admin-portal/using-zscaler-help-browser/zpc-help-browser-options.png?1655380047)
Closing the Help Browser clears your navigation history. When you reopen the browser, it displays the article relevant to the page you are viewing, and you will not be able to navigate back and forth to articles you've viewed in the previous Help Browser session.
