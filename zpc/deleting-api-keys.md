# Deleting API Keys
Source: https://help.zscaler.com/zpc/deleting-api-keys
PDF: https://help.zscaler.com/pdf/com/en/1455871.pdf

To delete an API key:
- Go to **Administration **> **Authentication** **& Authorization** > **API Keys**.
- Locate the key you want to delete in the table, click the **Menu** icon, and then click **Delete**.
The **Delete API Key** window appears.
[See image.](#deleteAPIKey)
- In the **Delete API Key** window, confirm you want to delete the API key, and then click **Delete**.
Deleting an API key cannot be undone. If you delete an API key, that API key is permanently deleted and a new one must be created.
