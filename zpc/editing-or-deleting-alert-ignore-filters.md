# Editing or Deleting Alert Ignore Filters
Source: https://help.zscaler.com/zpc/editing-or-deleting-alert-ignore-filters
PDF: https://help.zscaler.com/pdf/com/en/1433971.pdf

You can edit or delete the ignore filters that you've set up for alerts.
Editing an Ignore Filter
To edit an ignore filter:
- In the left-side navigation, select **Alerts**.
- On the **Alerts** page, click **Ignore Filters**.
- Under **Actions**, select the **Actions** icon (![Edit Icon](/downloads/zpc/alerts/editing-or-deleting-ignore-filters/action-icon.png)
) for the filter you want to edit.
- Click **Edit**.
[See image.](#edit-icon)
- On the **General Information** page, change the **Ignore Filter Name** or **Filter Description** as required, then click **Next**.
- Update the details on the **Filter Scope** page as required, then click **Next**.
[See image.](#edit-ignore-filter)
- Review the changes and click **Finish**.
Deleting an Ignore Filter
To delete an ignore filter:
- In the left-side navigation, select **Alerts**.
- On the **Alerts** page, click **Ignore Filters**.
- Under **Actions**, select the  **Actions** icon (![Edit Icon](/downloads/zpc/alerts/editing-or-deleting-ignore-filters/action-icon.png)
) for the filter you want to edit.
- Click **Delete**.
[See image.](#delete)
- A confirmation message appears. Click **Delete**.
[See image.](#confirmation)
The deleted filter is not applicable to any newly triggered alert. Existing ignored alerts under this filter will remain ignored even after deleting the filter.
