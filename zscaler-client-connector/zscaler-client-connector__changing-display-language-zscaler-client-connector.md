# Changing the Display Language for Zscaler Client Connector
Source: https://help.zscaler.com/zscaler-client-connector/changing-display-language-zscaler-client-connector
PDF: https://help.zscaler.com/pdf/com/en/1492641.pdf

If your organization has enabled the option, you can change the language in which the Zscaler Client Connector app displays.
This feature is available only with Zscaler Client Connector version 4.8 and later for Windows. The default language is based on the app profile.
To change the display language:
- Open Zscaler Client Connector and click **More**.
-
Select a different **UI Language**.
The app UI and pop-up notifications change to the language you selected.
