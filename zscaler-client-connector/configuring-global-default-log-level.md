# Configuring a Default Global Log Mode
Source: https://help.zscaler.com/zscaler-client-connector/configuring-global-default-log-level
PDF: https://help.zscaler.com/pdf/com/en/1443701.pdf

You can configure a default global log mode for new profiles. This saves you from having to configure the log mode for each new profile in [App Profiles](/zscaler-client-connector/configuring-zscaler-client-connector-profiles). You can override this setting for individual profiles in [App Profiles](/zscaler-client-connector/configuring-zscaler-client-connector-profiles).
To configure the default global log level:
- In the Zscaler Client Connector Portal, go to **Administration**.
- In the left-side navigation, select **Platform Settings**.
- Select a platform tab.
- From the **Log Mode** drop-down menu, select a mode. For a description of each log mode, see [Configuring Zscaler Client Connector Profiles](/zscaler-client-connector/configuring-zscaler-client-connector-profiles).
![Configure global log mode in Platform Settings](/downloads/client-connector/zscaler-client-connector-profile-management/configuring-global-default-log-level/Platform_Settings.png)
- Click **Save**.
