# Searching for a ZIA Posture Profile
Source: https://help.zscaler.com/zscaler-client-connector/searching-zia-posture-profile
PDF: https://help.zscaler.com/pdf/com/en/1444031.pdf

You can quickly locate a specific ZIA posture profile using the Search field.
To find a ZIA Posture Profile using the Search field:
- In the Zscaler Client Connector Portal, go to **Administration**.
- In the left-side navigation, select **ZIA Posture Profile**.
- In the **Search **field, enter search words and press `Enter` to display your search results.
To clear the **Search **field, delete the search term you typed and press `Enter`.
