# Advanced Configuration for Zscaler Client Connector and Portal
Source: https://help.zscaler.com/zscaler-client-connector/advanced-configuration-zscaler-app-and-portal
PDF: https://help.zscaler.com/pdf/com/en/1317666.pdf

In the Zscaler Client Connector  Portal, you can configure advanced settings for the portal itself and Zscaler Client Connector.
About the Advanced Configuration Page
On the Advanced Configuration page, you can do the following:
- [Check the next sync time and manually sync groups between the ZIA Admin Portal and the Zscaler Client Connector Portal.](/zscaler-client-connector/syncing-directory-groups-between-zscaler-admin-portal-and-app-portal)
- [Upload your organization's custom SSL certificate for Zscaler to automatically use with Zscaler Client Connector.](/zscaler-client-connector/uploading-custom-ssl-certificate-zscaler-app)
