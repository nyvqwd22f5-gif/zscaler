# Understanding Error Codes
Source: https://help.zscaler.com/zscaler-client-connector/about-error-codes
PDF: https://help.zscaler.com/pdf/com/en/1416786.pdf

The following table lists the HTTP status codes returned by the Zscaler Client Connector API:
[](/zscaler-client-connector/understanding-rate-limiting)
| Code | Description |
| ---- | ----------- |
| 200 | Successful |
| 400 | Bad Request |
| 401 | Unauthorized |
| 403 | Forbidden |
| 404 | Not Found |
| 415 | Unsupported Media Type |
| 429 | Too Many RequestsThe user has sent too many requests in a given amount of time and exceeded the rate limit or device threshold. To learn more, see Understanding Rate Limiting. |
| 500 | Not Implemented |
