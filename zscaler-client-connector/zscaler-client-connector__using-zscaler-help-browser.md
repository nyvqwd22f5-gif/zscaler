# Using the Zscaler Help Browser
Source: https://help.zscaler.com/zscaler-client-connector/using-zscaler-help-browser
PDF: https://help.zscaler.com/pdf/com/en/1334541.pdf

The Zscaler Help Browser is available for all pages within the Zscaler Client Connector  Portal. Through the Help Browser, you can view Zscaler Help Portal articles in context with the page you're on without leaving the Zscaler Client Connector Portal.
Accessing the Zscaler Help Browser
To open the Help Browser:
- Go to the page where you want to view an article.
- Click the **Open Inline Help **icon.
[See image.](#open-inline-help-icon)
The Help Browser displays the article relevant to the page you are viewing. You can then read about the page.

Changing the Font Size in the Help Browser
There are a variety of font sizes available in the Help Browser. You can click the **Font** icon to increase or decrease the text size.
![The Font icons for the Zscaler Client Connector Portal Documentation Browser](/downloads/z-app/getting-started/admin-portal/using-zscaler-app-portal-documentation-browser/Help-Browser-Zscaler-App-Portal-font-size.png)
Zscaler saves your customization until you log out or the session expires.
Navigating within the Help Browser
You can do the following to navigate within the Help Browser:
- Go to the home page (i.e., initial article relevant to the page you are viewing).
- Navigate back and forth across articles you've viewed within the Help Browser.
- Search for an article.
- Copy the URL of an article to your clipboard.
- Close the Help Browser.
- Open the article in new tab.
![Navigating the Zscaler Client Connector Portal Documentation Browser](/downloads/z-app/getting-started/admin-portal/using-zscaler-app-portal-documentation-browser/Help-Browser-Zscaler-App-Portal-navigation.png?1601982472)
Closing the Help Browser clears your navigation history. When you reopen the Help Browser, it displays the article relevant to the page you are viewing, and you are unable to navigate back and forth to articles you've viewed in the previous browser session.
