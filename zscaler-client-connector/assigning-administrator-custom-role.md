# Edit a Role
Source: https://help.zscaler.com/zscaler-client-connector/assigning-administrator-custom-role
PDF: https://help.zscaler.com/pdf/com/en/1414996.pdf

You can edit a custom role assigned to an admin synced from Zscaler Internet Access (ZIA), Zscaler Private Access (ZPA), and Zscaler Digital Experience (ZDX).
- In the Administrators tab, click the **Edit **icon.
![Assign or edit admin role](/downloads/z-app/policy-administration-settings/assigning-administrator-custom-role/client-connector-edit_admin-sm_0.png)
- In the **Edit Admin User **window, select a role from the **Role **drop-down menu.
![assign role to admin](/downloads/z-app/policy-administration-settings/assigning-administrator-custom-role/client-connector-role-mgmt-assign-role_sm.png)
- Click **Save**.
