# About Zscaler Client Connector Support
Source: https://help.zscaler.com/zscaler-client-connector/about-zscaler-client-connector-support
PDF: https://help.zscaler.com/pdf/com/en/1328926.pdf

Zscaler Client Connector Support allows users to configure settings for customizing Zscaler Client Connector.
From the Zscaler Client Connector Portal, you can configure the following settings that allow you to customize Zscaler Client Connector:
- Allow users to configure app supportability features such as logging controls and support access.
- Allow users to collect traceroute information between devices and the Zscaler cloud.
- Allow users to configure fail-open, user privacy, endpoint integration, automated device removal, and advanced configuration settings.
