# Allowing Non-Administrator Users Access to Zscaler Client Connector Log Files
Source: https://help.zscaler.com/zscaler-client-connector/allowing-non-administrators-access-client-connector-log-files
PDF: https://help.zscaler.com/pdf/com/en/1452936.pdf

This feature enables Zscaler Client Connector Portal administrators to grant Zscaler Client Connector log file access to users who donâ€™t have administrator privileges. This allows non-administrator users to send specific log files to admins instead of having to send the entire log folder in a ZIP file, making the troubleshooting process more efficient.
To allow non-admin users access to the Zscaler program data folder to access log files:
- In the Zscaler Client Connector Portal, go to **Administration**.
- In the left-side navigation, select** Client Connector Support**.
- Click the **User Privacy** tab.
- Select** Allow access to Client Connector logs folder for non-administrators**.
![Allow non admin access to the Zscaler Program Data folder log files](/downloads/z-app/configuring-policy-and-administration-settings/zscaler-app-support-settings/user-privacy/allowing-non-administrators-access-client-connector-log-files/Allow%20access%20to%20Client%20Connector%20Log%20folder_1500.png)
- Click **Save**.
