# About Zscaler Client Connector Notifications
Source: https://help.zscaler.com/zscaler-client-connector/about-zscaler-client-connector-notifications
PDF: https://help.zscaler.com/pdf/com/en/1328916.pdf

From the Zscaler Client Connector Portal, you can configure notification settings for Zscaler Client Connector.
There are two types of notifications: the default Windows-based notification system and Zscalerâ€™s Notification Framework. While both notification systems provide the same informational messages, users cannot disable notifications from the Zscaler Notification Framework in Windows settings. To learn more, see [Using the Zscaler Notification Framework](/zscaler-client-connector/using-zscaler-notification-framework).
Zscaler Client Connector Notification Settings provide the following benefits and allow you to:
- Control the notifications that end users receive by default in the system tray and the Zscaler Notification Framework.
- Customize the Acceptable Use Policy (AUP) users must accept before they can connect to the internet or access your organization's internal resources from devices protected by Zscaler Client Connector.
- Configure a custom banner that displays IT support options for your organization to help users resolve issues.
- Create configuration reminder messages for SecureAgent users and set how often the reminder displays.
- Configure notification templates that you can assign to specific groups of users.
About the Zscaler Client Connector Notifications Page
On the Zscaler Client Connector Notifications page (Administration > Client Connector Notifications), you can do the following:
- [Configure end user notification settings](/zscaler-client-connector/configuring-end-user-notifications-zscaler-client-connector).
- [Create an Acceptable Use Policy](/zscaler-client-connector/configuring-acceptable-use-policy-aup-zscaler-app).
- [Configure a custom banner](/zscaler-client-connector/configuring-custom-banner).
- Configure a reminder notification to SecureAgent users.
- [Configure a notification template](/zscaler-client-connector/configuring-notification-templates-zscaler-client-connector) (Zscaler Client Connector for Windows only).
