# Displaying Minimum Version Requirements in the Zscaler Client Connector Portal
Source: https://help.zscaler.com/zscaler-client-connector/displaying-minimum-version-requirements-zscaler-app-portal
PDF: https://help.zscaler.com/pdf/com/en/1353956.pdf

You can configure the Zscaler Client Connector  Portal to display the minimum Zscaler Client Connector version number for features that require a specific Zscaler Client Connector version.
To configure the Zscaler Client Connector Portal to display minimum version numbers:
- Go to any page in the Zscaler Client Connector Portal.
- Click the **Versions** switch in the left-hand corner.
[See image.](#zscaler-app-portal-versions-button)
-  If enabled, the Zscaler Client Connector Portal displays the minimum version numbers next to all features that require specific versions. For example, to configure the Tunnel Internal Zscaler Client Connector Traffic setting in [Zscaler Client Connector profiles](/zscaler-client-connector/configuring-zscaler-app-profiles), you must have Zscaler Client Connector version 2.1.2 or later.
[See image.](#versions-button-enabled)
- If disabled, the Zscaler Client Connector Portal does not display the version numbers.
[See image.](#versions-number-disabled)
