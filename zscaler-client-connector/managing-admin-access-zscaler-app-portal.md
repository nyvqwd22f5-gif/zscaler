# Managing Admin Access to the Zscaler Client Connector Portal
Source: https://help.zscaler.com/zscaler-client-connector/managing-admin-access-zscaler-app-portal
PDF: https://help.zscaler.com/pdf/com/en/1340896.pdf

In the ZIA Admin Portal, you can manage admin access to the Zscaler Client Connector  Portal by defining the admin role permission, and then assigning the role to admins.
Admin roles dictate the level of access admins have to the ZIA Admin Portal. To learn more about admin roles, see [About Role Management](/zia/about-role-management).
Configuring the Admin Role
For more information on configuring admin roles, see [Adding Admin Roles](/zia/adding-admin-roles).
To configure admin access to the Zscaler Client Connector Portal:
- In the ZIA Admin Portal, go to **Administration** > **Role Management**.
- Click **Add Administration Role** or edit an existing admin role.
- Under **Permissions**, choose the level of access for **Policy Access**:
- **Full**: Allows admins full access to the Zscaler Client Connector Portal.
- **View Only**: Allows admins to view, but not edit, items in the Zscaler Client Connector Portal.
- **None**: Does not allow admins access to the Zscaler Client Connector Portal. The ZIA Admin Portal’s link to the Zscaler Client Connector Portal disappears. Additionally, the Zscaler Client Connector Portal does not load if the admin accesses the portal unconventionally, e.g., a bookmark.
This option also determines the level of access to the **Policy** and **Administration** pages of the ZIA Admin Portal.
![The Policy Access permissions in the ZIA Admin Portal](/downloads/z-app/getting-started/admin-portal/managing-admin-access-zscaler-app-portal/add-admin-role-zia.png)
- Click **Save** and [activate the change](/zia/saving-and-activating-changes-admin-portal).
Assigning the Admin Role
For more information about configuring admins, see [Adding Admins](/zia/adding-admins).
To assign the role to the admin:
- In the ZIA Admin Portal, go to** Administration **> **Administration Management**.
- Click **Add Administrator** or edit an existing admin.
- From the **Role** menu, choose the role with the correct level of access to the Zscaler Client Connector Portal. You can also search for roles or click the **Add** icon to [add a new role](/zia/adding-admin-roles).
![Assigning the admin role with access to the Zscaler Client Connector Portal](/downloads/z-app/getting-started/admin-portal/managing-admin-access-zscaler-app-portal/select-admin-role-zia.png)
- Click **Save** and [activate the change](/zia/saving-and-activating-changes-admin-portal).
