# Configuring Dedicated Proxy Ports
Source: https://help.zscaler.com/zscaler-client-connector/configuring-dedicated-proxy-ports
PDF: https://help.zscaler.com/pdf/com/en/1374356.pdf

You can use and manage dedicated proxy ports in the Zscaler Client Connector Portal. To learn more about dedicated proxy ports for the Zscaler service, see [Configuring Dedicated Proxy Ports](/zia/configuring-dedicated-proxy-ports).
- Go to **Administration **> **Dedicated Proxy Port**.
- Enable **Dedicated Proxy Port**.
-
Select an available port from the **Select Dedicated Proxy Port** drop-down menu. Zscaler Client Connector attempts connections with this port before using the following ports in this order: 443 or 80.
![Enable the dedicated proxy port setting and select a port](/downloads/zclient-connector-dedicated-proxy-port.png)
- Click **Save**.
