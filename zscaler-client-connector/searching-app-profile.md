# Searching for a Zscaler Client Connector Profile
Source: https://help.zscaler.com/zscaler-client-connector/searching-app-profile
PDF: https://help.zscaler.com/pdf/com/en/1443771.pdf

You can quickly find a specific Zscaler Client Connector [Profile](/zscaler-client-connector/configuring-zscaler-client-connector-profiles) using the Search** **field by selecting different criteria to narrow down your search results.
To find a Zscaler Client Connector Profile using the Search field:
- In the Zscaler Client Connector Portal, go to **App Profiles**.
- In the **Search** field, select a search option from the drop-down menu. You can search using the following criteria:
- Policy Name
- Group Name
- Policy Token
- Enter search words and press `Enter` to display your search results.
To clear the **Search **field, delete the search term you typed and press `Enter`.
