# Upgrading to macOS Catalina
Source: https://help.zscaler.com/zscaler-client-connector/upgrading-macos-catalina
PDF: https://help.zscaler.com/pdf/com/en/1352046.pdf

Zscaler Client Connector 2.1 and Catalina are no longer supported. To learn more, see [Supported Versions](/eos-eol/supported-versions).
If you want to upgrade to macOS Catalina, you must use Zscaler Client Connector 2.1 or later. To learn more, see [(General Availability) Zscaler Client Connector 2.1.0.190 for macOS](/zscaler-client-connector/client-connector-app-release-summary-2020?applicable_category=macOS&applicable_version=2.1.0.190).
