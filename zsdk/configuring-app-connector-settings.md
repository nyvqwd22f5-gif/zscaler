# Configuring App Connector Settings
Source: https://help.zscaler.com/zsdk/configuring-app-connector-settings
PDF: https://help.zscaler.com/pdf/com/en/1508221.pdf

The App Connector Settings allow you to enable Auto Delete for App Connectors with a disconnected status. You can opt to remove disconnected or optionally disabled App Connectors after a set number of days.
To configure your App Connector Settings:
- Go to **Configuration & Control** > **Private Infrastructure** > **App Connector Management** > **App Connectors**.
-
Click **Settings**. If your window is compact and resized, click the **Menu** icon (![Menu icon in the Application Segments page](/downloads/zpa/documentation-knowledgebase/applications/application-segments/setting-application-segment-configuration-warnings/menu%20icon.png)
) to view a drop-down menu of the features.
The **App Connector Settings** window appears.
-
In the **App Connector Settings** window:
- **Auto Delete**: Enable this if you want to allow the deletion of App Connectors with a disconnected status. This field is set to **Disabled **by default.
- **Delete App Connectors after**: Set the number of days for App Connectors with a disconnected status to be deleted.
- (Optional) **Include disabled App Connectors**: Select to automatically delete disabled App Connectors at the set time.
[See image.](#window)
- Click **Save**.
