# Viewing the Resource Details
Source: https://help.zscaler.com/dspm/viewing-resource-details
PDF: https://help.zscaler.com/pdf/com/en/1474781.pdf

You can view additional details of the resource that contains sensitive data or that has vulnerabilities. The granular information about the resource allows you to quickly investigate and take the necessary action.
To view the resource details:
- Go to **Analytics** > **Resource Inventory**.
- Click any **Resource Name** to view the following tabs:
- **Risk Explorer**: The scan results are [displayed on a graph](/dspm/viewing-resource-inventory-graph).
- **Sensitive Data**: View the [sensitive data details](/dspm/viewing-sensitive-data-details).
- **Alerts**: View the[ alert details](/dspm/viewing-alert-details).
- **Access**: View the [access level details](/dspm/viewing-resource-access-levels).
- **Vulnerabilities**: View the [vulnerabilities](/dspm/viewing-vulnerability-details) detected in the resource. Vulnerabilities are displayed only for EC2 instances.
- **Malware**: View the [malware](/dspm/viewing-malware-details) detected in the resource.
