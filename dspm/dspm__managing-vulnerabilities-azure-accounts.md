# Managing Vulnerabilities in Azure Accounts
Source: https://help.zscaler.com/dspm/managing-vulnerabilities-azure-accounts
PDF: https://help.zscaler.com/pdf/com/en/1486701.pdf

DSPM scans your Microsoft Azure cloud accounts to detect vulnerabilities and displays the scan results on the [Data Inventory page](/dspm/about-data-inventory).
- DSPM supports the scanning of workloads that are created from images present in Azure Compute Gallery and Azure Marketplace.
- DSPM does not support the vulnerability scanning of Azure workloads created from unmanaged OS disks (.vhd).
