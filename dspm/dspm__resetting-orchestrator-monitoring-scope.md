# Resetting Orchestrator & Monitoring Scope
Source: https://help.zscaler.com/dspm/resetting-orchestrator-monitoring-scope
PDF: https://help.zscaler.com/pdf/com/en/1504071.pdf

If you want to reset the orchestrator or monitoring scope subscriptions after deploying the onboarding template, run the `terraform destroy` command to purge the template and delete any recource that was created.
To reset the configured subscriptions:
-
Select **Orchestrator & Monitoring Scope** and click **Reset**.
[See image.](#resetsubscriptions)
-
In the **Reset Configured Subscription** window, click **Confirm**.
[See image.](#resetwindow.png)
- [Configure new subscriptions.](/dspm/onboarding-microsoft-azure-tenant#orch-moni-scope)
- [Deploy the onboarding template.](/dspm/onboarding-microsoft-azure-tenant#grantpermission)
