# Downloading Reports
Source: https://help.zscaler.com/zpc/downloading-reports
PDF: https://help.zscaler.com/pdf/com/en/1398501.pdf

You can download reports for evaluating events that occurred in your cloud workloads and for assessing the security posture of your cloud environment. You can review the reports and assess ZPC's performance in proactively monitoring your workloads for security vulnerabilities and policy violations and how you can take the necessary remediation steps.
You can download Excel reports for:
- Cloud Alerts
- IaC Alerts
- Cloud Assets
- Vulnerabilities
You can schedule and generate Compliance reports. To learn more, see [Scheduling Compliance Reports](/zpc/scheduling-compliance-reports).
Downloading Alerts Reports
To download a cloud alerts or an IaC alerts report:
- On the left navigation menu, select **Alerts**.
- On the **Alerts** page, select the **Cloud Alerts** or **IaC Alerts** tab.
- Click the **Export** icon.
[See image.](#cloud)
A message appears indicating that the request is submitted and the report is being exported.
[See image.](#export)
- Next, click the **Profile** (![Profile icon](/downloads/zpc/scheduled-reports/downloading-reports/zpc-profile-icon.png)
) icon on the left navigation menu.
The **My** **Profile** page appears.
- On the **My Profile** page, click the **My Downloads** tab.
You can see the list of reports that you've requested.
[See image.](#download)
- Click the **Download** icon to download the report.
If the report is not yet generated, then the status shows as **Pending**. Refresh the page or revisit the page after some time to see the **Download** icon.
Downloading Cloud Assets Reports
To download a Cloud Asset report:
- Go to **Cloud Posture** > **Cloud Assets**.
- On the **Cloud Assets** page, click the **Export** icon.
[See image.](#assets)
A message appears indicating that the request is submitted and the report is being exported.
[See image.](#exportassetreport)
- Next, click the **Profile** (![Profile icon](/downloads/zpc/scheduled-reports/downloading-reports/zpc-profile-icon.png)
) icon on the left navigation menu.
- On the **Profiles** page, click the **My Downloads** tab.
You can see the list of reports that you've requested.
[See image.](#downloadca)
- Click the **Download** icon to download the report.
If the report is not yet generated, then the status shows as **Pending**. Refresh the page or revisit the page after some time to see the **Download** icon.

Downloading Vulnerability Reports
To download a vulnerability report:
- Go to **Cloud Posture** > **Vulnerability Management**.
- On the **Vulnerability Management** page, select the **Vulnerabilities** or **Container Registry Images** or **Cloud Workloads** tab.
- Click the **Export** icon.
[See image.](#export-icon)
A message appears indicating that the request is submitted and the report is being exported.
[See image.](#export-vul)
- Next, click the **Profile** (![Profile icon](/downloads/zpc/scheduled-reports/downloading-reports/zpc-profile-icon.png)
) icon on the left navigation menu.
- On the **My Profile** page, click the **My Downloads** tab.
You can see the list of reports that you've requested.
[See image.](#vul-report)
- Click the **Download** icon to download the report.
If the report is not yet generated, then the status shows as **Pending**. Refresh the page or revisit the page after some time to see the **Download** icon.
