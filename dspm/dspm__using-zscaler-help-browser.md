# Using the Zscaler Help Browser
Source: https://help.zscaler.com/dspm/using-zscaler-help-browser
PDF: https://help.zscaler.com/pdf/com/en/1474751.pdf

The Zscaler Help Browser is available for all pages within the DSPM Admin Portal. Through the Help Browser, you can view the DSPM help articles in context with the page you're on, without leaving the DSPM Admin Portal.
To open the Help Browser:
- Go to the page where you want to view an article.
-
Click the **Open Help Browser** icon in the lower-right corner.
[See image.](#ds-helpicon)
The Help Browser displays the article relevant to the page you are viewing.
[See image.](#ds-help-article)
Navigating within the Help Browser
You can do the following to navigate within the Help Browser:
- Close the Help Browser.
- Go to the home page (i.e., initial article relevant to the page you are viewing) or navigate back and forth across articles you've viewed within the Help Browser, search for an article, or copy the article's URL to your clipboard.
- Click the **Font** icons to increase or decrease the text size.
- Open the article in a new tab.
![Use the options to navigate within the help browser](/downloads/dspm/getting-started/admin-portal/ds-helpbrowseroptions.png)
Closing the Help Browser clears your navigation history. When you reopen the browser, it displays the article relevant to the page you are viewing, and you will not be able to navigate back and forth to articles you've viewed in the previous Help Browser session.
