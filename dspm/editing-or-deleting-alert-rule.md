# Editing or Deleting an Alert Rule
Source: https://help.zscaler.com/dspm/editing-or-deleting-alert-rule
PDF: https://help.zscaler.com/pdf/com/en/1477806.pdf

You can edit or delete the alert rule that you've [added](/dspm/adding-alert-notification).
Editing an Alert Rule
To edit an alert rule:
- Go to **Alerts** > **Alert Settings** > **Alert Notification**.
- In the table, locate the alert rule you want to edit and click the **Actions** icon (![dspm-action-icon.png](/downloads/dspm/alerts/alert-ignore-filters/editing-or-deleting-ignore-filters/dspm-action-icon.png)
).
-
Select **Edit**.
[See image.](#edit-icon)
- On the **General Information** page, change the **Alert Rule** **Name **or **Alert Rule Description **as required, then click **Next**.
- On the **Notification Scope** page, update the filters as required, then click **Next**.
- On the **Notifications** page, update the notification channels as required, then click **Next**.
-
Review the changes and click **Finish**.
[See image.](#review-summary)
A message appears indicating that the alert rule is updated successfully.
Deleting an Alert Rule
To delete an alert rule:
- Go to **Alerts** > **Alert Settings** > **Alert Notification**.
- In the table, locate the alert rule you want to delete and click the **Actions** icon (![dspm-action-icon.png](/downloads/dspm/alerts/alert-ignore-filters/editing-or-deleting-ignore-filters/dspm-action-icon.png)
).
-
Select **Delete**.
[See image.](#delete-icon)
-
A confirmation message appears. Read the message and click **Delete**.
[See image.](#delete-window)
