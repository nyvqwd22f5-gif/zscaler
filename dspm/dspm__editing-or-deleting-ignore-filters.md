# Editing or Deleting Ignore Alert Filters
Source: https://help.zscaler.com/dspm/editing-or-deleting-ignore-filters
PDF: https://help.zscaler.com/pdf/com/en/1477826.pdf

You can edit or delete the [ignore alert filters](/dspm/adding-ignore-filters) that you've added.
Editing or deleting the ignore alert filters impact only the newly triggered alerts.
Editing an Ignore Alert Filter
To edit an ignore alert filter:
- Go to **Alerts** > **Alert Settings** > **Ignore Alert Filter**.
- In the table, locate the ignore alert filter you want to edit and click the **Actions** icon (![dspm-action-icon.png](/downloads/dspm/alerts/alert-ignore-filters/editing-or-deleting-ignore-filters/dspm-action-icon.png)
).
-
Select **Edit**.
[See image.](#editicon)
- On the **General Information** page, change the **Ignore Filter Name** or **Filter Description **as required, then click **Next**.
- On the **Filter Scope** page, update the filters as required, then click **Next**.
-
Review the changes and click **Finish**.
[See image.](#summary)
A message appears indicating that the ignore alert filter is updated successfully.
Deleting an Ignore Alert Filter
To delete an ignore alert filter:
- Go to **Alerts** > **Alert Settings** > **Ignore Alert Filter**.
- In the table, locate the ignore alert filter you want to delete and click the **Actions** icon (![dspm-action-icon.png](/downloads/dspm/alerts/alert-ignore-filters/editing-or-deleting-ignore-filters/dspm-action-icon.png)
).
-
Select **Delete**.
[See image.](#deleteicon)
-
A confirmation message appears. Read the message and click **Delete**.
[See image.](#deletewindow)
The ignore alert filter is deleted from the DSPM Admin Portal and is no longer available in the ignore alerts table.
