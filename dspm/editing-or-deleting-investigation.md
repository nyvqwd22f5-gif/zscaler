# Editing or Deleting an Investigation
Source: https://help.zscaler.com/dspm/editing-or-deleting-investigation
PDF: https://help.zscaler.com/pdf/com/en/1480631.pdf

You can edit an existing or a saved query. But you can delete only saved investigation queries.
Editing an Investigation
To edit an investigation query:
- Go to **Analytics** > **Investigation**.
- You can edit the investigation by using any of these options:
- On the **History** tab:
-
Click the **Query** to view the drawer, then click the **Edit** icon at the top right corner.
[See image.](#ds-editicon)
-
Click the **Actions** icon (![ds-actionsicon.png](/downloads/dspm/analytics/investigation/editing-or-deleting-investigation/ds-actionsicon.png)
) and select **Open in Editor**.
[See image.](#ds-editaction)
- On the **Saved** tab:
-
Click the **Actions **icon (![ds-actionsicon.png](/downloads/dspm/analytics/investigation/editing-or-deleting-investigation/ds-actionsicon.png)
) and select **Edit Query**.
[See image.](#ds-editsavedqueryaction)
- Edit the query by adding or removing predicates, as required.
-
Click **Run Query** to view the results.
[See image.](#ds-results)
- Click **Save Investigation** to save the query.
-
Enter a name for the query.
[See image.](#ds-savequery)
- Click **Save**.
Deleting an Investigation
You can delete only those investigations that you've saved.
To delete an investigation:
- Go to **Analytics** > **Investigation**.
-
On the **Investigation** page, select the** Saved** tab.
[See image.](#ds-savetab)
- You can do the following:
-
Click the **Actions **icon (![ds-actionsicon.png](/downloads/dspm/analytics/investigation/editing-or-deleting-investigation/ds-actionsicon.png)
) and select **Delete**.
[See image.](#ds-deleteaction)
-
Click the **Name** or **Query** to view the drawer, then click the **Delete** icon.
[See image.](#ds-deleteicon)
-
Read the confirmation message, then click **Delete**.
[See image.](#ds-deletemsg)
The query is deleted.
