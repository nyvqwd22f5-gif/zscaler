# Using Tables
Source: https://help.zscaler.com/dspm/using-tables
PDF: https://help.zscaler.com/pdf/com/en/1478136.pdf

Data is organized and displayed in tables on some of the pages (e.g. Alerts, Policies, Data Sensitivity Settings, etc.) in the DSPM Admin Portal. You can modify the default settings of the table. Every time you make a change, the settings are stored in your browser's local cache, so you are able to see the table with the modified settings the next time you log on.
If you clear your cache or use a different browser, your settings are lost.
You can do the following:
- [Resize Columns](#resizecolumn)
- [Sort the Column Data](#sort)
- [Reorder Columns](#order-columns)
- [Show or Hide Columns](#hidecolumn)
- [View Specific Number of Rows](#ds-viewrows)
- [Search for Specific Information](#search)
[]When the data in a column takes up more space than the viewing area allows, it gets clipped.
To increase the column width:
-
Hover over the right border of the column.
A **Resize** icon appears.
-
Drag and drop the icon to the desired column width.
[See image.](#ds-columnwidth)
[]Some columns have a **Sort** icon. Click the icon to sort the data in ascending or descending order.
[See image.](#ds-sort)
- []Click the **Column Menu** icon at the top right corner of the table.
-
Drag and drop the **Column** icon or the Column name to the desired position.
The table refreshes and displays the columns in the new order.
[See image.](#ds-columnorders)
[]When you don't want to view all the columns at one time, you can hide a few columns and view only specific ones.
- Click the **Column Menu** icon at the top right corner of the table.
- Select the required checkboxes for the columns you want to see in the table. The table refreshes and displays only the selected columns.
-
Click **Reset** to revert to the default column settings.
[See image.](#ds-showhidecolumns)
[]You can choose to view a specific number of rows in the table.
- Click the **Column Menu** icon at the top right corner of the table.
- Select **Rows**.
- Select the number of rows that must be displayed in the table.
-
Click **Reset** to revert to the default row settings.
[See image.](#ds-selectrows)
[]You can use the search bar to filter and view specific data.
The search results display the search data even if it is in a hidden column.
[See image.](#ds-searchbar)
