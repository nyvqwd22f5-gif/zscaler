# Editing or Deleting Scan Scopes
Source: https://help.zscaler.com/dspm/editing-or-deleting-scan-scopes
PDF: https://help.zscaler.com/pdf/com/en/1510006.pdf

You can modify or delete the existing scan scope as required. To learn more about scan scopes, see [Configuring Scan Scope](/dspm/configuring-scan-scope).
[]Editing the Scan Scope
To edit an existing scan scope:
- Go to **Administration** > **Scan Settings**.
- Select the **Scan Scope** tab.
- On the [Scan Scope](#about_the_scan_scope) page or [Scan Scope Details](#Viewing_the_Scan_Scope) page, click **Actions** of the desired scope.
-
Choose **Edit** from the drop-down menu.
[See image.](#img_scope_details_edit)
-
On the **General Information** page, modify the details for the following fields if needed:
- **Engine Scope Name**: A concise and descriptive name for your scan scope.
- **Engine Scope Description**: Clear and detailed explanation of what your scan scope covers, allowing for a clear understanding of the scope of work.
[See image.](#img_scope_general)
- Click **Next**.
-
On the **Select Criteria** page, modify the following options if needed:
- **DLP Engine**: Choose DLP engines for the scan scope. Choose all for a comprehensive scan or specific DLP engines to focus on specific DLP engines.
- **Specific DLP Engines**: Choose the DLP engines you want to include in your scope. This option is enabled only if you select specific DLP engines to scan.
- **File Type**: Choose the file types you want to include in your scan scope.
-
**Machine Learning Document Scan**: Enable if you want to use machine learning-powered document scanning.
File type filters and machine learning document scans are optimized for cloud storage and virtual machine resources.
[See image.](#img_scope_criteria)
- Click **Next**.
-
On the Summary page, review the modified details. If any changes are required, click the **Edit** icon to make the necessary modifications.
[See image.](#img_scope_summary)
- Click **Finish** to complete editing the scan scope.

[]Deleting the Scan Scope
To delete an existing scan scope:
- Go to **Administration** > **Scan Settings**.
- Select the **Scan Scope** tab.
- On the [Scan Scope](#about_the_scan_scope) page or [Scan Scope Details](#Viewing_the_Scan_Scope) page, click **Actions** of the desired scope.
- Choose **Delete **from the drop-down menu.
[See image.](#img_scope_details_delete)
- In the **Delete Scan Scope **confirmation window, click **Confirm**.
[See image.](#img_scope_delete_confirmation)
