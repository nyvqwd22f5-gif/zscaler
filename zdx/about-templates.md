# About Templates
Source: https://help.zscaler.com/zdx/about-templates
PDF: https://help.zscaler.com/pdf/com/en/1488201.pdf

[Watch a video about Templates in ZDX.](https://fast.wistia.net/embed/iframe/t4mafl179c)
You can create templates to save frequently used alert rule criteria. After saving a template, you can then use it to prefill an alert rule when you configure a Zscaler Hosted probe. To learn more, see [Configuring Zscaler Hosted Probes](/ZDX/configuring-zscaler-hosted-probes).
Templates provide the following benefits and allow you to:
- Create prefilled criteria and fields when configuring an alert rule for Zscaler Hosted probes.
- Save time and maintain consistency when configuring alert rules.
- Flexibility to configure an alert rule.
About the Templates Page
On the Templates page (Alerts > Templates), you can do the following:
- [Add a new template](/zdx/managing-templates).
- View a list of created templates.
- **Name**: The template's name during configuration.
- **Severity**: The severity level of the event. Red indicates high severity, orange is medium severity, and green indicates low severity.
- **Type**: The template type is **Predefined** or **Custom**.
- **Probe Type**: The probe type is **Cloud Path** or **Web**.
- **Criteria**: The alert rule criteria configured for the template. To learn more, see [Configuring an Alert Rule](/zdx/configuring-alert-rule).
-
**Actions**: Edit, copy, or delete the template. To learn more, see [Managing Templates](/zdx/managing-templates).
You cannot edit or delete a predefined template. You can copy the predefined template.
- Use the filters to sort and view templates.
- Configure the columns to display.
- Access the navigation menu to go to the following pages:
- [Alerts](/zdx/about-alerts)
- [Rules](/zdx/about-rules)
