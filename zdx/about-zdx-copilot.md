# About ZDX Copilot
Source: https://help.zscaler.com/zdx/about-zdx-copilot
PDF: https://help.zscaler.com/pdf/com/en/1478896.pdf

ZDX Copilot is powered by AI/ML to become an interactive, conversational assistant to complement ZDX. Admins can use ZDX Copilot to effectively combine a multitude of ZDX features to optimize and compile users' details (e.g., data retrieval, summarization, or recognizing rising trends for an incident).
ZDX Copilot provides the following benefits and enables you to:
- Deliver relevant insights and graphs or chart visualizations.
- Compress complex datasets into an intelligent summarization.
- Discover the root causes of user experience issues and offer recommendations for troubleshooting.
There are many different types of use cases for ZDX Copilot as it can interact with many ZDX features. To learn more, see [Understanding ZDX Copilot with Use Cases](/zdx/understanding-zdx-copilot-use-cases).
ZDX Copilot retains prompts and responses for a rolling period of 14 days as part of the session history. To learn more, see [ZDX Customer Logs and Data](/customer-logs-fair-use/zdx-customer-logs-and-data).
ZDX Copilot strives to assist with the most updated information, but can be incomplete, inaccurate, or inappropriate in certain instances due to the subjective nature of generative AI/ML models.
Prerequisites
To access ZDX Copilot, you must have:
- The appropriate ZDX subscription level for ZDX Copilot. To learn more, see [Ranges & Limitations](/zdx/ranges-limitations).
- Full permission level to ZDX Copilot. To learn more, see [Adding ZDX Roles](/zdx/adding-zdx-roles).
About the ZDX Copilot Page
On the ZDX Copilot page, you can do the following:
- Create a new ZDX Copilot session.
- Review chat session history. No data is shown if there were no chat sessions in the past 14 days.
- Select a topic based on your chat session history. If there is no chat session history, then the topics are defaulted to recommended topics.
- Ask a question with ZDX Copilot by entering text.
