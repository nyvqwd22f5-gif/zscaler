# ZCSPM Licensing and Fair Use
Source: https://help.zscaler.com/logs-fair-use/zcspm-licensing-and-fair-use
PDF: https://help.zscaler.com/pdf/com/en/1392901.pdf

Zscaler Cloud Security Posture Management (ZCSPM) is a Software as a Service (SaaS) product that is licensed by the number of servers and workloads needed.
License Types
Zscaler offers the following two types of licenses:
- Server: A subscription license for a unique domain name, an IP address, or a set of IP addresses assigned to an asset.
- Workload: A subscription license for a single cloud workload such as virtual machines (VMs), database services, cloud storage, and other similar Infrastructure as a Service (IaaS) and Platform as a Service (PaaS) resources that your organization utilizes in your public cloud environments.
Ensure that you order the correct number of servers and workloads. If you require additional servers or workloads, contact Zscaler Sales or your Channel Partner. Zscaler periodically reviews the number of servers and workloads to ensure license compliance.
The usage of ZCSPM is expected to be in accordance with the [Acceptable Use Policy](/legal/acceptable-use-policy).
