# About Root Certificates for Isolation in ZPA
Source: https://help.zscaler.com/isolation/about-root-certificates-isolation-zpa
PDF: https://help.zscaler.com/pdf/com/en/1450586.pdf

[Watch a video about Custom Root Certificates in Isolation.](https://fast.wistia.net/embed/iframe/ty0r2pt9gx)
Root certificates allow admins to have more control over isolation profile authentication. Admins can manage root certificates for their organization and associate them with specific isolation profiles. To learn more, see [Creating Isolation Profiles for ZPA](/isolation/creating-isolation-profiles-zpa).
About the Root Certificates for Isolation Page
On the Root Certificates for Isolation page (Configuration & Control > Certificate Management > Root Certificates for Isolation), you can do the following:
- [Add](/isolation/adding-root-certificates-isolation-zpa) a new root certificate.
- [Edit](/isolation/editing-root-certificate-isolation-zpa) any custom root certificates that are not the default.
- [Delete](/isolation/deleting-root-certificate-isolation-zpa) any custom root certificates that are not the default and are not associated with any isolation profile.
- Expand details for all root certificates in the list.
- Expand details for an individual root certificate in the list.
- Go to the **Enrollment Certificates **page to manage [enrollment certificates](/zpa/about-enrollment-ca-certificates).
- Go to the **Certificates **page to manage certificates.
