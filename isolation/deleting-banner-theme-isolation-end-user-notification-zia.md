# Deleting a Banner Theme for the Isolation End User Notification in ZIA
Source: https://help.zscaler.com/isolation/deleting-banner-theme-isolation-end-user-notification-zia
PDF: https://help.zscaler.com/pdf/com/en/1447381.pdf

Admins manage the [banner themes](/isolation/adding-banner-theme-isolation-end-user-notification-zia) for the isolation session's end user notification. You can delete any banner theme except the Default.
To edit a banner theme:
- Go **Administration** > **Secure Browsing** > **Browser  Isolation**.
- Click ** Isolation Banners**.
- Click the **Edit** button for the banner.
[See image.](#Isolation-Banners_Edit)
The **Edit Isolation Banner **window appears. In the **Edit Isolation Banner **window:
- Click **Delete**.
[See image.](#Edit-iso-banner-window)
- A warning message appears to confirm that you would like to delete the certificate. Click **Delete**.
[See image.](#Delete-iso-banner-window)
