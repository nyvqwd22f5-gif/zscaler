# Using Search in Isolation
Source: https://help.zscaler.com/isolation/using-search-isolation
PDF: https://help.zscaler.com/pdf/com/en/1403351.pdf

The Search function in Isolation replicates a typical find-in-page experience for users. It allows you to search for text within a web page, and scroll through all possible results.
To use Search in Isolation:
- Click **Show isolation bar** to view the isolation bar options.
[See image.](#show-isolation-bar)
- Click the isolation bar menu to display the drop-down menu.
- Click **Find...**
The **Find** bar appears.
You can also use the commands Ctrl+F on Windows devices or CMD+F on Mac devices to initiate the Search function. Depending on the website that is isolated, the page might display a search function in addition to the isolation browser search function.
[See image.](#Click-Find)
- In the **Find **bar:
- Enter the text in the search field. The results appear on the page as you type. The limit of text searches that can be found on a web page is capped at 1,017.
- To view the results, click the arrow icons or the arrow keys on your device.
- To exit the Search function, click **Close find bar.**
[See image.](#Find-bar)
