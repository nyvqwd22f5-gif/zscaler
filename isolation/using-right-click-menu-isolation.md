# Using the Right-Click Menu in Isolation
Source: https://help.zscaler.com/isolation/using-right-click-menu-isolation
PDF: https://help.zscaler.com/pdf/com/en/1392986.pdf

The right-click menu in Isolation replicates the typical menu of functions that users normally see in a standard browser window. Depending on where a user is on a web page and what capabilities have been enabled for their [isolation profile](/isolation/profiles), different options will display in the right-click menu.
The right-click menu is available in both Native Experience Mode and Browser-in-Browser Experience Mode. The right-click menu is different from [the isolation bar](/zia/using-isolation-bar-native-browser-experience), which displays only in Browser-in-Browser Experience Mode. To learn more, see [User Experience Modes in Isolation](/zia/user-experience-modes-cloud-browser-isolation).
Right-Click Menu Options by Web Page Location
If the web application you are accessing offers a custom right-click menu, then the application-specific right-click menu displays instead of the default Isolation contextual right-click menu.
Default Menu
When right-clicking anywhere on the web page, you can:
- Reload
- Print
- Translate
If printing is disabled for your isolation profile, you cannot print the web page or document that is rendered in isolation. You cannot see the print option from the right-click menu, or the print button in the isolation bar. If you try to select the option of File > Print from the browser menu, or enter the key command `CTRL/CMD+P` shortcut, a message appears noting it is disabled by policy.
[See image.](#Default-Menu)
Anchor Menu
When right-clicking a hyperlink, or any web element wrapped by a hyperlink, you can:
- Open Link in New Tab
- Copy Link Address
[See image.](#Anchor-Menu)
Image Menu
When right-clicking an image, you can:
- Open Image in New Tab
- Save Image As...
- Copy Image Address
[See image.](#Image-Menu)
Combined Anchor and Image Menu
When right-clicking an image wrapped by a hyperlink, you can:
- Open Link in New Tab
- Copy Link Address
- Open Image in New Tab
- Save Image As...
- Copy Image Address
[See image.](#Anchor-and-Image-Menu)
Selection Menu
When right-clicking a selection you have highlighted from the page, you can:
- Copy
- Copy link to highlight
- Print
- Translate selection to
[See image.](#Selection-Menu)
Text Field Menu
When right-clicking a text field element, you can:
- Undo
- Redo
- Cut
- Copy
- Paste
- Paste as plain text
- Select all
[See image.](#Text-Field-Menu)
