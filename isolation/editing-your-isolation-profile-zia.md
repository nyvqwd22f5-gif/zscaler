# Editing Your Isolation Profile for ZIA
Source: https://help.zscaler.com/isolation/editing-your-isolation-profile-zia
PDF: https://help.zscaler.com/pdf/com/en/1447276.pdf

After [creating a ZIA isolation profile](/isolation/creating-zia-isolation-profile-isolation), you can edit the profile at any time.
To edit your ZIA isolation profile:
- From the **Isolation Profiles** page, click the **Edit** icon for the isolation profile.
The **Edit Isolation Profile** window appears.
[See image.](#Iso-Profiles-List_Edit-button)
- In the **Edit Isolation Profile **window, modify the information as necessary. To learn more, see [Creating Isolation Profiles for ZIA](/isolation/creating-zia-isolation-profile-isolation).
You can use this isolation profile to create policies in ZIA integrated with Isolation. To learn more, see [Configuring ZIA for Isolation](/isolation/configuring-zia-isolation).
