# Interactive Reports for Isolation in the ZIA Admin Portal
Source: https://help.zscaler.com/isolation/isolation-reports-zia-admin-portal
PDF: https://help.zscaler.com/pdf/com/en/1477996.pdf

Zscaler Internet Access (ZIA) provides data reports for admins who use Isolation within their organization. Any organization that uses Isolation automatically sees these default reports in the ZIA Admin Portal.
All functions of the Isolation Reports are the same as the functions of the Interactive Reports in ZIA. To learn more, see [About Interactive Reports](/zia/about-interactive-reports).
Isolation Report Types
The Interactive Isolation Reports consist of data for Consumption, Isolation Traffic Summary Overview, Security Overview, and File and Document in Isolation.
-
**Consumption**: This report shows an overview of how your organization is consuming Isolation as service.
[See image.](#Consumption_report)
-
**Isolation Traffic Summary Overview**: This report shows an overview of the web traffic being isolated for your organization.
[See image.](#Traffic_report)
-
**Security Overview**: This report shows an overview of the security events and trends that were detected in the isolated web traffic for your organization.
[See image.](#Security_report)
-
**File and Document in Isolation**: This report shows an overview of the files that were uploaded, downloaded, or viewed in Isolation.
[See image.](#File_report)
