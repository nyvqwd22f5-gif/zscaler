# Editing Your Isolation Profile for ZPA
Source: https://help.zscaler.com/isolation/editing-your-isolation-profile-zpa
PDF: https://help.zscaler.com/pdf/com/en/1450576.pdf

After [creating a ZPA isolation profile](/isolation/creating-isolation-profiles-zpa), you can edit the profile at any time.
To edit your ZPA isolation profile:
- From the **Isolation Profiles** page, click the **Edit** icon for the isolation profile that you want to edit.
[See image.](#Edit-icon)
The **Edit Isolation Profile** window appears.
- In the **Edit Isolation Profile **window, modify the information as necessary. To learn more, see [Creating Isolation Profiles for ZPA](/isolation/creating-isolation-profiles-zpa).
You can use this isolation profile to create policies in ZPA to isolate specific web applications. To learn more, see [Configuring Isolation Policies](/zpa/configuring-isolation-policies).
