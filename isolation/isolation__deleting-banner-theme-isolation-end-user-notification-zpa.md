# Deleting a Banner Theme for the Isolation End User Notification in ZPA
Source: https://help.zscaler.com/isolation/deleting-banner-theme-isolation-end-user-notification-zpa
PDF: https://help.zscaler.com/pdf/com/en/1450616.pdf

Admins manage the [banner themes](/isolation/adding-banner-theme-isolation-end-user-notification-zia) for the isolation session's end user notification. You can delete any banner theme except the Default.
To delete a banner theme:
- Go to **Configuration & Control** > **Browser Isolation** > **Isolation Banners**.
- Click the **Delete** icon for the banner you want to delete.
[See image.](#Delete-icon)
- A warning message appears to confirm deletion of the certificate. Click **Delete**.
[See image.](#Delete-Isolation-Banner-window)
The Isolation Banners page refreshes and removes the deleted banner theme from the list.
