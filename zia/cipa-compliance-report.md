# CIPA Compliance Report
Source: https://help.zscaler.com/zia/cipa-compliance-report
PDF: https://help.zscaler.com/pdf/com/en/1401296.pdf

The CIPA Compliance Report is an interactive report that provides information on the top URL categories that are blocked from the Legal Liability class, and the top users and domains blocked from accessing obscene or harmful material. To learn more, see [About CIPA Compliance](/zia/about-cipa-compliance).
To display the CIPA Compliance Report within Interactive Reports, you must have CIPA Compliance enabled within Advanced Policy Settings. To learn more, see [Configuring Advanced URL Policy Settings](/zia/configuring-advanced-url-policy-settings).
This report cannot be copied. You'll only be able to see data for this report up to the last 7 days, so when scheduling the report, the **Monthly** option is grayed out. To learn more, see [Scheduling Reports](/zia/scheduling-reports).
The report contains the following interactive widgets:
- Top URL Categories Blocked from Legal Liability
- Top Users Blocked from Adult Materials
- Top Users Blocked from Drugs
- Top Users Blocked from Gambling
- Top Users Blocked from Violence
- Top Users Blocked from Weapons & Bombs
- Top Users Blocked from Tasteless
- Top Users Blocked from Military, Hate & Extremism
- Top Users Blocked from Illegal or Questionable
- Top Users Blocked from Tobacco/Alcohol
- Top Users Blocked from Newly Registered and Observed Domains
- Top Domains Blocked
