# Exporting Your Google Chrome Extensions List
Source: https://help.zscaler.com/zia/exporting-your-google-chrome-extensions-list
PDF: https://help.zscaler.com/pdf/com/en/1461441.pdf

You can export a list of all Chrome extensions installed on Google Chrome in your organization and [upload](/zia/uploading-apps-bulk) it to the Inventory** **in Zscaler 3rd-Party App Governance.
Exporting Extensions as an Admin
This method is supported if the Chrome browsers in your organization are cloud-managed. To export extensions as an admin:
- [Enroll your userâ€™s browsers to be cloud-managed using an MDM software](https://support.google.com/chrome/a/answer/9301891).
- [Turn on Chrome browser reporting](https://support.google.com/chrome/a/answer/9301421).
- [Generate a report of the installed extensions](https://support.google.com/chrome/a/answer/9902456).
- Click the** Export** button to export the report to a CSV file.
Exporting Extensions as a User
To export extensions as a user:
- Open Google Chrome, and enter `chrome://system/` into the address bar.
- Click*** *****Expand **next to** **`extensions`.

- Copy the list of extensions.
