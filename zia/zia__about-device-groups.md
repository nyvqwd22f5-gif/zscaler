# About Device Groups
Source: https://help.zscaler.com/zia/about-device-groups
PDF: https://help.zscaler.com/pdf/com/en/1402251.pdf

The Device Groups page displays a list of predefined groups. These groups are defined based on the OS types of the devices in your organization. The devices that have the Zscaler Client Connector deployed are categorized under respective groups based on their OS type.
**About the Device Groups Page**
On the Device Groups page (Administration > Device Management > Device Groups), you can:
- View a list of all configured device groups. For each device group, you can view the following information:
- **Group Name**: The name of the device group.
- **Device OS Type**: The OS type of the device.
- **No. of Devices**: The number of devices belonging to the device group.
- **Description**: The description of the device.
- View more information about the device group.
- [Modify the table and its columns](/zia/how-do-i-use-tables-admin-portal).
- Search for a device group.
- Go to the [Devices](/zia/about-devices) page.
