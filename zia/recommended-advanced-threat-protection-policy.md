# Recommended Advanced Threat Protection Policy
Source: https://help.zscaler.com/zia/recommended-advanced-threat-protection-policy
PDF: https://help.zscaler.com/pdf/com/en/1400136.pdf

Zscaler recommends the following configuration for the [Advanced Threat Protection policy](/zia/about-advanced-threat-protection):
- **Suspicious Content Protection (Page Risk)**: 35
- **Botnet Protection**: Block all
- **Malicious Active Content Protection**: Block all
- **Fraud Protection**: Block all
- **Unauthorized Communication Protection**: Block all
- **Cross-Site Scripting (XSS) Protection**: Block all
- **P2P File Sharing Protection**:
- **BitTorrent**: Block
- **P2P Anonymizer Protection**:
- **Tor**: Block
- **P2P VOIP Protection**:
- **Google Hangouts**: Block
