# Adding an IP Pool
Source: https://help.zscaler.com/zia/adding-ip-pool
PDF: https://help.zscaler.com/pdf/com/en/1532865.pdf

You can configure custom IP pools for [Zscaler Extranet Application Support](/zia/understanding-extranet-application-support). Only extranet-type IP pools can be created.
To access Extranet Application Support, contact your Zscaler Account team.
To add an IP pool:
-
On the **IP Pool** page (Administration > IP & FQDN Groups > IP Pool), click **Add IP Pool**.
The **Add IP Pool **window appears.
[See image.](#add-ip-pool)
- Enter a **Name** for the IP pool.
- In the **IP Addresses **field, enter IP addresses as a subnet, or range of addresses, making each entry on a new line, and click **Add Items**.
- (Optional) In the **Description **field, enter a description** **for the IP pool.
- Click **Save **and [activate the change](/zia/saving-and-activating-changes-admin-portal).
