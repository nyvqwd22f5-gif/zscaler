# Downloading User Information to a CSV File
Source: https://help.zscaler.com/zia/downloading-user-information-csv-file
PDF: https://help.zscaler.com/pdf/com/en/1401096.pdf

To download a CSV file of users and their information:
- Go to **Administration** > **User Management**.
- Ensure you're in the** Users **tab.
- Click **Download**.
[See image](#Image1).
[]![CSV File Download Tab](/downloads/zia/authentication-administration/provisioning-authenticating-users/user-management/users/downloading-user-information-csv-file/CSV%20file%20-%20Download%20tab.png?1647968467)
This CSV file can't be used to [import users](/zia/importing-user-information-csv-file) because it uses a different format. If you need to import a CSV file to make modifications to users, use the same format as the **Sample Import CSV file** provided by Zscaler.
