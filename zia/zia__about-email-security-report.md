# About the Email Security Report
Source: https://help.zscaler.com/zia/about-email-security-report
PDF: https://help.zscaler.com/pdf/com/en/1479341.pdf

The Email Data Loss Prevention (DLP) Report gives visibility and insight into emails with sensitive data and users with incidents within your organization. The report is divided into two tabs: Overview and Incidents.
The Email Security Report provides the following benefits and enables you to:
- Gain visibility into sensitive data movements in your organization via emails.
- Analyze email activities and users that triggered Email DLP rules within your organization.
About the Email Security Report Page
On the Email Security Report page (Analytics > Email Security Report > Overview), you can do the following:
- Filter the report for the last 1 day, 7 days, or last month.
- **Analyzed Emails**: View the overall count of emails. The bottom of the section shows an increase or decrease in the percentage of emails for the selected time period.
- **Emails with Sensitive Information**: View the number of emails with sensitive information.
- **Incidents: **View the number of incidents that got triggered by the matching Email DLP rules. The bottom of the section shows an increase or decrease in the percentage of incidents for the selected time period.
- **Users Generating Incidents**: View the number of unique users generating the incidents. The bottom of the section shows an increase or decrease in the percentage of users for the selected time period.
- **Email with Sensitive Data by DLP Engines**: View the number of emails that are categorized based on DLP engines.
- **Incidents Distribution by DLP Engines**: View the number emails incidents that are categorized based on DLP engines.
- Go to the [Incidents](/zia/about-email-security-report-incidents) page.
