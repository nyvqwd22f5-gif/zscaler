# IoT Report
Source: https://help.zscaler.com/zia/iot-report

API Reference Guide for the ZIA Cloud Service and Sandbox Submission APIs

**Base URL:** https://zsapi.[zscaler-cloud-name]/api/v1

## `GET /iotDiscovery/categories`

Retrieve the mapping between the device category universally unique identifier (UUID) values and the category names for all the device categories supported by the Zscaler AI/ML. The parent of device category is device type.

**Tags:** IoT Report
**Consumes:** application/json
**Produces:** application/json
No parameters.

| Code | Description | Schema |
| ---- | ----------- | ------ |
| 200 | successful operation | array[IotItemInfo] |

## `GET /iotDiscovery/classifications`

Retrieve the mapping between the device classification universally unique identifier (UUID) values and the classification names for all the device  classifications supported by Zscaler AI/ML. The parent of device classification is device category.

**Tags:** IoT Report
**Consumes:** application/json
**Produces:** application/json
No parameters.

| Code | Description | Schema |
| ---- | ----------- | ------ |
| 200 | successful operation | array[IotItemInfo] |

## `GET /iotDiscovery/deviceList`

Retrieve a list of discovered devices with the following key contexts, IP address, location, ML auto-label, classification, category, and type.

**Tags:** IoT Report
**Consumes:** application/json
**Produces:** application/json
No parameters.

| Code | Description | Schema |
| ---- | ----------- | ------ |
| 200 | successful operation | IotDeviceInfo |

## `GET /iotDiscovery/deviceTypes`

Retrieve the mapping between device type universally unique identifier (UUID) values and the device type names for all the device types supported by the Zscaler AI/ML.

**Tags:** IoT Report
**Consumes:** application/json
**Produces:** application/json
No parameters.

| Code | Description | Schema |
| ---- | ----------- | ------ |
| 200 | successful operation | array[IotItemInfo] |

## Models
### DeviceInfo
| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| autoLabel | string | No | A label generated by the Zscaler AI/ML engine to describe the device's prominent characteristics. |
| categoryUuid | string | No | The UUID for the device category. |
| classificationUuid | string | No | The UUID for the device classification. |
| deviceTypeUuid | string | No | The UUID of the device type identified by the Zscaler AI/Ml. |
| deviceUuid | string | No | The universally unique identifier (UUID) of the device identified by the Zscaler AI/ML. |
| flowEndTime | integer (int32) | No | The end timestamp at which Zscaler AI/ML engine stops evaluating the device's weblog records. It's noted in epoch seconds. |
| flowStartTime | integer (int32) | No | The start timestamp at which Zscaler AI/ML engine starts evaluating the device's weblog records. It's noted in epoch seconds. |
| ipAddress | string | No | The IP address of the device. |
| locationId | string | No | The location ID where the device resides. |

### IotDeviceInfo
Information about all discovered devices.
| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| cloudName | string | No | The Zscaler cloud on which your organization is provisioned. |
| customerId | integer (int32) | No | The unique identifier for your organization. |
| devices | array[DeviceInfo] | No | A list of DeviceInfo objects. |

### IotItemInfo
Information about the items.
| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| name | string | No | The name of the item. |
| parent_uuid | string | No | The parent UUID of the item.
**Note**: This field is not applicable for the device types endpoint. |
| uuid | string | No | The universally unique identifier (UUID) of the item. |
