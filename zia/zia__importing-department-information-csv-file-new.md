# Importing Department Information from a CSV File
Source: https://help.zscaler.com/zia/importing-department-information-csv-file-new
PDF: https://help.zscaler.com/pdf/com/en/1401121.pdf

You can import up to 3,000 departments using a CSV file. If you want to add multiple users to an existing department or change the department for multiple users, you must use the [user sample import CSV file](/zia/importing-user-information-csv-file).
To import new departments, or modify existing departments, using a CSV file:
- Go to **Administration** > **User Management**.
- Click the **Departments** tab.
- Click **Sample Import CSV file** to download the department information template.
- Enter your department information in the CSV file template.
- Once you have the CSV file in the correct format, click **Import**.
The **Import Departments** window appears.
- In the **Import Departments** window, click **Choose file**, and navigate to the CSV file, then click **Open**.
- Click **Import**.
- After the CSV file is successfully imported, click **Close**.
