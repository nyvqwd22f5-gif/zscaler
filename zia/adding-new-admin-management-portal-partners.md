# Adding a New Admin in the Management Portal for Partners
Source: https://help.zscaler.com/zia/adding-new-admin-management-portal-partners
PDF: https://help.zscaler.com/pdf/com/en/1402166.pdf

Zscaler partners can create admin users with different permissions to manage intended tenants or tenant groups on the Management Portal for Partners.
To add a new admin user:
- Go to **Admins** and click **Add Admin**.
The **Add Admin** window appears.
-
In the **Add Admin** window:
- **Login ID**:
Enter a login ID for the admin user. Select a domain name from the list of domains configured for your organization. A unique login ID is created for the admin in an email format with the information that you provide here.
- **Admin Name**:
Enter a name for the admin to be associated with the login ID.
- **Email ID**: Enter the email ID of the admin user.
- **Role**: Select a role template for the admin.
- **Cloud**: Select a cloud.
- **Tenants**: Select tenants from the respective cloud.
- **Comments**:
Enter additional notes or information. The description cannot exceed 10,240 characters.
- **Password**:
Enter the password to be set for the login ID.
- **Confirm Password**:
Retype the password for confirmation.
[See image.](#add-admin-window)
- Click **Save**.
