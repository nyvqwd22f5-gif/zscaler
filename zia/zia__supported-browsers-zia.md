# Supported Browsers for ZIA
Source: https://help.zscaler.com/zia/supported-browsers-zia
PDF: https://help.zscaler.com/pdf/com/en/1400716.pdf

Zscaler actively tests operating system and browser compatibility with the ZIA Admin Portal.
Windows 10
The ZIA Admin Portal supports the following browsers on Windows 10:
- Google Chrome (two most recent versions)
- Mozilla Firefox (two most recent versions)
- Microsoft Edge (two most recent versions)
Older versions of these browsers should be compatible, but are no longer actively tested.
Other browsers should be compatible, but are not actively tested.
Mac OS X & macOS
The ZIA Admin Portal supports the following Apple operating systems:
- OS X 9
- OS X 10
- macOS 10
The ZIA Admin Portal supports the following browsers on OS X & macOS:
- Google Chrome (two most recent versions)
- Mozilla Firefox (two most recent versions)
Older versions of these browsers should be compatible, but are no longer actively tested.
Other browsers should be compatible, but are not actively tested.
