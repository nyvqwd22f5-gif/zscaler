# Recommended FTP Control Policy
Source: https://help.zscaler.com/zia/recommended-ftp-control-policy
PDF: https://help.zscaler.com/pdf/com/en/1398796.pdf

Zscaler recommends that you configure the following [FTP Control](/zia/about-ftp-control) policy:
- Disable the **Allow FTP over HTTP **option
- Disable the **Allow Native FTP **option
- Disable the** Allow Any URL Category **option
- Set **Allowed URL Categories** to **None**
