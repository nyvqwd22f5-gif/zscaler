# About Devices
Source: https://help.zscaler.com/zia/about-devices
PDF: https://help.zscaler.com/pdf/com/en/1402246.pdf

The Devices page displays all the devices in your organization that have the Zscaler Client Connector deployed. You can select devices when you configure policies​​​​​.
**About the Devices Page**
On the Devices page (Administration > Device Management > Devices), you can:
- View a list of all configured devices. For each device, you can view:
- **Device ID**: The ID of the device.
- **Device Name**: The name of the device.
-
**Device Hostname**: The hostname of the device.
This field displays values only if you have configured [Zscaler Client Connector](/zscaler-client-connector/configuring-zscaler-client-connector-collect-hostnames) to collect the hostname information.
- **OS Type**: The OS type of the device.
- **OS Version**: The OS version of the device.
- **Device Model**: The model of the device.
- **Device Owner**: The owner of the device.
- **Description**: The description of the device.
- View more information about the device.
- [Modify the table and its columns](/zia/how-do-i-use-tables-admin-portal).
- Search for a device.
- Filter the search for devices based on the device name, device model, device owner, device ID, OS type, or OS version.
- Go to the [Device Groups](/zia/about-device-groups) page.
