# Configuring Network Application Groups
Source: https://help.zscaler.com/zia/adding-network-application-groups
PDF: https://help.zscaler.com/pdf/com/en/1399941.pdf

[Watch a video about Network Applications, including how to add a network application group](https://fast.wistia.net/embed/iframe/uog2xsgkhz)
To group together applications that you want to control in a Firewall policy rule, create a network application group:
- Go to **Administration** >** ****Network Applications**.
- In the **Application Groups** tab, click **Add Network Application Group**.
- Enter a **Name** for the application group. It can include any character and spaces.
- Click the down arrow beside **Applications**.
Choose any number of applications or application categories that you want to include in the group. When you are finished, click **Done.**
- (Optional) Type in any additional notes or information about the group. The description cannot exceed 10,240 characters. To learn more, see [Ranges & Limitations](/zia/ranges-limitations).
- Click **Save** and[ activate the change](/zia/saving-and-activating-changes-admin-portal).
