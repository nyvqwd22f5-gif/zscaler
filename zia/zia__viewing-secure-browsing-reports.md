# Viewing Secure Browsing Reports
Source: https://help.zscaler.com/zia/viewing-secure-browsing-reports
PDF: https://help.zscaler.com/pdf/com/en/1399841.pdf

The Zscaler service has Secure Browsing reports, which you can run to track the use of vulnerable browsers, plugins, and applications, and then modify your [Browser Control policy](/zia/about-browser-control) if needed.
Viewing Secure Browsing Reports
To view Secure Browsing reports:
- Go to **Analytics **>** Interactive Reports**.
- In the **Standard Reports** tab, expand the **Secure Browsing** section to see the available browsing reports.
[See image.](#secure-browsing)
To learn more about Interactive Reports, see [About Interactive Reports](/zia/about-interactive-reports).
Viewing Top Users With Vulnerable Browsers, Plugins, and Applications
Additionally, on the Web Insights page, you can view a list of the top users with vulnerable browsers, plugins, and applications. To learn more, see [About Insights](/zia/about-insights).
To view top users with vulnerable browsers, plugins, and applications:
- Go to **Analytics** >** Web Insights**.
- At the top, choose** User **as the **Data Type**.
- On the left pane, choose a time frame and chart type.
- Select the **Secure Browsing Status** filter, and then select **Vulnerable**.
- Click **Apply Filters**.
The service lists users who are using vulnerable browsers, plugins, and applications for which you set warnings.
[See image.](#web-insights)
