# Adding an Index Tool Configuration
Source: https://help.zscaler.com/zia/adding-index-tool-configuration
PDF: https://help.zscaler.com/pdf/com/en/1400661.pdf

To add an [Index Tool](/zia/about-index-tool) configuration:
- Go to **Administration **>** Index Tool**.
- Click **Add Index Tool Configuration**.
The **Add Index Tool Configuration** window appears.
-
In the **Add Index Tool Configuration** window:
- **VM Name**: Enter a unique name for the virtual machine (VM).
- **Status**: Make sure that the VM is **Enabled**.
- **Enable Single Sign-On**: Select **Enabled **if you are configuring single sign-on (SSO) for the Index Tool. To learn more, see [Configuring Single Sign-On for the Index Tool](/zia/configuring-single-sign-index-tool).
[See image.](#add-index-tool)
-
Click **Save** and [activate the change](/zia/saving-and-activating-changes-admin-portal).
After you save, you can download the SSL certificate for the configuration from the [Index Tool page](/zia/about-index-tool) or from the [Edit Index Tool Configuration window](/zia/modifying-index-tool-configuration).
