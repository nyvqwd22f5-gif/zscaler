# About Tenant Groups
Source: https://help.zscaler.com/zia/about-tenant-groups
PDF: https://help.zscaler.com/pdf/com/en/1506526.pdf

In the Management Portal for Partners, you can group tenants across clouds and industries. This feature facilitates easy tenant management as well as policy deployment to multiple tenants across clouds.
The Tenant Groups page provides the following benefits and enables you to:
- View a complete list of tenant groups.
- Deploy policies to multiple tenants across clouds more efficiently.
- Assign tenant groups to admin users for easy tenant management.
About Tenant Groups Page
On the Tenant Groups page (Tenant > Tenant Groups), you can do the following:
- View a list of all tenant groups. For each group, you can view:
- **Name**: The name of the group.
- **Policy Template**: The linked policy's name. A group can only be provisioned in one policy template.
- **Number of Tenants**: The number of tenants in a group.
- **Description**: The group's description.
- **Action**: The action (Edit or Delete) you can take on the group.
