# Configuring Network Service Groups
Source: https://help.zscaler.com/cloud-branch-connector/configuring-network-service-groups
PDF: https://help.zscaler.com/pdf/com/en/1420446.pdf

You can group network services into a service group for easy application of firewall policies.
To create a network services group:
- Go to **Administration **>** ****Network Services**.
- On the **Service Groups** tab, click **Add Network Service Group**.
[See image.](#AddNetworkServiceGroup)
[![Add Network Service Group in the Zscaler Cloud & Branch Connector Admin Portal](/downloads/cloud-connector/administration/firewall-filtering/configuring-network-service-groups/Add%20Network%20Service%20Group.jpg)
]
- In the **Add Network Service Group **window:
- **Name**: Enter a name for the network services group.
- **Service**: Choose any number of custom and predefined services you want to include in the group from the drop-down menu.
- **Description**: Enter additional notes or information about the network services group. The description cannot exceed 10,240 characters.
[See image.](#AddNetworkServiceWindow)
[![Add Network Service Group Window in the Zscaler Cloud & Branch Connector Admin Portal](/downloads/cloud-connector/administration/firewall-filtering/configuring-network-service-groups/Add%20Network%20Service%20Group%20Window.jpg)
]
- Click **Save** and [activate the change](/cloud-branch-connector/saving-and-activating-changes).
