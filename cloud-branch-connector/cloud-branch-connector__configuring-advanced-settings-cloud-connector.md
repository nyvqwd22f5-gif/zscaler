# Configuring Advanced Settings for Cloud Connector
Source: https://help.zscaler.com/cloud-branch-connector/configuring-advanced-settings-cloud-connector
PDF: https://help.zscaler.com/pdf/com/en/1420861.pdf

On the Advanced Settings page, your imported ID is added to your [session logs ](/cloud-branch-connector/session-insights-logs-columns)to correlate your workloads with your Amazon Web Services (AWS) account, Microsoft Azure subscription, and/or Google Cloud Platform (GCP) subscription.
To configure advanced settings:
- Go to** Administration** > **Advanced Settings**.
- Under **AWS**, enable or disable **Import Account ID**. This is the account ID of the Cloud Connector deployed with AWS.
- Under **Azure**, enable or disable **Import Subscription ID**. This is the subscription ID of the Cloud Connector deployed with Azure.
- Under **GCP**, enable or disable **Import Project ID**. This is the project ID of the Cloud Connector deployed with GCP.
- Click **Save** and [activate the change](/cloud-branch-connector/saving-and-activating-changes).
