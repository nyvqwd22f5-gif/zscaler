# Editing Zero Trust Branch Devices
Source: https://help.zscaler.com/cloud-branch-connector/editing-zero-trust-branch-devices
PDF: https://help.zscaler.com/pdf/com/en/1483146.pdf

Modifying Zero Trust Branch Devices is one of the tasks you can complete after deployment.
To edit a Zero Trust Branch Device:
- Go to **Administration** > **ZT Devices**.
-
From the [ZT Devices](/cloud-branch-connector/about-zero-trust-branch-devices) page, locate the device you want to modify and click the **Edit** icon.
[See image.](#EditZTDeviceButton)
- In the **Edit ZT Device** window:
- **Device Name**: Enter a name for the Zero Trust Branch Device.
- **Serial Number**: View the serial number of the Zero Trust Branch Device.
- **Location**: View the location of the Zero Trust Branch Device.
- **Template**: View the template the Zero Trust Branch Device.
- **Description**: Enter a description for the Zero Trust Branch Device.
- **Zscaler Initiated On-Demand Support Tunnel**: Select **Enable** to give Zscaler Support control over the tunnel's start and stop. Select **Disable** to not grant Zscaler Support control.
-
**Establish Support Tunnel**: Select **Enable** to allow Zscaler Support to access the Zero Trust Branch Device through the tunnel. Select **Disable** to not grant Zscaler Support access.
[See image.](#EditZTDeviceWindow)
- Click Save and [activate the change](/cloud-branch-connector/saving-and-activating-changes).
