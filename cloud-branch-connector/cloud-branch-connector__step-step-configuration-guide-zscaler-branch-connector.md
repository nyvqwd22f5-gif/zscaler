# Step-by-Step Configuration Guide for Zscaler Branch Connector
Source: https://help.zscaler.com/cloud-branch-connector/step-step-configuration-guide-zscaler-branch-connector
PDF: https://help.zscaler.com/pdf/com/en/1420916.pdf

This guide takes you through the configuration steps you need to complete before using Zscaler Branch Connector for your organization.
Before you begin configuring Branch Connector, Zscaler recommends reading the following articles:
- [What is Zscaler Branch Connector?](/cloud-branch-connector/what-zscaler-branch-connector)
- [Accessing and Navigating the Zscaler Cloud & Branch Connector Admin Portal](/cloud-branch-connector/accessing-navigating-zscaler-cloud-branch-connector-admin-portal)
- [Using the Zscaler Help Browser](/cloud-branch-connector/using-zscaler-help-browser)
Configuring Branch Connector
To configure Branch Connector, complete the following steps:
- [Step 1: Update Administrator and Role Management Information](#admin)
- [Step 2: Configure your Locations](#locations)
- [Step 3: Configure your Provisioning Template](#provisioning)
- [Step 4: Configure your Firewall Filtering](#firewall-filtering)
- [Step 5: Configure your Gateways](#gateways)
- [Step 6: Configure your Forwarding Methods](#forwarding-methods)
- [Step 7: Configure your Analytics](#analytics)
