# Zscaler Client Connector for VDI Release Summary (2025)
Source: https://help.zscaler.com/cloud-branch-connector/zscaler-client-connector-vdi-release-summary-2025

This article provides a summary of all new features and enhancements released per operating system (OS) for the Zscaler Client Connector for VDI on Zscaler Cloud & Branch Connector.

**Clouds:** Windows, macOS, iOS, Android, ChromeOS, Linux
The Zscaler Client Connector for VDI versions for Windows listed below were deployed on

## November 07, 2025
### Feature in Limited Availability: Zscaler Client Connector for VDI 1.7.0.4 for Windows
#### Zscaler Client Connector for VDI 1.7.0.4 Enhancements and Fixes
- Contains a new installer parameter, `SHOWREAUTHNOTIFICATION`, which makes reauthentication notifications configurable.
- Fixes an issue where the user does not receive a notification from Zscaler Client Connector for VDI when a Zscaler Private Access (ZPA) timeout policy is activated.

## October 31, 2025
### Feature Available: Zscaler Client Connector for VDI 1.6.0.6 for Windows
#### Zscaler Client Connector for VDI 1.6.0.6 Enhancements and Fixes
Fixes an issue where Zscaler Client Connector for VDI was crashing due to a race condition when the system shut down.

## October 03, 2025
### Feature Available: Zscaler Client Connector for VDI 1.6.0.5 for Windows
#### Zscaler Client Connector for VDI 1.6.0.5 Enhancements and Fixes
Fixes an issue that stopped the control channel from restarting when sending outbound packets and not receiving inbound packets.

## August 12, 2025
### Feature in Limited Availability: Zscaler Client Connector for VDI 1.6.0.3 for Windows
#### Zscaler Client Connector for VDI 1.6.0.3 Enhancements and Fixes
Improves the stability of the Zscaler Client Connector for VDI control channel.

## June 13, 2025
### Feature Available: Zscaler Client Connector for VDI 1.5.0.8 for Windows
#### Zscaler Client Connector for VDI 1.5.0.8 Enhancements and Fixes
Fixes an issue where WebView2 has intermittent issues with the discovery of PAC settings.

## June 03, 2025
### Feature Available: Zscaler Client Connector for VDI 1.5.0.7 for Windows
#### Zscaler Client Connector for VDI 1.5.0.7 Enhancements and Fixes
- Fixes an issue where the Zscaler Client Connector for VDI application window did not open for some users.
- Fixes an issue where Zscaler Client Connector for VDI dropped traffic as a non-standard user.

## March 31, 2025
### Feature Available: Zscaler Client Connector for VDI 1.5.0.5 for Windows
#### Zscaler Client Connector for VDI 1.5.0.5 Enhancements and Fixes
Adds a registry option to prevent Zscaler Client Connector for VDI from onboarding on system reboot.

## February 07, 2025
### Feature Available: Zscaler Client Connector for VDI 1.4.0.8 for Windows
#### Zscaler Client Connector for VDI 1.4.0.8 Enhancements and Fixes
Fixes an error that caused control channel connections to fail when the certificate had not expired.

## January 06, 2025
### Feature Available: Zscaler Client Connector for VDI 1.4.0.7 for Windows
#### Zscaler Client Connector for VDI 1.4.0.7 Enhancements and Fixes
- Increases the authorization timeout from 60 seconds to 180 seconds.
- Fixes an issue where default policies in Virtual Desktop Infrastructure (VDI) forwarding profiles were not bypassed.
