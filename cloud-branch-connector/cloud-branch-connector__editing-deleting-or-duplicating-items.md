# Editing, Deleting, or Duplicating Items
Source: https://help.zscaler.com/cloud-branch-connector/editing-deleting-or-duplicating-items
PDF: https://help.zscaler.com/pdf/com/en/1420491.pdf

In the Zscaler Cloud & Branch Connector Admin Portal, you can edit, delete, or duplicate items that you've configured. For example, policy rules and other administrative items are configurable.
- [To edit items](#edit)
- [To delete items](#delete)
- [To duplicate items](#duplicate)
- []Navigate to the item you want to edit.
- Click the **Edit** icon.
![Edit icon from the Traffic Forwarding page in the Zscaler Cloud & Branch Connector Admin Portal](/downloads/cloud-connector/getting-started/admin-portal/editing-deleting-or-duplicating-items/Edit%20icon%201.png)
- Make the necessary changes and click **Save**.
- []Navigate to the item you want to delete.
- Click the **Delete** icon.
![Delete icon from the Traffic Forwarding page in the Zscaler Cloud & Branch Connector Admin Portal](/downloads/cloud-connector/getting-started/admin-portal/editing-deleting-or-duplicating-items/Delete%20icon%201.png)
- Read the warning message that appears, then click **Delete**.
![Delete Confirmation window from the Traffic Forwarding page in the Zscaler Cloud & Branch Connector Admin Portal](/downloads/tech-pubs-drafts/cloud-connector-draft-articles/editing-deleting-or-duplicating-items-draft-doc-46452/Delete%20Confirmation%20Window.jpg)
- []Navigate to the item you want to duplicate.
- Click the **Duplicate** icon.
![Duplicate icon from the Traffic Forwarding page in the Zscaler Cloud & Branch Connector Admin Portal](/downloads/cloud-connector/getting-started/admin-portal/editing-deleting-or-duplicating-items/Duplicate%20icon%202.png)
If you duplicate a rule, the newly created rule is identical to the existing one, except for the value in the Rule Order field. The new rule shows the next available rule order value.
