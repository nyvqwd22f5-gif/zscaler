# Searching in the Zscaler Cloud & Branch Connector Admin Portal
Source: https://help.zscaler.com/cloud-branch-connector/searching-zscaler-cloud-branch-connector-admin-portal
PDF: https://help.zscaler.com/pdf/com/en/1420761.pdf

To help you navigate faster, you can search for menus in the Zscaler Cloud & Branch Connector Admin Portal. Search results include the portal's menu in the left-side navigation (except for items in the Help menu) and do not include:
- Tabs
- Field names
- Toggle names
- Tooltips or other text
To search the portal:
- Hover over the **Search **icon in the left-side navigation. The search panel appears.
[See image.](#Search)
- Enter your search text.
The search provides auto-complete suggestions.
[See image.](#Search-Suggestions)
- Select your desired suggestion.
