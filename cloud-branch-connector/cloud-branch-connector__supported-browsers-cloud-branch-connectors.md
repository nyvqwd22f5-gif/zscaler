# Supported Browsers for Cloud & Branch Connectors
Source: https://help.zscaler.com/cloud-branch-connector/supported-browsers-cloud-branch-connectors
PDF: https://help.zscaler.com/pdf/com/en/1420766.pdf

Zscaler actively tests the operating system and browser compatibility with the Zscaler Cloud & Branch Connector Admin Portal.
Windows 10 & 11
The portal supports the following browsers on Windows 10 Pro and Windows 11 Pro:
- Google Chrome (two most recent versions)
- Mozilla Firefox (two most recent versions)
- Microsoft Edge (two most recent versions)
Older versions of these browsers should be compatible, but are no longer actively tested.
Other browsers should be compatible, but are not actively tested.
Mac OS X & macOS
The portal supports the following Apple operating systems:
- OS X 9
- OS X 10
- macOS 10
The portal supports the following browsers on OS X & macOS:
- Google Chrome (two most recent versions)
- Mozilla Firefox (two most recent versions)
Older versions of these browsers should be compatible, but are no longer actively tested.
Other browsers should be compatible, but are not actively tested.
