# Usage Guidelines for Zero Trust SD-WAN Devices
Source: https://help.zscaler.com/cloud-branch-connector/usage-guidelines-zero-trust-branch-devices
PDF: https://help.zscaler.com/pdf/com/en/1468281.pdf

Zero Trust Software-Defined Wide Area Network (SD-WAN) Devices are installed in an organization's data center and are dedicated to the organizationâ€™s traffic. The Zscaler Operations team performs the following tasks on the organization's Zero Trust SD-WAN Device:
- Provides the required hardware and software for the initial installation of the Zero Trust SD-WAN Device.
- Ensures that the Zero Trust SD-WAN Device is suitably packed for shipment.
- Upgrades or replaces the Zero Trust SD-WAN Device to maintain the operational standards of Zscaler. Zscaler will ship the hardware to the organization. It is the organization's responsibility to set up the replacement hardware.
- Maintains the technical refresh status of the Zero Trust SD-WAN Device.
- Provides remote upgrades of software on the Zero Trust SD-WAN Device on a schedule that conforms with Zscaler's standard release and maintenance practices.
Zero Trust SD-WAN Devices remain the property of Zscaler during the applicable subscription term.
Organization's Responsibilities
For Zscaler Operations to service, manage, and maintain the Zero Trust SD-WAN Device, Zscaler requires the organization to:
- Provide a local point of contact for delivery acceptance and assist Zscaler with any specific requirements for delivery.
- Ensure that the Zero Trust SD-WAN Device is not moved from one physical location to another without prior approval from Zscaler.
- Provide a continuous, uninterrupted, and suitable power supply.
- Implement and maintain temperature, humidity, and other environmental conditions that Zscaler recommends.
- Implement suitable surge protection devices.
- Ensure that each Zero Trust SD-WAN Device has internet connectivity.
- Ensure that the Zero Trust SD-WAN Device is used to provide Zscaler service only and is not used for any other purpose.
- Ensure that only Zscaler staff access or service the Zero Trust SD-WAN Device. Neither the organization nor any third party can make any repair attempts or other changes to the Zero Trust SD-WAN Device unless they have explicit approval from Zscaler.
- Manage all network-related issues.
- Use the Zero Trust SD-WAN Device only in accordance with instructions prescribed by Zscaler and only with hardware and software provided by Zscaler. Also, ensure that the Zero Trust SD-WAN Device is installed and operated in accordance with the specifications and recommendations provided by Zscaler.
- Comply with the firewall access requirements, if you use a firewall to control access to the products on the Zero Trust SD-WAN Device.
- Connect the Zero Trust SD-WAN Device to the appropriate network resources recommended by Zscaler and provide the network resources the necessary IP addresses to run the service.
If there are any issues, you can receive Zero Trust SD-WAN support services through the Zscaler Cloud & Branch Connector Admin Portal (https://connector.<Zscaler Cloud Name>.net/login) or the [Zscaler Help Portal](https://help.zscaler.com/contact-support). Zscaler Support works with you to troubleshoot and diagnose the issues remotely. If the issue cannot be resolved remotely or if it is a hardware fault, Zscaler Support might decide to replace the Branch Connector equipment.
Zscaler offers comprehensive return and advanced replacement services for Zero Trust SD-WAN Devices. For details and for information about initiating hardware replacement, see [Return Merchandise Authorization Process and Policy](/cloud-branch-connector/return-merchandise-authorization-process-and-policy).
To learn more about the Zero Trust SD-WAN Devices, see the detailed [device requirements](https://www.zscaler.com/legal/zero-trust-branch-connector-device).
