# About Zero Trust Branch Devices
Source: https://help.zscaler.com/cloud-branch-connector/about-zero-trust-branch-devices
PDF: https://help.zscaler.com/pdf/com/en/1467021.pdf

Zero Trust Branch Devices are installed in your organizationâ€™s on-premises locations such as branches and data centers. They enable Zero Trust connectivity from any unauthenticated users, servers, or IoT/OT devices in any location or over any network. To learn more, see [Understanding Zero Trust Branch Devices](/cloud-branch-connector/understanding-zero-trust-branch-devices).
Zero Trust Branch Devices provide the following benefits and enable you to:
- Use Zero Trust Branch Devices everywhere for all unauthenticated users, devices, servers, and IoT/OT regardless of location.
- Eliminate the attack surface by connecting branches and data centers through the Zero Trust Exchange (ZTE) independent of the underlying wide area network (WAN) transport network.
- Prevent lateral threat movement because the ZTE builds a foundation for secure connectivity that enables east-to-west segmentation.
About the Zero Trust Branch Devices Page
On the ZT Devices page (Administration > ZT Devices), you can do the following:
- View a list of appliances configured for your organization. You can see:
- **Appliance Name**: The name of the Zero Trust Branch Device.
- **Serial Number**: The serial number of the Zero Trust Branch Device.
- **Device Model**: The model of the Zero Trust Branch Device (ZT400, ZT600, or ZT800).
- **Location**: The location of the Zero Trust Branch Device.
- **Template**: The template for the Zero Trust Branch Device.
- **Status**: The status of the Zero Trust Branch Device.
- [Edit](/cloud-branch-connector/editing-zero-trust-branch-devices) a Zero Trust Branch Device.
- Modify the table and its columns.
- Search for a Zero Trust Branch Device.
