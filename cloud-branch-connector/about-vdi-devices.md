# About VDI Devices
Source: https://help.zscaler.com/cloud-branch-connector/about-vdi-devices
PDF: https://help.zscaler.com/pdf/com/en/1471691.pdf

In the Zscaler Cloud & Branch Connector Admin Portal, the VDI Devices page displays all the devices in your organization that have Zscaler Client Connector for VDI deployed.
Virtual Desktop Infrastructure (VDI) devices provide the following benefits and enable you to:
- View the VDI devices with Zscaler Client Connector for VDI installed.
- Assess the VDI device status and details.
About the VDI Devices Page
On the VDI Devices page (Administration > VDI Device Management), you can do the following:
- View a list of all VDI devices. For each device, you can see:
- **Hostname**: The name of the VDI device.
- **OS Type**: The operating system (OS) of the VDI devices associated with the VDI group. Zscaler Client Connector for VDI uses the OS type that it detects to add the device to a group.
- **OS Version**: The version of the OS associated with the VDI device.
- **Platform**: The platform associated with the VDI device.
- Modify the table and its columns.
- Search for a VDI device.
- Go to the [VDI Groups](/cloud-branch-connector/about-vdi-groups) page.
