# Configuring Network Service Groups
Source: https://help.zscaler.com/zia/configuring-network-service-groups
PDF: https://help.zscaler.com/pdf/com/en/1399976.pdf

[Watch a video about Network Services, including how to add a network service group.](https://fast.wistia.net/embed/iframe/fa5a6c5wtm)
To create a network services group:
- Go to **Administration **>** Network Services**.
- In the **Service Groups** tab, click **Add Network Service Group**.
- Enter a **Name** for the network services group. It can include any character and spaces.
- Click the down arrow beside **Services**.
- Choose any number of custom and predefined services you want to include in the group.
- (Optional) Type in any additional notes or information about the group. The description cannot exceed 10,240 characters. To learn more, see [Ranges & Limitations](/zia/ranges-limitations).
- Click **Save** and [activate the change](/zia/saving-and-activating-changes-admin-portal).
