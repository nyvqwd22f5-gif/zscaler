## `GET /mgmtconfig/v1/admin/customers/{customerId}/idp/{idpId}`
Gets details of the IdP for the specified ID.

**Tags:** IdP Management

| Name | In | Type | Required | Description |
| ---- | --- | ---- | -------- | ----------- |
| customerId | path | integer (int64) | Yes | The unique identifier of the ZPA tenant. |
| idpId | path | integer (int64) | Yes | The unique identifier of the IdP. |

## `GET /mgmtconfig/v2/admin/customers/{customerId}/idp`
Gets all configured IdP details for the specified customer.

**Tags:** IdP Management

| Name | In | Type | Required | Description |
| ---- | --- | ---- | -------- | ----------- |
| customerId | path | integer (int64) | Yes | The unique identifier of the ZPA tenant. |
| scimEnabled | query | boolean | No | Returns all SCIM IdPs if set to true. Returns all non SCIM IdPs if set to false. |
| userAttributes | query | boolean | No |  |
| search | query | string | No | The search string used to support search by features and fields for the API. |
| page | query | integer (int32) | No | Specifies the page number. |
| pagesize | query | integer (int32) | No | Specifies the page size. If not provided, the default page size is 20. The max page size is 500. |
