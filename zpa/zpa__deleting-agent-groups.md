# Deleting Agent Groups
Source: https://help.zscaler.com/zpa/deleting-agent-groups
PDF: https://help.zscaler.com/pdf/com/en/1531958.pdf

Admins can [configure](/zpa/configuring-agent-groups), [edit](/zpa/editing-agent-groups), and delete agents and agent groups as needed.
To delete an agent group:
- Go to **Microsegmentation** > **Agent Management**.
- Click the** Agent** **Groups **tab.
[See image.](#Agent-Management_Delete-Agent)
- Click the **Delete **icon for the agent group you want to delete.
The **Delete Agent Group** confirmation window appears.
[See image.](#Delete-Agent-window)
- Click **Delete**.
The page refreshes and removes the agent group from the list.
