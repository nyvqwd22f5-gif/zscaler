# About Agent Telemetry
Source: https://help.zscaler.com/zpa/about-agent-telemetry
PDF: https://help.zscaler.com/pdf/com/en/1531940.pdf

The Agent Telemetry page gives insight into your organization's agent data. It allows you to monitor and analyze the agent data for all installed agents in your organization.
The Agent Telemetry page provides the following benefits and enables you to:
- View a summary of agent data in your organization.
- View, monitor, and analyze agent data from a single workload.
- Take appropriate actions based on the data displayed on the page about the agents.
- Discover insights about any agent errors.
About the Agent Telemetry Page
On the Agent Telemetry page (Microsegmentation > Agent Management > Agent Telemetry), you can do the following:
- Filter the time range for up to 14 days in the past or a custom time range.
- Refresh the Agent Telemetry page to reflect the most current information.
- View a list of applied filters available from the current and previous user sessions. Applied filters must be saved to the user session first before they can be viewed. Use the drop-down menu to select the applied filters to view. To learn more, see [Using Tables](https://help.zscaler.com/zpa/using-tables).
- Show or hide the filter options.
- Filter the information by **Agent CPU Utilization**,** Agent Memory Utilization**,** Agent ID**,** **and **Agent Virtual Memory Utilization**.
- View a list of all agent data for your organization. For each agent, you can see:
- **Timestamp**: The date and time of the agent's last usage.
- **Agent CPU Utilization**: The percentage of CPU in megabytes that the agent used.
- **Agent Memory Utilization**: The total memory in megabytes that the agent used.
- **Agent ID**: The agent ID number.
- Expand all agents to view their complete information.
- Expand an individual agent to view its complete information.
- [Modify the columns displayed in the table.](https://help.zscaler.com/zpa/using-tables)
- Open the [Zscaler Help Browser](https://help.zscaler.com/zpa/using-zscaler-help-browser) and view Help Portal articles without leaving the ZPA Admin Portal.
- Go to the [Flow Logs](/zpa/about-flow-logs-page) page to view, monitor, and analyze flow data.
- Go to the [Agent Connection Status Logs](/zpa/about-agent-connection-status-logs) page to view, monitor, and analyze agent connection data.
