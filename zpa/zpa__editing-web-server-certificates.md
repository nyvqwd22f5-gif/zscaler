# Editing (Web Server) Certificates
Source: https://help.zscaler.com/zpa/editing-web-server-certificates
PDF: https://help.zscaler.com/pdf/com/en/1484106.pdf

[]To edit a (web server) certificate:
- Go to **Configuration & Control **>** Certificate Management** > **Certificates**.
- In the table, locate the certificate you want to modify and click the **Edit** icon.
The **Edit Certificate** window appears.
- In the **Edit Certificate** window, modify fields as necessary. You can see more details about the certificate, including the common name (CN) and certificate's expiration date.
In order to replace an expired certificate, you must upload a new one. To learn more, see [About (Web Server) Certificates](/zpa/about-web-server-certificates).
![Browser Access Certificates page with Edit Certificate window within the ZPA Admin Portal](/downloads/zpa/documentation-knowledgebase/certificates/browser-access-certificates/editing-browser-access-web-server-certificates/zpa-editbrowseraccesscert.png)
- Click **Save**.
