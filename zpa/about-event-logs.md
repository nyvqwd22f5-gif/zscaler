# About Event Logs
Source: https://help.zscaler.com/zpa/about-event-logs
PDF: https://help.zscaler.com/pdf/com/en/1531941.pdf

The Event Logs page provides logs of resource and agent diagnostics data. The logs displayed on this page can help you troubleshoot resource and agent issues depending on the priority.
The Event Logs page provides the following benefits and enables you to:
- View a summary of resource and agent data in your organization.
- View, monitor, and analyze agent diagnostics from a single workload.
- Troubleshoot resource and agent issues.
- Take appropriate actions based on the event log data displayed.
About the Event Logs Page
On the Event Logs page (Microsegmentation > Analytics > Event Logs), you can do the following:
- Filter the time range up to 14 days in the past or a custom time range.
- Refresh the Event Logs page to reflect the most current information.
- View the number of **Total**, **High Priority**, **Medium Priority**, and **Low Priority** event logs.
- View a list of applied filters available from the current and previous user sessions. Applied filters must be saved to the user session first before they can be viewed. Use the drop-down menu to select the applied filters to view. To learn more, see [Using Tables](https://help.zscaler.com/zpa/using-tables).
- Show or hide the filter options.
- Filter the information by** Agent Group Name**, **Agent Name**, **AppZone Name**,** Category**,** Component Name**,** Event Name**,** Resource Group Name**, and** Resource Name**.
- Expand all event logs to view their complete information.
- View a list of all event logs that are configured for your organization. For each log, you can see:
- **Timestamp**: The date and time the log occurred.
- **Event Name**: The title of the log.
- **Category**: The action type of the log.
- **Component Name**: The area of Microsegmentation where the log came from.
- **Priority**: The ranked importance of the log shown in Low, Medium, and High.
- Expand an individual event log to view its complete information.
- [Modify the columns displayed in the table.](https://help.zscaler.com/zpa/using-tables)
- Go to the [Flow Logs](/zpa/about-flow-logs-page) page to see network and agent flows.
- Go to the [Agent Telemetry](/zpa/about-agent-telemetry-page) page to see specific agent data.
- Go to the the [Agent Connection Status Logs](https://help.zscaler.com/zpa/about-agent-connection-status-logs) page to see agent connection data.
