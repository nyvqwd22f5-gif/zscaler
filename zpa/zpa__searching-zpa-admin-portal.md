# Searching on the ZPA Admin Portal
Source: https://help.zscaler.com/zpa/searching-zpa-admin-portal
PDF: https://help.zscaler.com/pdf/com/en/1484271.pdf

To help you navigate faster, you can search for menus in the ZPA Admin Portal. Search results include Admin Portal menus in the left-side navigation.
Search results do not include items in the Help menu, tabs, field names, toggle names, tooltips, or other UI text.
Enter your search in the **Search** field.
