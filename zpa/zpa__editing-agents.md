# Editing Agents
Source: https://help.zscaler.com/zpa/editing-agents
PDF: https://help.zscaler.com/pdf/com/en/1531953.pdf

Admins can [configure](/zpa/configuring-agents), edit, and [delete](/zpa/deleting-agents) agents as needed.
To edit an agent:
- Go to **Microsegmentation** > **Agent Management**.
- Click the** Agents** tab.
- Click the **Edit **icon for the agent you want to edit.
The **Edit Agent** window appears.
[See image.](#Agent-Management_Edit-Agent)
-
In the **Edit Agent** window:
- **Name**: Edit the name of the agent.
- **Admin Status**: Select **Enabled**,** Disabled**, or** Inherit From Group**. Depending on the settings of the agent group that the agent is a part of, the inherit option will be either **Inherit (Enabled)** or **Inherit** **(Disabled)**.
- **Policy Status**: Enable or disable the Policy Status.
- **Description**: Edit the description.
You cannot edit the additional information. Review it as necessary.
[See image.](#Edit-Agent-window)
- Click **Save**.
Your updated agent appears in the list of Agents.
