# Editing Privileged Portals
Source: https://help.zscaler.com/zpa/editing-privileged-portals
PDF: https://help.zscaler.com/pdf/com/en/1485106.pdf

To edit a privileged portal:
- Go to **Resource Management **>** Privileged Remote Access** >** Privileged Portals**.
- In the table, locate the privileged portal you want to modify and click the **Edit **icon.
- In the **Edit Portal **window, modify fields as necessary. To learn more about each field, see [Configuring Privileged Portals](/zpa/configuring-privileged-portals).
![Edit Portal window within the ZPA Admin Portal](/downloads/zpa/privileged-remote-access-management/privileged-portals/editing-privileged-portals/Editing%20Privileged%20Portals.jpg)
- Click **Save**.
