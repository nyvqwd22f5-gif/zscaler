# Editing User Portal Links
Source: https://help.zscaler.com/zpa/editing-application-links
PDF: https://help.zscaler.com/pdf/com/en/1484341.pdf

To edit a portal link:
- Go to **Resource Management **>** User Portal **> **Portal Links**.
- In the table, locate the link you want to modify and click the **Edit **icon.
- In the **Edit Portal Link **window, modify fields as necessary. To learn more about each field, see [Configuring Portal Links](/zpa/configuring-application-links-user-portals).
After you update a field, you can preview how the link will appear within the user portal by clicking **Preview Link**.
![Edit links in ZPA Admin Portal](/downloads/zpa/user-portal/editing-application-links/zpa-editportallinks.png)
- Click **Save**.
