# Editing User Portals
Source: https://help.zscaler.com/zpa/editing-user-portals
PDF: https://help.zscaler.com/pdf/com/en/1484331.pdf

To edit a user portal:
- Go to **Resource Management **> **User Portal **> **User Portals**.
- In the table, locate the user portal you want to modify and click the **Edit **icon.
- In the **Edit User Portal **window, modify fields as necessary. To learn more about each field, see [Configuring User Portals](/zpa/configuring-user-portals).
![Edit User Portal window within the ZPA Admin Portal](/downloads/zpa/user-portal/editing-user-portals/Editing%20User%20Portal_0.png)
- Click **Save**.
