# About Resources
Source: https://help.zscaler.com/zpa/about-resources
PDF: https://help.zscaler.com/pdf/com/en/1498241.pdf

The Resources page gives insight into your organization's configured object data. It allows you to monitor and analyze the data for all configured agents and AppZones in your organization.
The Resources page provides the following benefits and enables you to:
- View a summary of resource data in your organization.
- View, monitor, and analyze resource data for a single workload.
- Take appropriate actions based on the resource data displayed.
- Discover insights about collective resource data.
About the Resources Page
On the Resources page (Microsegmentation > Resource Management > Resources), you can do the following:
- Refresh the Resources page to reflect the most current information.
- View a list of applied filters available from the current and previous user sessions. Applied filters must be saved to the user session first before they can be viewed. Use the drop-down menu to select the applied filters to view. To learn more, see [Using Tables](https://help.zscaler.com/zpa/using-tables).
- Show or hide the filter options.
- Filter the information by **Name**, **Type**, **Platform**, **AppZone**, **Agent ID**, **Cloud**, **Resource ID**, and **Status**.
- View a list of all resources that are configured for your organization. For each resource, you can see:
- **Name**: The name of the resource.
- **AppZone**: The name of the AppZone connected to the resource.
- **Type**: The type of resource.
- **Cloud**: The cloud name where the resource resides.
- **Platform**: The operating system where the resource resides.
- **Status**: The status of the resource.
- Click an individual resource to view its complete information.
[See image.](#resource-info-window)
- [Modify the columns displayed in the table.](https://help.zscaler.com/zpa/using-tables)
- Display more rows or a different page of the table.
- Open the [Zscaler Help Browser](https://help.zscaler.com/zpa/using-zscaler-help-browser) and view Help Portal articles without leaving the ZPA Admin Portal.
- Go to the [Resource Groups](/zpa/about-resource-groups) page to view resource groups information.
- Go to the [AppZone](/zpa/about-appzone-page) page to view AppZone cloud and group details.
