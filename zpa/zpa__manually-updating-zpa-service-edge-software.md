# Manually Updating ZPA Private Service Edge Software
Source: https://help.zscaler.com/zpa/manually-updating-zpa-service-edge-software
PDF: https://help.zscaler.com/pdf/com/en/1484531.pdf

Once a ZPA Private Service Edge is scheduled for an update you can't [change the periodic software update](/zpa/scheduling-periodic-software-updates-zpa-service-edge-group) time for that particular ZPA Private Service Edge. However, you can update the ZPA Private Service Edge's software manually.
To manually update software for a ZPA Private Service Edge:
- Go to **Configuration & Control** >** Private Infrastructure** > **Service Edge Management** > **Service Edges**.
- In the table, locate the ZPA Private Service Edge you want to update.
- In the **Update Status** column, verify that the status is **Scheduled**. If the status is not **Scheduled**, then you can not manually update the ZPA Private Service Edge.
- Click the **Update Now** icon.
![Manually updating a Private Service Edge for ZPA](/downloads/zpa/manually-updating-zpa-service-edge-software/zpa-serviceedgegroup-updatenowicon.png)
- Click **Confirm**.
