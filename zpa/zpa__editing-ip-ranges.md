# Editing IP Ranges
Source: https://help.zscaler.com/zpa/editing-ip-ranges
PDF: https://help.zscaler.com/pdf/com/en/1485606.pdf

To edit an IP range:
- Go to **Configuration & Control **>** Administration Control **> **Client Connector IP Assignment** > **IP Ranges**.
- In the table, locate the IP range you want to modify and click the **Edit **icon.
- In the **Edit IP Range** window, modify the fields as necessary. To learn more about each field, see [Adding IP Ranges](/zpa/adding-ip-ranges).
![Edit IP Range window in the ZPA Admin Portal](/downloads/zpa/administration/peer-peer-connectivity/editing-ip-ranges/zpa-editing-ip-ranges.png)
- Click **Save**.
