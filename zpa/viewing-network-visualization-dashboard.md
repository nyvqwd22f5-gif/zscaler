# Viewing the Network Visualization Dashboard
Source: https://help.zscaler.com/zpa/viewing-network-visualization-dashboard
PDF: https://help.zscaler.com/pdf/com/en/1498196.pdf

The Network Viz dashboard provides graphed visualizations of network data in your organization.
To view the Network Viz dashboard, go to **Microsegmentation** > **Dashboard** > **Network Viz**.
![A GIF of the Network Viz dashboard. ](/downloads/zpa/microsegmentation/viewing-network-visualization-dashboard/Network-viz-dashboard_interactive-view_with-graph_GIF-VERSION_3.gif)
Dashboard Tools
The Network Visualization dashboard displays the following information and functionality:
- Use the arrow keys on the left side of the graph to change the view of the data.
- Change the selection tool by clicking the hand button below the arrow keys on the left side of the graph.
- Zoom in or out to expand or narrow the view of the data.
- Enter a VPC or App Zone into the search field of the graph to display that item's specific data.
- Click an App Zone or VPC from the graph to open the drawer of its full details:
- **General Information**
- Name
- Description
- Member Count
- Include All VPCs
- Include All Subnets
[See image.](#Network-Viz-Gen-Info)
- **Member Subnets**
- App Zone clouds selected
[See image.](#Network-Viz-AppZone-clouds)
- **Member Resources**
- Name
- Type
- Platform
[See image.](#Network-Viz-Member-Resources)
- Click on a Resource to open the drawer of its full details:
- **General Information**
- Host Name
- Platform
- Platform Distro
- Platform Version
- Agent ID
- Resource ID
- IP Addresses
- Last Seen
- **Environment**
- Account ID
- Cloud Resource ID
- Cloud Provider
- Region
- VPC IDs
- Subnet IDs
- SG IDs
- AppZone Names
- **Tags**
- ApplicationTier
- Stack
- AppName
- Name
[See image.](#Network-viz-resource)
