# ZPA Private Service Edge Software by Platform
Source: https://help.zscaler.com/zpa/zpa-private-service-edge-software-by-platform
PDF: https://help.zscaler.com/pdf/com/en/1485961.pdf

ZPA Private Service Edges are supported on [many different platforms](/zpa/private-service-edge-management/private-service-edge-deployment-guides-supported-platforms). Each supported platform has a ZPA Private Service Edge image you can use to deploy ZPA Private Service Edges on that platform. To learn more, see [Private Service Edge Deployment Guides for Supported Platforms](/zpa/private-service-edge-management/private-service-edge-deployment-guides-supported-platforms) for detailed deployment instructions.
The following platforms support ZPA Private Service Edge software packages. Where applicable, see the download links for images, public keys, and checksums.
