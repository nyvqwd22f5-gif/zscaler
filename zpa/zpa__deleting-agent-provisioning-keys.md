# Deleting Agent Provisioning Keys
Source: https://help.zscaler.com/zpa/deleting-agent-provisioning-keys
PDF: https://help.zscaler.com/pdf/com/en/1531961.pdf

Agent provisioning keys are created during an agent or agent group configuration. You can [edit](/zpa/editing-agent-provisioning-keys) or delete provisioning keys as needed outside of the agent or agent group configuration.
To delete an agent provisioning key:
- Go to **Microsegmentation** > **Agent Management**.
- Click the** Agent** **Provisioning Keys **tab.
- Click the **Delete** icon for the provisioning key you want to delete.
The **Delete Agent Provisioning Key** confirmation window appears.
[See image.](#Agent-Management_Delete-Agent-Provisioning-Key)
- Click **Delete**.
[See image.](#Delete-Agent-provisioning-key-window)
The page refreshes and removes the agent provisioning key from the list.
