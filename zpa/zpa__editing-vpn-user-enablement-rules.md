# Editing VPN User Enablement Rules
Source: https://help.zscaler.com/zpa/editing-vpn-user-enablement-rules
PDF: https://help.zscaler.com/pdf/com/en/1529083.pdf

To edit a VPN user enablement rule:
- Go to **VPN (for Legacy Apps) **> **User Enablement** > **User Enablement**.
- Locate the user enablement rule you want to modify and click the **Edit **icon.
If you only want to change a user enablement rule's place in the **Rule Order**, you can manually enter a new value on the [VPN User Enablement page](/zpa/about-vpn-user-enablement).
- In the **Edit User Enablement **window, modify fields as necessary. For more information about each field, see [Configuring VPN User Enablement](/zpa/configuring-vpn-user-enablement).
![Edit User Enablement window within the ZPA Admin Portal](/downloads/zpa/privileged-remote-access-management/privileged-approvals/editing-privileged-approvals/Editing%20User%20Enablement%20window.png)
- Click **Save**.
