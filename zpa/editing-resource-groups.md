# Editing Resource Groups
Source: https://help.zscaler.com/zpa/editing-resource-groups
PDF: https://help.zscaler.com/pdf/com/en/1531947.pdf

You can [configure](/zpa/configuring-resource-groups), edit, and [delete](/zpa/deleting-resource-groups) resource groups at any time to provide flexibility in managing security posture.
To edit a resource group:
- Go to **Microsegmentation** > **Resource Management**.
- Click the **Resource Groups **tab.
- Click the **Edit** icon.
The **Edit Resource Group **window appears.
[See image.](#Policy-Resource-Policy_Add-Rule)
- In the **Edit Resource Group **window, edit the options as necessary. To learn more, see [Configuring Resource Groups](/zpa/configuring-resource-groups).
- Click **Save**.
