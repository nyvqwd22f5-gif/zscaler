# About Agent Connection Status Logs
Source: https://help.zscaler.com/zpa/about-agent-connection-status-logs
PDF: https://help.zscaler.com/pdf/com/en/1498236.pdf

The Agent Connection Status Logs page provides agent connection logs to the Zscaler cloud. The logs displayed on this page can help you troubleshoot connection issues.
The Agent Connection Status Logs page provides the following benefits and enables you to:
- View a summary of agent connection data in your organization.
- View, monitor, and analyze agent connection data from a single workload.
- Troubleshoot agent connection issues.
- Debug the count of logs occurring.
- Discover insights about different types of agent connection errors.
- Take appropriate actions based on the agent connection data displayed.
About the Agent Connection Status Logs Page
On the Agent Connection Status Logs page (Microsegmentation > Analytics > Agent Connection Status Logs), you can do the following:
- Filter the time range up to 14 days in the past.
- Refresh the Agent Connection Status Logs page to reflect the most current information.
- View a list of applied filters available from the current and previous user sessions. Applied filters must be saved to the user session first before they can be viewed. Use the drop-down menu to select the applied filters to view. To learn more, see [Using Tables](https://help.zscaler.com/zpa/using-tables).
- Show or hide the filter options.
- Filter the information by **Status**,** Agent Public IP**,** Agent ID**,** Agent Name**,** Agent Source Port**,** Connection Type**,** **and **Disconnect Reason**.
- Expand all status logs to view their complete information.
- View a list of all agent connection status logs that are configured for your organization. For each log, you can see:
- **Date**: The date the log occurred.
- **Status**: The authentication status of the log.
- **Authentication/Disconnect Time**: The time when the agent connection was authenticated or disconnected.
- **Agent Name**: The name of the agent.
- **Agent Controller Name**: The name of the agent controller.
- **Agent Public IP**: The IP address for the agent.
- Expand an individual status log to view its complete information.
- [Modify the columns displayed in the table.](https://help.zscaler.com/zpa/using-tables)
- Display more rows or a different page of the table.
- Open the [Zscaler Help Browser](https://help.zscaler.com/zpa/using-zscaler-help-browser) and view Help Portal articles without leaving the ZPA Admin Portal.
- Go to the [Flow Logs](/zpa/about-flow-logs-page) page to see network and agent flows.
- Go to the [Agent Telemetry](/zpa/about-agent-telemetry-page) page to see specific agent data.
