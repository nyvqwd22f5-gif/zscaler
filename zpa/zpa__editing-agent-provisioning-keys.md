# Editing Agent Provisioning Keys
Source: https://help.zscaler.com/zpa/editing-agent-provisioning-keys
PDF: https://help.zscaler.com/pdf/com/en/1531960.pdf

Agent provisioning keys are created during an agent or agent group configuration. You can edit or [delete](/zpa/deleting-agent-provisioning-keys) provisioning keys as needed outside of the agent or agent group configuration.
To edit an agent provisioning key:
- Go to **Microsegmentation** > **Agent Management**.
- Click the** Agent Provisioning Keys** tab.
- Click the **Edit **icon for the provisioning key you want to edit.
The **Edit Agent Provisioning Key** window appears.
[See image.](#Agent-Management_Edit-Provisioning-Key)
- In the **Edit Agent Provisioning Key** window:
- **Name: **Edit the provisioning key name.
- **Maximum Reuse of Key**: The number of times the key can be reused. Enter a number from 1 to 1,000.
- **Agent Group**: View the agent group selection. This cannot be changed.
- **Signing Certificate**: Select a signing certificate from the drop-down menu.
[See image.](#Edit-Agent-Provisioning-Key-window)
- Click **Save**.
Your updated provisioning key appears in the list of agent provisioning keys.
