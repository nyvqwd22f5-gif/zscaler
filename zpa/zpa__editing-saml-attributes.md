# Editing SAML Attributes
Source: https://help.zscaler.com/zpa/editing-saml-attributes
PDF: https://help.zscaler.com/pdf/com/en/1483651.pdf

To edit a SAML attribute:
- Go to **Authentication **>** User Authentication **>** SAML Management**.
- In the table, locate the SAML attribute you want to modify and click the **Edit icon**.
- In the **Edit SAML Attribute **window, modify fields as necessary. You cannot modify the selected **IdP Configuration**. If you need to adjust this attribute for another [IdP configuration](/zpa/about-idps), you must import or add it again for that IdP. To learn more, see [Importing SAML Attributes](/zpa/about-importSamlAttributes) and [Manually Adding SAML Attributes](/zpa/about-samlattribute/new).
![Edit SAML Attribute window within the ZPA Admin Portal](/downloads/zpa/documentation-knowledgebase/saml-attributes/midp-about-samlattribute-edit/zpa-editsamlattribute.png)
- Click **Save**.
