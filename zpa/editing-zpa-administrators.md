# Editing ZPA Administrators
Source: https://help.zscaler.com/zpa/editing-zpa-administrators
PDF: https://help.zscaler.com/pdf/com/en/1483666.pdf

To edit an administrator:
- Go to **Configuration & Control > Administration Control** > **Administrators**.
- In the table, locate the admin you want to modify and click the **Edit** icon.
- In the** Edit Administrator** window, modify fields as necessary. To learn more, including how to set up two factor authentication (2FA) for existing admin accounts, see [Configuring ZPA Administrators](/zpa/about-user/new).
![Edit Administrator window within the ZPA Admin Portal](/downloads/zpa/documentation-knowledgebase/admin-settings/admins-and-roles/editing-zpa-administrators/zpa-editadmin.png)
- Click **Save**.
