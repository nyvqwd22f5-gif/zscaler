# About Agent Groups
Source: https://help.zscaler.com/zpa/about-agent-groups
PDF: https://help.zscaler.com/pdf/com/en/1531955.pdf

The Agent Groups page gives insight into your organization's agent group data. It allows you to monitor and analyze the data for all configured agent groups in your organization.
The Agent Groups page provides the following benefits and enables you to:
- View a summary of agent group data in your organization.
- View, monitor, and analyze agent group data for a single workload.
- Take appropriate actions based on the agent group data displayed.
- Discover insights about collective agent group data.
About the Agent Groups Page
On the Agent Groups page (Microsegmentation > Agent > Agent Groups), you can do the following:
- Refresh the Agent Groups page to reflect the most current information.
- Hide the filter options.
- Filter the information by **Name**, **Count, Status**, **Upgrade Status**, **Version Profile**, **Region**, **Subnet ID**, and **VPC ID**.
- [Add](/zpa/configuring-agent-groups) an agent group.
- Click an individual agent group to view its complete information.
[See image.](#resource-info-window)
- [Edit](/zpa/editing-agent-groups) an agent group.
- Download an agent group CSV file.
- [Delete](/zpa/deleting-agent-groups) an agent group.
- Go to the** **[Agents](/zpa/about-agents) page to view, monitor, and analyze agent data.
- Go to the** **[Agent Provisioning Keys](/zpa/about-agent-provisioning-keys-page) page to view, monitor, and analyze agent provisioning keys data.
