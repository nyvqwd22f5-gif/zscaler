# About Supported Browsers
Source: https://help.zscaler.com/zpa/about-supported-browsers
PDF: https://help.zscaler.com/pdf/com/en/1484151.pdf

Zscaler actively tests operating system and browser compatibility with the ZPA Admin Portal.
Windows 10
The ZPA Admin Portal supports the following browsers on Windows 10:
- Google Chrome (two most recent versions)
- Mozilla Firefox (two most recent versions)
- Microsoft Edge (two most recent versions)
- Safari (two most recent versions)
Older versions of these browsers should be compatible, but are no longer actively tested.
Other browsers should be compatible, but are not actively tested.
Mac OS X & macOS
The ZPA Admin Portal supports the following Apple operating systems:
- OS X 9
- OS X 10
- macOS 10
The ZPA Admin Portal supports the following browsers on OS X and macOS:
- Google Chrome (two most recent versions)
- Mozilla Firefox (two most recent versions)
Older versions of these browsers should be compatible, but are no longer actively tested.
Other browsers should be compatible, but are not actively tested.
