# Adding a Private Cloud Controller to an Existing Private Cloud Controller Group
Source: https://help.zscaler.com/zpa/adding-private-cloud-controller-existing-private-cloud-controller-group
PDF: https://help.zscaler.com/pdf/com/en/1507166.pdf

To add a Private Cloud Controller to an existing Private Cloud Controller group:
- Go to **Configuration & Control** > **Business Continuity** > **Business Continuity Management** > **Private Cloud Controllers**.
- Click **Add Private Cloud Controller**.
The **Add Private Cloud Controller** page appears.
- On the **Choose Key** tab:
- Select **Choose an existing provisioning key**.
- Select an existing provisioning key from the drop-down menu. Ensure that the key you select is associated with the appropriate Private Cloud Controller group. You can click **Clear **to remove any selections.
[See image.](#seeimage8)
- Click **Next**.
- On the **Review Documentation** tab:
- **Copy Provisioning Key**: Copy the Private Cloud Controller provisioning key. You need to enter this key when you deploy the Private Cloud Controller to a platform.
- **Review Documentation**: Select the platform you want to deploy your Private Cloud Controller on and follow the instructions that appear.
[See image.](#seeimage7)
- Deploy your Private Cloud Controller on the chosen platform.
- Click **Done**.
