# Requesting Approvals in the PRA Portal
Source: https://help.zscaler.com/zpa/requesting-approvals-pra-portal
PDF: https://help.zscaler.com/pdf/com/en/1520621.pdf

After the admin has enabled privileged requests in a privileged portal policy, you can access the[ My Requests page](/zpa/about-my-approvals) in the PRA Portal.
To create a privileged approval request:
- Click the **My Approvals** icon on the [PRA Portal home page](/zpa/accessing-privileged-remote-access-portal).
- Click the **My Requests** tab.
- Click **Add Request**.
The **Create Approvals** page appears.
- On the **Create Approvals** page:
- Select a Microtenant from the drop-down menu. The Default Microtenant is automatically selected.
- Select privileged consoles.
- Click **Next**.
[See image.](#step1)
- Select a start date in the calendar.
- Select the duration that you want this privileged approval to last.
- (Optional) Enter a reason for access.
[See image.](#time)
- Click **Next**.
- Review the privileged approval request.
[See image.](#review)
- Click **Save**.
The saved privileged approval request appears on the My Requests page and is ready to be reviewed by an approver.
