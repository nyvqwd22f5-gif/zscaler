# Editing Agent Groups
Source: https://help.zscaler.com/zpa/editing-agent-groups
PDF: https://help.zscaler.com/pdf/com/en/1531957.pdf

Admins can [configure](/zpa/configuring-agent-groups), edit, and [delete](/zpa/deleting-agent-groups) agent groups as needed.
To edit an agent group:
- Go to **Microsegmentation** > **Agent Management**.
- Click the** Agent Groups** tab.
- Click the **Edit** icon for the agent group you want to edit.
The **Edit Agent Group** window appears.
[See image.](#Agent-Management_Add-Agent-Group)
- In the **Edit Agent Group** window, modify the information as necessary. To learn more, see [Configuring Agent Groups](/zpa/configuring-agent-groups).
If you change the Admin Status, you are asked if you also want to change the Admin Status for all agents in that group. To learn more, see [Editing Agents](https://help.zscaler.com/zpa/editing-agents).
[See image.](#Update-Agent-Settings_Override-Admin-Status)
- Click **Save**.
Your updated agent group reflects the changes in the list of agent groups.
