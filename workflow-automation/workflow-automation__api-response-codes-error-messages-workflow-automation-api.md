# API Response Codes and Error Messages
Source: https://help.zscaler.com/workflow-automation/api-response-codes-error-messages-workflow-automation-api
PDF: https://help.zscaler.com/pdf/com/en/1452111.pdf

The following HTTP status codes are returned by the API:
| Code | Description |
| ---- | ----------- |
| 200 | Successful |
| 201 | Successful and a new resource is created |
| 400 | Bad request |
| 401 | Authorization failure |
| 404 | Resource does not exist |
| 500 | Failed to process the request |
