# Adding Integrations for ZDX Alerts
Source: https://help.zscaler.com/workflow-automation/adding-integrations-zdx-alerts
PDF: https://help.zscaler.com/pdf/com/en/1487461.pdf

The Integration page allows you to integrate Workflow Automation with SaaS application, ServiceNow, sanctioned by Zscaler. For Zscaler Digital Experience (ZDX) alerts that are captured in Workflow Automation, admins can configure the ServiceNow ticketing system when creating automated workflows to create and assign tickets for every alert. To learn more, see [Managing ServiceNow Integration for ZDX Alerts](/workflow-automation/managing-workflow-automation-integration-servicenow-zdx).
