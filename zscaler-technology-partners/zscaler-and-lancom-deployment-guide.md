# Zscaler and LANCOM Deployment Guide
Source: https://help.zscaler.com/zscaler-technology-partners/zscaler-and-lancom-deployment-guide
PDF: https://help.zscaler.com/pdf/com/en/1390341.pdf

The Zscaler and LANCOM Deployment Guide provides instructions on integrating Zscaler Internet Access (ZIA) and Zscaler Private Access (ZPA) with the LANCOM platform.
This deployment guide describes the configuration of a VPN connection to the Zscaler cloud. Deploying IPSec allows the use of dynamic WAN addresses on the client side.
