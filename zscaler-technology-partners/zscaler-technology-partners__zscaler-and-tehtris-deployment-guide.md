# Zscaler and TEHTRIS Deployment Guide
Source: https://help.zscaler.com/zscaler-technology-partners/zscaler-and-tehtris-deployment-guide
PDF: https://help.zscaler.com/pdf/com/en/1451861.pdf

The Zscaler and TEHTRIS Deployment Guide provides instructions on integrating Zscaler Internet Access (ZIA) and Zscaler Private Access (ZPA) to work with the TEHTRIS security information and event management (SIEM) platform.
TEHTRIS SIEM collects and filters billions of logs from different network sources (operating systems, cloud, firewalls, proxies, databases, antivirus, etc.). Thanks to multiple alert rules, TEHTRIS SIEM raises security alerts on specific events logged. From the TEHTRIS XDR Platform, TEHTRIS SIEM lets you search for more precise details directly in collected and parsed logs.
When integrated with Zscaler, TEHTRIS SIEM can ingest raw log data from ZIA and ZPA using the Zscaler Nanolog Streaming Service (NSS).
