# Zscaler and Rubrik Deployment Guide
Source: https://help.zscaler.com/zscaler-technology-partners/zscaler-and-rubrik-deployment-guide
PDF: https://help.zscaler.com/pdf/com/en/1451691.pdf

The Zscaler and Rubrik Deployment Guide provides instructions on integrating Zscaler Internet Access (ZIA) with the Rubrik platform.
The Rubrik and Zscaler Data Loss Prevention (DLP) integration automatically sends sensitive files found by Sensitive Data Discovery and backed up by Rubrik to Zscaler’s service for DLP.
Zscaler’s implementation prevents sending any files indexed to be exfiltrated out of the network to services such as SFTP, email, FTP, etc.
There are three core parts to the integration:
- Zscaler DLP configuration.
- Zscaler Rubrik Integration configuration.
- Rubrik Sensitive Data Discovery policy enablement.
For more information on Rubrik as a Zscaler partner, see the [Rubrik partner page](https://www.zscaler.com/partners/rubrik).
