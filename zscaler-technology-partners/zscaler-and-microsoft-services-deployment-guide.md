# Zscaler UVM and Microsoft Services Deployment Guide
Source: https://help.zscaler.com/zscaler-technology-partners/zscaler-and-microsoft-services-deployment-guide
PDF: https://help.zscaler.com/pdf/com/en/1517791.pdf

The Zscaler UVM and Microsoft Services Deployment Guide provides instructions on integrating Zscaler Unified Vulnerability Management (UVM) with Microsoft Services.
Zscaler Unified Vulnerability Management (UVM) is a cloud-native solution that delivers comprehensive visibility and control over vulnerabilities, misconfigurations, and security risks across an organizationâ€™s environment. It seamlessly integrates with Microsoft technologies and Azure services to enhance risk management.
Key integrations include:
- Microsoft Defender for Cloud: Leverages findings for robust risk assessment.
- Microsoft Intune: Ingests audit events and asset data to enhance endpoint security.
- Microsoft Defender for Endpoint: Uses alerts, vulnerabilities, and software insights by machine for actionable intelligence.
- Azure Cloud Assets and Azure Blob: Offers visibility into cloud storage and resources to address risks like data breaches and misconfigurations.
