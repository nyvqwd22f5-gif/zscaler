# Zscaler and Imprivata Deployment Guide
Source: https://help.zscaler.com/zscaler-technology-partners/zscaler-and-imprivata-deployment-guide
PDF: https://help.zscaler.com/pdf/com/en/1478731.pdf

The Zscaler and Imprivata Deployment Guide provides instructions on integrating the Zscaler Client Connector in both EXE and MSI formats on the Imprivata Enterprise Access Management shared workstation.
The Imprivata Enterprise Access Management is an appliance-based solution that integrates strong authentication, application single-sign on, physical access control, and event reporting. Imprivata Enterprise Access Management delivers automated access management capabilities within a single easy-to-use administrative framework.
This deployment guide describes how to install the Zscaler Client Connector in both EXE and MSI formats on the Imprivata Enterprise Access Management shared workstation. While there are specific options required, the installation is otherwise standard. You can also use additional CLI options as needed.
