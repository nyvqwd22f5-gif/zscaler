# Zscaler and NetWitness Deployment Guide
Source: https://help.zscaler.com/zscaler-technology-partners/zscaler-and-netwitness-deployment-guide
PDF: https://help.zscaler.com/pdf/com/en/1474061.pdf

The Zscaler and NetWitness Deployment Guide provides instructions on how to configure Zscaler Internet Access (ZIA) to work with the NetWitness Platform XDR.
You can configure Zscaler to send logs via the Nanolog Streaming Service (NSS) to a NetWitness Platform XDR Data Collector. This guide provides the necessary actions and steps to configure log forwarding on Zscaler NSS.
