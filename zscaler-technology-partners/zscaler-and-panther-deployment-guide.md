# Zscaler and Panther Deployment Guide
Source: https://help.zscaler.com/zscaler-technology-partners/zscaler-and-panther-deployment-guide
PDF: https://help.zscaler.com/pdf/com/en/1510336.pdf

The Zscaler and Panther Deployment Guide provides instructions on using Zscaler Internet Access (ZIA) and Zscaler Private Access (ZPA) features with the Panther OS.
Panther is a leading provider of next-generation SIEM solutions that detect and respond to security threats in real time. Leveraging a modern architecture, Panther offers speed, scale, and accuracy in threat detection, helping organizations stay ahead of cyber threats.
You can ingest ZIA Admin Audit logs into Panther by using either an HTTP source or an AWS S3 bucket.
