# Zscaler and XM Cyber Deployment Guide
Source: https://help.zscaler.com/zscaler-technology-partners/zscaler-and-xm-cyber-deployment-guide
PDF: https://help.zscaler.com/pdf/com/en/1529419.pdf

The Zscaler and XM Cyber Deployment Guide provides instructions on using Zscaler Internet Access (ZIA) with the XM Cyber Security Controls Monitoring (SCM).
XM SCM provides a comprehensive cybersecurity awareness and compliance management platform that acts as a single source of truth for the security posture of your entire hybrid infrastructure.
This guide helps users to integrate Zscaler and XM Cyber to enhance threat detection and response efficiency.
