# Zscaler and CyberArk Deployment Guide
Source: https://help.zscaler.com/zscaler-technology-partners/zscaler-and-cyberark-deployment-guide
PDF: https://help.zscaler.com/pdf/com/en/1412791.pdf

The Zscaler and CyberArk Deployment Guide demonstrates how to integrate Zscaler Internet Access (ZIA) and Zscaler Private Access (ZPA) with the CyberArk Access platform.
CyberArk provides comprehensive security offerings for identity across business applications, distributed workforces, hybrid cloud workloads, and throughout the DevOps lifecycle. CyberArk uses SAML authentication to implement an Identity Provider (IdP) solution and benefit from an SSO workflow across multiple domains. This deployment guide is an overview of the steps required to configure ZIA or ZPA for single sign-on (SSO) via SAML.
