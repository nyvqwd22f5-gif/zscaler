# Zscaler and IGEL Deployment Guide
Source: https://help.zscaler.com/zscaler-technology-partners/zscaler-and-igel-deployment-guide
PDF: https://help.zscaler.com/pdf/com/en/1509981.pdf

The Zscaler and IGEL Deployment Guide provides instructions on using clientless Zscaler Internet Access (ZIA) and Zscaler Private Access (ZPA) features with the IGEL OS.
IGEL OS and Zscaler work together to provide a streamlined approach to empowering browser-based productivity from secure employee and contractor endpoints. The IGEL OS offers a robust platform to access Zscaler's Isolation and Privileged Remote Access (PRA) solutions. This interoperability delivers a secure, zero trust endpoint, and enables streamlined access to Software as a Service (SaaS), private apps, remote desktops, servers, or devices.
