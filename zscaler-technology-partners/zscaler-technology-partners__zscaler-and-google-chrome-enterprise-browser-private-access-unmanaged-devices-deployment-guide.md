# Zscaler and Google Chrome Enterprise Browser for Private Access Unmanaged Devices Deployment Guide
Source: https://help.zscaler.com/zscaler-technology-partners/zscaler-and-google-chrome-enterprise-browser-private-access-unmanaged-devices-deployment-guide
PDF: https://help.zscaler.com/pdf/com/en/1526286.pdf

The Zscaler and Google Chrome Enterprise Browser for Private Access Unmanaged Devices Deployment Guide provides instructions on integrating Zscaler Private Access (ZPA) with the Google Chrome Enterprise browser.
The Zscaler and Google Chrome Enterprise Browser integration enables organizations requiring device posture access policies for unmanaged devices. This integration leverages Chrome Enterprise Device Trust Connectors as inputs in ZPA posture control for Access policy validation before granting access to a web-based private application.
