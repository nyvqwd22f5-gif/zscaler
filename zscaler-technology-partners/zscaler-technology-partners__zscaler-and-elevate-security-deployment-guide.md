# Zscaler and Elevate Security Deployment Guide
Source: https://help.zscaler.com/zscaler-technology-partners/zscaler-and-elevate-security-deployment-guide
PDF: https://help.zscaler.com/pdf/com/en/1420991.pdf

The Zscaler and Elevate Deployment Guide provides instructions on integrating Zscaler Internet Access (ZIA) with the Elevate Security Platform.
The Elevate Security Platform simplifies analyzing and defending your organization against insider risk by integrating and ingesting valuable employee security data from all your current cybersecurity tools and systems.
The integration between Zscaler and Elevate Security leverages the Zscaler cloud Nanolog Streaming Service (NSS) to get activity pushed into the Elevate Security Platform.
