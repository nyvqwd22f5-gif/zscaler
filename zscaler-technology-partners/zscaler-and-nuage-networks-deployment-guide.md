# Zscaler and Nuage Networks Deployment Guide
Source: https://help.zscaler.com/zscaler-technology-partners/zscaler-and-nuage-networks-deployment-guide
PDF: https://help.zscaler.com/pdf/com/en/1474556.pdf

The Zscaler and Nuage Networks Deployment Guide provides instructions on how to configure Zscaler Internet Access (ZIA) for use with the Nuage Networks Software Defined Wide Area Network (SD-WAN) Portal.
The Nuage SD-WAN Portal provides service providers and subscribers a quick way to set up and manage wide area networks, including privacy and security.
This guide describes how to use ZIA to manage security in the Nuage Networks SD-WAN Portal.
