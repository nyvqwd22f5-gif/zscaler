# Zscaler UVM and Snyk Deployment Guide
Source: https://help.zscaler.com/zscaler-technology-partners/zscaler-and-snyk-deployment-guide
PDF: https://help.zscaler.com/pdf/com/en/1528920.pdf

The Zscaler UVM and Snyk Deployment Guide provides instructions on using Zscaler Unified Vulnerability Management (UVM) with the Snyk platform.
The Zscaler UVM solution ingests, normalizes, and unifies data across enterprise security and business systems to deliver actionable insights, analytics, and operational efficiencies.
Zscaler UVM offers the following preconfigured Snyk connectors:
- Snyk Connector: Retrieves Software Composition Analysis (SCA) vulnerabilities discovered in your project dependencies.
- Snyk SAST Connector: Static Application Security Testing (SAST) retrieves potential vulnerabilities discovered by analyzing your source code.
