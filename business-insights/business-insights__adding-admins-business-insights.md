# Adding Admins in Business Insights
Source: https://help.zscaler.com/business-insights/adding-admins-business-insights
PDF: https://help.zscaler.com/pdf/com/en/1473176.pdf

This article describes how to add admins for the Business Insights Admin Portal.
To add an admin:
- Go to **Administration **> **Administrators**.
- Click **Add Admin**.
The **Add Administrator** drawer appears.
- In the **Add Administrator** drawer:
- **Login ID**: Enter the login ID the admin uses to log in and select the appropriate domain name. The domain names you provided to Zscaler appear in the drop-down menu.
- **Email**: Enter the admin's email address.
- **Name**: Enter a name for the admin.
- **Role**: Choose a role to specify the admin's level of access to the Business Insights Admin Portal. To configure a new role, see [Adding Admin Roles](/business-insights/adding-admin-roles-business-insights).
- **Scope**: Choose the admin scope to specify which areas of the organization the admin can manage in the Business Insights Admin Portal:
- **Organization**: Provides the ability to manage all areas in the Business Insights Admin Portal.
- **Offices**: Select admin scope over specific configured offices.
- **Office Geolocations**: Select admin scope over configured offices based on the geography (i.e., city, state, country, etc.).
- **Comments**: (Optional) Enter additional notes or information. The comments cannot exceed 10,240 characters.
- **Set Password**: Enable **Password Based Login** if you want the admins to authenticate using a password.
- **Password**: Enter a password for the admin. It can be 8 to 100 characters and must contain at least one number, one special character, and one uppercase letter.
- **Confirm Password**: Re-enter the password to confirm.

-
Click **Save**.
The admin is successfully created.
You can [edit or delete](/business-insights/editing-or-deleting-items-business-insights) admins at any time.
