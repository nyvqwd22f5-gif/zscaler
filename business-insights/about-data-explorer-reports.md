# About Data Explorer Reports
Source: https://help.zscaler.com/business-insights/about-data-explorer-reports
PDF: https://help.zscaler.com/pdf/com/en/1532022.pdf

You can schedule Data Explorer reports for your customized views on the Data Explorer page. To schedule a Data Explorer report, see [Scheduling Data Explorer Reports](/business-insights/scheduling-data-explorer-reports).
Data Explorer reports provide the following benefits and enable you to:
- Schedule reports for various criteria and distribute them to email recipients.
- Generate organized reports based on department, geolocation, user, or workplace.
About the Data Explorer Reports Page
On the Data Explorer Reports page (Analytics > Reports > Data Explorer Reports), you can do the following:
- Search for a scheduled report by its name.
- View a list of all configured scheduled reports. For each report, you can view:
- **Name:** The name of the scheduled report.
- **Summary:** A brief summary of the report. This is autogenerated based on the scheduled report.
- **Scheduled**: Indicates whether the report is active or not.
- **Next Run**: The date when the next report is scheduled to be generated.
- **Last Updated**: The date when the scheduled report was last modified.
- **Last Updated By**: The email address of the user who last modified the scheduled report.
- [Modify the table and its columns](/business-insights/using-tables-business-insights).
- [Edit a scheduled report](/business-insights/editing-or-deleting-items-business-insights).
- Download a report.
- [Delete a scheduled report](/business-insights/editing-or-deleting-items-business-insights).
