# Uploading Application Contracts
Source: https://help.zscaler.com/business-insights/uploading-application-contracts
PDF: https://help.zscaler.com/pdf/com/en/1505871.pdf

The article explains how to upload contracts in the Business Insights Admin Portal to view their insights.
To upload a contract:
-
Go to **Applications** > **All Contracts** > **Upload Contracts**.
The **Upload Contracts** window appears.
-
Click **Upload Files**. You can upload multiple files in one attempt. The Business Insights service supports PDF, DOC, and XLS files.
[See image.](#upload-window)
- Locate the contract file in your system.
-
Select the files and click **Open**.
The file is uploaded successfully and moved to the [Processing Queue](/business-insights/about-processing-queue) page where the file is parsed for necessary metadata. You can go to the Processing Queue page to view its status. Depending on the size and number of files, the data is processed within minutes, hours, or days. After the file is successfully processed, the contract information is available in the All Contracts table on the [All Contracts](/business-insights/about-contracts) page.
