# About ZIA Users
Source: https://help.zscaler.com/business-insights/about-zia-users
PDF: https://help.zscaler.com/pdf/com/en/1508371.pdf

The ZIA Users page show the user data synced from your Zscaler Internet Access (ZIA) tenant. Any user configuration or changes that you make on the [User Management](/zia/authentication-administration/provisioning-authenticating-users/user-management) pages in the ZIA Admin Portal are synced to this page.
The ZIA Users page provides the following benefits and allows you to:
- View the users added to your ZIA tenant.
- View the list of users that are computed for Business Insights.
About the ZIA Users Page
On the ZIA Users page (Administration > ZIA Users > ZIA Users), you can do the following:
- Search for a user by User ID or Name, Group, or Department.
- View a list of configured users. For each user, you can view:
- **User ID or Name**: The user's username or domain name.
- **User Name**: The user's display name.
- **Groups**: The groups the user belongs to.
- **Department**: The department the user belongs to.
- **Status**: The status of the user (**Enabled **or **Disabled**).
- **Comments**: The comments regarding the user, if available.
- [Modify the table and its columns](/business-insights/using-tables-business-insights).
- Go to the [Departments](/business-insights/about-departments)** **page.
