# Configuring Data Explorer Views
Source: https://help.zscaler.com/business-insights/configuring-data-explorer-views
PDF: https://help.zscaler.com/pdf/com/en/1529732.pdf

Data Explorer provides the flexibility to build and organize your own customized views of metrics to analyze data.
To configure a view:
-
Go to **Analytics **> **Data Explorer**.
The **Data Explorer** page appears.
[See image.](#template)
You can select from existing templates and modify them as necessary, or you can create your own view.
- In the **Metrics** section, select the metrics for the view. You can select multiple metrics:
- Bytes
- Visits
- Occupancy
- Attendance
- In the **Operations** section, structure the view using the following operators:
- [a. Group By](#view-groupby)
- [b. Aggregation](#aggregation)
- [c. Aggregation Type](#view-aggtype)
- In the **Filters** section, click **Add** to narrow the results of your view for a specific **Division**, **Department**, and **Work Mode**.
- In the **Chart Type** section, select a chart type icon to display data in a specific chart or tabular format.
- In the **Time Period** section, select an option to process the data for the **Last 7 days**, **Week**, **Month**, or **3 Months**.
-
Click **Run Query** to display your customized view in the right pane.
[See image.](#config-view)
You can click **Reset** to configure another view.
You can analyze the data using the following options:
- Click **Export Full Report** to download a detailed report.
- Click **Export **to download the data as a CSV file.
- Click **Save and Schedule** to schedule a report based on the applied filter. You can schedule Data Explorer reports only when the time period selected is **Last Day**, **Last Week**, or **Last Month**. To learn more, see [Scheduling Data Explorer Reports](/business-insights/scheduling-data-explorer-reports).
- Search for a specific metric to select or deselect it from the view.
-
Select or deselect metrics from the chart view.
