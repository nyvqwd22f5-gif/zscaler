# Integrating a SaaS Application for Workflow Automation
Source: https://help.zscaler.com/business-insights/integrating-saas-application-workflow-automation
PDF: https://help.zscaler.com/pdf/com/en/1500051.pdf

You can integrate different SaaS applications with Workflow Automation for the Business Insights service.
To integrate a SaaS application, you can:
- [Add the SaaS application](/business-insights/adding-saas-application-tenant-workflow-automation) or
- [Ensure Workflow Automation is enabled for an already added application](#enable-workflow)
The SaaS application is successfully integrated with Workflow Automation for the Business Insights service. To learn more, see [What Is Workflow Automation?](/workflow-automation/what-workflow-automation)
[]To enable Workflow Automation for an application:
- Go to **Administration **> **Workflow Integration**.
- Locate the application for which you want to enable Workflow Automation, and click the **Edit** icon.
-
Select the **Workflow Automation** checkbox.
[See image.](#workflow-option-img)
- Click **Save**.
