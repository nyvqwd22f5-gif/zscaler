# Using the Zscaler Help Browser
Source: https://help.zscaler.com/business-insights/using-zscaler-help-browser
PDF: https://help.zscaler.com/pdf/com/en/1480196.pdf

The Zscaler Help Browser is available for all pages within the Business Insights Admin Portal. Through the Help Browser you can view Zscaler Help Portal articles in context with the page you're on, without leaving the Business Insights Admin Portal.
To open the Help Browser:
- Go to the page where you want to view an article.
- Click the **Open Help Browser** icon in the lower-right corner.
[See image.](#seeimage1)
The Help Browser displays the article and any video relevant to the page you're viewing
[See image.](#seeimage2)
About the Help Browser
The Help Browser is also adjustable. There are different features to help you customize the browser and navigate within it.
Customizing the Help Browser
You can do the following to customize the Help Browser:
- [Move the Help Browser](#move)
- [Resize the Help Browser](#resize)
- [Change the font size](#fonts)
Zscaler saves your customization until you log out or the session expires.
Navigating within the Help Browser
You can do the following to navigate within the Help Browser:
- View the article relevant to the page you're viewing in the Business Insights Admin Portal.
- Navigate back and forth across articles you've viewed.
- Search for an article.
- Copy the URL of an article.
- Close the Help Browser.
- Open the article in new browser tab.
![Screenshot of the Help Browser and its features](/downloads/business-insights/getting-started/admin-portal/using-zscaler-help-browser/Business-Insights-documentation-browser-navigation-options_0.png)
Closing the Help Browser clears your navigation history. When you reopen the browser, it displays the article relevant to the page you are viewing, and you are unable to navigate back and forth to articles you've viewed in the previous Help Browser session.

[]You can resize the Help Browser to expand videos, pictures, and content. When you hover to the top-left corner of the browser, an **Expand** cursor appears. Click and drag the corner to resize the window.
