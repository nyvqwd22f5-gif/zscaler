# About Locations
Source: https://help.zscaler.com/business-insights/about-locations-business-insights
PDF: https://help.zscaler.com/pdf/com/en/1506226.pdf

You can view the locations synced from the ZIA Admin Portal on the ZIA Locations page in the Business Insights Admin Portal.
Locations provide the following benefits and enable you to:
- View locations configured from the ZIA Admin Portal.
- Download the location data to a CSV file.
About the ZIA Locations Page
On the Locations page (Administration > ZIA Locations), you can do the following:
- Download the data to a CSV file.
- Search for a location.
- View a list of configured locations in the [ZIA Admin Portal](/zia/about-locations). For each location, you can view:
- **Name**: The name of the location.
- **Sublocations**: The number of sublocations, if any.
- **IP Addresses**: The static IP addresses of the local gateway for the location.
- [Modify the table and its columns](/business-insights/using-tables-business-insights).
