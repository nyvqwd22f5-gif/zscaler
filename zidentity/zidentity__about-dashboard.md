# About the Dashboard
Source: https://help.zscaler.com/zidentity/about-dashboard
PDF: https://help.zscaler.com/pdf/com/en/1500041.pdf

The ZIdentity dashboard provides a high-level overview of all the users that are managed from ZIdentity, their authentication status, and user management activities in the ZIdentity Admin Portal.
The Dashboard provides the following benefits and enables you to:
- View the total number of users managed from ZIdentity.
- Obtain visibility into the user authentication status.
- Track user management activities.
About the Dashboard Page
On the dashboard, you can do the following:
- View the total number of users that are managed from ZIdentity. You can see:
- **IdP Users**: The number of registered and guest users accessing ZIdentity through an [external IdP](/zidentity/about-external-identity-providers).
- **Hosted Users**: The number of registered and guest users hosted on ZIdentity.
- **Service Entitlements**: The number of users assigned to different Zscaler services. To learn more, see [About Service Entitlements](/zidentity/about-service-entitlements).
- Select the duration for which you want to view the data. The minimum is 1 day and the maximum is 90 days.
- View the authentication status (Success or Failure) of hosted users and external IdP users. The graph depicts the total number of users that accessed the ZIdentity Admin Portal along with the date and time of authentication. You can see the number of successful and failed authentications.
- View the statistics for successful user management activities (Created, Updated, and Deleted) on a specific date and time.
- View the statistics for failed user management activities (Created, Updated, and Deleted) on a specific date and time.
