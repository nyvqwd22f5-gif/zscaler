# Assigning CXO Insight User Role
Source: https://help.zscaler.com/zidentity/assigning-cxo-insight-user-role
PDF: https://help.zscaler.com/pdf/com/en/1509541.pdf

The CXO Insight User role enables executive users to access the [Executive Insights mobile application](/zia/accessing-and-using-executive-insights-app). The executive users are also assigned to a specific Zscaler service in ZIdentity, which allows them to access the Zscaler service (i.e., Zscaler Internet Access (ZIA)) in the mobile app.
Prerequisites
Make sure the following prerequisites are met:
Before assigning the role in the ZIdentity Admin Portal, you might need to authorize the executive users to access ZIA on the Executive Insights App. To learn more, see [Adding Executive Insights App Admins](/zia/adding-executive-insights-app-admins).
Assigning CXO Insight User Role for ZIdentity
To assign the CXO Insight User role to users for ZIdentity:
- Go to **Administration** > **Entitlements** > **Administrative**.
-
On the **Administrative Entitlements** page, select **ZIdentity Administration**.
[See image.](#zid-administration)
- You can assign the role to individual users or user groups:
- [Assign a Role to a User Group](#zidentity-assign-groups)
- [Assign a Role to an Individual User](#zidentity-assign-users)
Assigning Executive Insights App Role to Access ZIA Tenants
Executive users must be assigned the Executive Insights App role on all the ZIA tenants that they need to access.
To assign the Executive Insights App role to executive users:
- Go to **Administration** > **Entitlements** > **Administrative**.
-
On the **Administrative Entitlements** page, select** Zscaler Internet Access**.
[See image.](#Select-zia)
- You can assign the role to individual users or user groups:
- [Assign a Role to a User Group](#zia-assign-groups-section)
- [Assign a Role to an Individual User](#zia-assign-users-section)
[]
-
On the **User Groups** tab, click **Assign Groups**.
[See image.](#zid-assign-groups)
The** Assign Groups **page appears.
- On the **Select Groups & Roles** page, select the user groups.
-
In the **Role **column, select the **CXO Insight User **role from the drop-down menu.
[See image.](#zid-assign-role)
- (Optional) Enable **Set same role for all selected groups **option if you want to assign the same role to all selected groups.
- Click **Next**.
-
On the **Summary **page, review the assignment details and click **Assign**.
[See image.](#zid-groups-summary)
The **CXO Insight User** role is assigned to the selected user groups.
[]
-
On the **Users **tab, click **Assign Users**.
[See image.](#zid-assign-users)
The** Assign Users **page appears.
- On the **Select Groups & Roles** page, select the users.
-
In the **Role **column, select the **CXO Insight User **role from the drop-down menu.
[See image.](#assign-user-role)
- (Optional) Enable **Set same role for all selected users **option if you want to assign the same role to all selected users.
- Click **Next**.
-
On the **Summary **page, review the assignment details and click **Assign**.
[See image.](#zid-users-summary)
The **CXO Insight User** role is assigned to the selected users.
[]
-
On the **User Groups** tab, click **Assign Groups**.
[See image.](#zia-assign-groups-button)
The** Assign Groups **page appears.
- On the **Select Groups & Roles** page, select the user groups.
-
In the **Role **column, select the **Executive Insights App **role from the drop-down menu.
[See image.](#zia-assign-role-to-group)
- (Optional) Enable **Set same role for all selected groups **option if you want to assign the same role to all selected groups.
- Click **Next**.
-
On the **Summary **page, review the assignment details and click **Assign**.
[See image.](#zia-groups-summary)
The **Executive Insights App** role is assigned to the selected user groups.
[]
-
On the **Users **tab, click **Assign Users**.
[See image.](#zia-assign-users-button)
The** Assign Users **page appears.
- On the **Select Groups & Roles** page, select the users.
-
In the **Role **column, select the **Executive Insights App **role from the drop-down menu.
[See image.](#zia-assign-roles-to-users)
- (Optional) Enable **Set same role for all selected users **option if you want to assign the same role to all selected users.
- Click **Next**.
-
On the **Summary **page, review the assignment details and click **Assign**.
[See image.](#zia-users-summary)
The **Executive Insights App** role is assigned to the selected users.
