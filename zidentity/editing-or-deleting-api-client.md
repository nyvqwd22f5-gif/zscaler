# Editing or Deleting an API Client
Source: https://help.zscaler.com/zidentity/editing-or-deleting-api-client
PDF: https://help.zscaler.com/pdf/com/en/1499446.pdf

You can edit or delete the API client as necessary.
Editing an API Client
To edit an API client:
- Go to **Integration** >** API Clients**.
-
On the **API Clients** page, click the **Edit** icon for the required API client.
[See image.](#oapi-editicon)
-
Edit the information in the **Client Information** and **Client Authentication** sections, or select the **Resources** tab to select a different role as necessary.
After you assign a different role, the access token issued by ZIdentity is applicable to the new scope for that Zscaler service.
[See image.](#opai-editwin)
- After you've changed the values, click **Update**.
A message is displayed indicating the API client is updated successfully.
Deleting an API Client
To delete an API client:
- Go to **Integration** >** API Clients**.
-
Click the **Delete** icon for the required API client.
[See image.](#opai-deleteicon)
-
Read the confirmation message, type **YES**, then click **Delete**.
[See image.](#opai-deletemsg)
