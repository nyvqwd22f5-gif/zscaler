# Accessing and Navigating the ZIdentity Admin Portal
Source: https://help.zscaler.com/zidentity/accessing-and-navigating-zidentity-admin-portal
PDF: https://help.zscaler.com/pdf/com/en/1499051.pdf

This article covers the following topics:
- [Accessing the ZIdentity Admin Portal](#accessanagementportal)
- [Navigating within the ZIdentity Admin Portal](#navigating-portal)
When you're ready to begin configuring ZIdentity, see the [Step-by-Step Configuration Guide for ZIdentity](/zidentity/step-step-configuration-guide-zidentity).
[]
- [Log in to the ZIdentity Landing Page](/zidentity/accessing-and-navigating-zidentity-landing-page).
- Click the **ZIdentity** **Administration** tile. You are redirected to the ZIdentity Admin Portal.
[See image.](#zsl-landingpage)
[]![ZIdentity Landing Page with different Zscaler services in tiles and annotation around the ZIdentity Administration tile. ](/downloads/zidentity/getting-started/admin-portal/accessing-and-navigating-zidentity-landing-page/Zid-landing%20page1.png)
[]The left-side navigation has the following menu items:
- [Dashboard](#dashboard)
- [Directory](#directory)
- [Integration](#integration)
- [Policy](#policy)
- [Administration](#Admin)
- [My Profile](#my-profile)
- [Remote Assistance](#remote-assistance)
- [Logout](#log-out)
[See image.](#zsl-adminportalpage)
[]![Viewing My Profile](/downloads/zslogin/getting-started/admin-portal/accessing-and-navigating-zslogin-admin-portal/ZIdentity-myprofile.png)
[]Click **Policy** to configure different polices for the ZIdentity users. To learn more, see the following articles:
- [Configuring Sign-On Policy](/zidentity/configuring-sign-on-policy)
- [Configuring the Password Policy](/zidentity/configuring-password-policy)
[]Click **Administration **to manage your authentication methods, locations, Zscaler services, ZIdentity roles, and audit logs. To learn more, see the following articles:
- [About IP Locations](/zidentity/about-ip-locations)
- [About IP Location Groups](/zidentity/about-ip-location-groups)
- [About Linked Services](/zidentity/about-linked-services)
- [Configuring Authentication Methods](/zidentity/configuring-authentication-methods)
- [Managing Device Token Authentication](/zidentity/managing-device-token-authentication)
- [About Administrative Entitlements](/zidentity/about-administrative-entitlements)
- [About Service Entitlements](/zidentity/about-service-entitlements)
- [About Audit Logs](/zidentity/about-audit-logs)
- [About ZIdentity Admin Roles](/zidentity/about-zidentity-admin-roles)
[]Click **My Profile** to view and manage your profile information. To learn more, see [Managing User Profile](/zidentity/managing-user-profile).
[]Click **Logout** to log out of the ZIdentity Admin Portal when you are done.
[]Click **Dashboard **to view a high-level overview of all users that are managed from ZIdentity. To learn more, see [About the Dashboard](/zidentity/about-dashboard).
[]Click **Directory **to manage your users, attributes, and departments. To learn more, see the following articles:
- [About Users](/zidentity/about-users)
- [About User Groups](/zidentity/about-user-groups)
- [About Attributes](/zidentity/about-attributes)
- [About Departments](/zidentity/about-departments)
[]Click **Integration **to manage identity providers. To learn more, see [About External Identity Providers](/zidentity/about-external-identity-providers).
[]Click **Remote Assistance** to manage the access Zscaler Support has to a customer's portal. To learn more, see [Enabling Remote Assistance](/zidentity/enabling-remote-assistance).
