# Editing or Deleting Items
Source: https://help.zscaler.com/zidentity/editing-or-deleting-items
PDF: https://help.zscaler.com/pdf/com/en/1499061.pdf

In the ZIdentity Admin Portal, you can often edit or delete items you've configured. For example, you can do so with policy rules and other administrative items you've configured in the ZIdentity Admin Portal (e.g., users, role, groups, IP locations, etc.).
Editing Items
To edit an item:
- Go to the item you want to edit. In this example, it is a user group.
-
Click the **Edit** icon.
[See image.](#zid-edititem)
- Edit the required values, then click **Update**.
Deleting Items
[Deleting a Single Item](#Deleting-Single-Item)
[Deleting Multiple Items](#Deleting-multiple-items)
[]
- Go to the item you want to delete. In this example, it is a user group.
-
Click the **Delete** icon.
[See image.](#zid-deleteitem)
-
Read the warning message that appears and type `YES` to confirm.
[See image.](#zid-confirmdelete)
- Click **Delete**.
[]
- Go to the item you want to delete. In this example, it is a user group.
-
Select the checkbox for more than one user group. The **Actions **dropdown appears.
[See image.](#selecting-usergroups)
- Click the **Actions **dropdown.
-
Click **Delete**.
[See image.](#clicking-delete)
-
Read the warning message that appears and type `YES` to confirm.
[See image.](#confirming-bulk-delete)
-
Click **Delete**.
