# About Settings
Source: https://help.zscaler.com/itdr/about-settings
PDF: https://help.zscaler.com/pdf/com/en/1531707.pdf

The Settings menu allows you to view and configure audit logs, safelists, advanced settings, users, roles, and identity providers (IdPs) for single sign-on (SSO).
The Settings menu provides the following benefits and enables you to:
- Configure users, roles, SSO, support users, and user-access settings.
- Access audit logs, system messages, and logs.
- Use advanced options to configure maintenance, retention, and debugging.
About the Settings Menu
On the Settings menu, you can do the following:
- Find options to view, create, or manage [users](/itdr/about-users-roles#itdr-user-role-about-user), [roles](/itdr/about-users-roles#itdr-user-role-about-role), [IdPs for SSO](/itdr/about-identity-providers-single-sign) (IdPs), and [Support team users](/itdr/about-users-roles#itdr-view-support-user-details).
- Find options to view [endpoint agents](/itdr/about-endpoint-agents), [agent configuration](/itdr/about-agent-configuration), [agent update groups](/itdr/about-agent-update-groups), and [safe processes](/itdr/about-safe-processes).
- Find options to view, create, and manage [server agent domains](/itdr/about-server-agent-domains), [server agents](/itdr/about-server-agents), [server agent configuration](/itdr/about-server-agent-configuration), and [server agent update groups](/itdr/about-server-agent-update-groups).
- Find options to [export audit logs](/itdr/exporting-audit-logs), [view and manage system messages](/itdr/viewing-and-managing-system-messages), and [download debug logs](/itdr/downloading-debug-logs).
- Go to [Advanced Settings](/deception/about-advanced-settings) to view license details, manage events, and more.
