# Editing or Deleting an API Token
Source: https://help.zscaler.com/itdr/editing-or-deleting-api-token
PDF: https://help.zscaler.com/pdf/com/en/1531693.pdf

You can edit or delete an API token that you configured in the Zscaler ITDR Admin Portal.
Editing an API Token
To edit an API token:
- Go to **Orchestrate** > **API Tokens**.
-
Locate the API token you want to modify, and under **Actions**, click the **Edit** icon.
[See image.](#itdr-edit-api-token-option)
-
In the **API Token Details **window, modify the required fields.
[See image.](#itdr-edit-api-token-details)
You cannot edit the API token expiration period.
- Click **Submit**.
Deleting an API Token
To delete an API token:
- Go to **Orchestrate** > **API Tokens**.
-
Locate the API token you want to delete, and under **Actions**, click the **Delete** icon.
[See image.](#itdr-delete-api-icon)
- In the confirmation window, click **OK**.
