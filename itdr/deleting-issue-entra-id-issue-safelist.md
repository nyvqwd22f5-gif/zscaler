# Deleting an Issue from the Entra ID Issue Safelist
Source: https://help.zscaler.com/itdr/deleting-issue-entra-id-issue-safelist
PDF: https://help.zscaler.com/pdf/com/en/1531765.pdf

You can remove an issue from the [Entra ID Issue Safelist](/itdr/about-entra-id-issue-safelist).
To remove the issue from Entra ID Issue Safelist:
- Go to **ITDR **> **Settings **> **Entra ID Issue Safelist**.
- Select an Entra ID tenant from the **Result for** drop-down menu.
-
Under **Actions**, click the **Delete** icon to remove the issue from the safelist.
[See image.](#entra-id-posture-safelist)
-
In the confirmation window, click **OK**.
The issue is added to the [Focus Area](/itdr/viewing-entra-id-focus-area-details) page and is included in the new and previous [vulnerability reports](/itdr/viewing-entra-id-vulnerability-report) for the tenant.
