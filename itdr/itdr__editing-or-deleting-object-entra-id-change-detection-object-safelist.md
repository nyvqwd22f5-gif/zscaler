# Editing or Deleting an Object from the Entra ID Change Detection Object Safelist
Source: https://help.zscaler.com/itdr/editing-or-deleting-object-entra-id-change-detection-object-safelist
PDF: https://help.zscaler.com/pdf/com/en/1531798.pdf

You can edit or delete an Entra ID change detection object that is marked safe from the [Entra ID Change Detection Object Safelist](/itdr/about-entra-id-change-detection-object-safelist).
Editing an Entra ID Change Detection Object from the Safelist
To edit an Entra ID change detection object from the safelist:
- Go to **ITDR** > **Settings** > **Entra ID** **Change Detection Object Safelist**.
- Select a tenant from the **Result for** drop-down menu.
-
Locate the change detection object you want to edit, and under **Actions**, click the **Edit **icon.
[See image.](#edit-icon-object)
-
Make changes to the **Reason **or the **Expiry **field as required.
[See image.](#edit-window-object)
- Click **Save**.
Deleting an Entra ID Change Detection Object from the Safelist
To delete an Entra ID change detection object from the safelist:
- Go to **ITDR** > **Settings** > **Entra ID** **Change Detection Object Safelist**.
- Select a tenant from the **Result for** drop-down menu.
-
Locate the change detection object you want to remove, and under **Actions**, click the **Delete **icon.
[See image.](#itdr-object-delete-safelist)
-
In the confirmation window, click **OK**.
When you remove an object from the safelist, it gets added to the [Entra ID Change Detection dashboard](/itdr/about-entra-id-change-detection-dashboard).
