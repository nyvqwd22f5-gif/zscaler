# Deleting an Issue from the Active Directory Issue Safelist
Source: https://help.zscaler.com/itdr/deleting-issue-active-directory-issue-safelist
PDF: https://help.zscaler.com/pdf/com/en/1531639.pdf

You can remove an issue from the Active Directory Issue Safelist page.
To remove the issue from the Active Directory Issue Safelist:
- Go to **Settings** > **ITDR Safelist **> **Active Directory Issue Safelist**.
- Select an AD domain from the **Result for** drop-down menu.
-
Under **Actions**, click the **Delete** icon to remove the issue from the safelist.
[See image.](#deception-itdr-issue-safelist-delete)
-
In the confirmation window, click **OK**.
The issue is added to the [Focus Area](/itdr/viewing-focus-area-details) page and the new and previous [vulnerability reports](/itdr/viewing-vulnerability-report) for the selected AD domain.
