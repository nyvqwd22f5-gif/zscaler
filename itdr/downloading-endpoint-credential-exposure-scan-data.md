# Downloading Endpoint Credential Exposure Scan Data
Source: https://help.zscaler.com/itdr/downloading-endpoint-credential-exposure-scan-data
PDF: https://help.zscaler.com/pdf/com/en/1531724.pdf

You can download the endpoint credential exposure scan data for the endpoints as a .zip file containing CSV files to your local system. You can extract the .zip file to view the details of each endpoint and their [exposure type](/itdr/viewing-endpoint-credential-exposure-issues) in individual CSV files.
To download the endpoint credential exposure scan data:
- Go to **ITDR** > **Dashboard** > **Endpoint Credential Exposure**.
-
Select a date from the **As of Date** calendar.
The exposed credential details for the selected date appears.
[See image.](#as-of-date)
-
Click **Download CSV Data**.
[See image.](#ece-report)
A .zip file containing multiple CSV files is downloaded to your system. The CSV files for each exposure type contain the list of endpoints and the exposed credential details.
[See image.](#csvfiles)
