# Viewing Extended Details
Source: https://help.zscaler.com/itdr/viewing-extended-details
PDF: https://help.zscaler.com/pdf/com/en/1531831.pdf

The extended details page shows the in-depth analysis of attack activities. On this page, you can view:
- [ThreatParse](/itdr/viewing-threatparse-details) that consists of easy-to-understand attack details.
- [Attack chronology](/itdr/viewing-attack-chronology-details) that displays a temporal overview and heatmap of the attackerâ€™s activities.
- [Event logs](/itdr/about-event-logs) that show details of alerts created by an attacker.
To view the extended details page:
-
On the **Investigate** page, click an element in the alert graph. For example, click an attacker icon or a Decoy Connector.
The details pane opens.
-
In the details pane, click **View Extended Details**.
[See image.](#extended-details-button)
The **Extended Details** window appears.
-
To view in-depth details, click one of the tabs:
- **ThreatParse**: Provides information about the attack-based MITRE ATT&CK  framework along with a risk score in plain English. To learn more, see [Viewing the ThreatParse Details](/itdr/viewing-threatparse-details).
- **Chronology**: Provides a temporal overview and heatmap of the attacker's activities. To learn more, see [Viewing the Attack Chronology Details](/itdr/viewing-attack-chronology-details).
- **Event Logs**: Provides detailed log information for the events associated with an activity. To learn more, see [About Event Logs](/itdr/about-event-logs).
