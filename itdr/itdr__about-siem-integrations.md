# About SIEM Integrations
Source: https://help.zscaler.com/itdr/about-siem-integrations
PDF: https://help.zscaler.com/pdf/com/en/1531722.pdf

Zscaler ITDR seamlessly integrates with leading security information and event management (SIEM) solutions to enhance your security operations workflows. You can integrate ITDR with supported SIEM solutions to transmit event logs in real time. SIEM integration provides visibility in a centralized console and allows your teams to leverage the solutionâ€™s existing security investigation workflows.
Integrations with SIEM solutions provide the following benefits and enable you to:
- Transmit event logs generated by ITDR to SIEM tools for real-time analysis of the log data in a centralized view.
- Monitor your network to gain visibility into suspicious activities, potential threats, and attacks, and respond to security incidents efficiently.
- Ensure that your organization meets auditing requirements and regulatory compliance standards.
About the SIEM Integrations Page
On the SIEM Integrations page (Orchestrate > SIEM Integrations), you can do the following:
- View a list of all configured SIEM integrations. For each SIEM integration, you can see:
- **Name**: The name of the SIEM integration.
- **Type**: The name of the SIEM solution.
- **Connector**: The Service Connector that is configured to forward logs to the SIEM.
- **Source**: The type of log (e.g., **Events** or **Audit Logs**) that is forwarded to the SIEM.
- **Last Failure**: The last error message, if any.
- **Enabled**: The **X** icon indicates that the SIEM integration is disabled, and the check mark icon indicates that the SIEM integration is enabled.
- Add and configure SIEM integrations on the following supported platforms:
- [ArcSight](/itdr/siem-configuration-guide-arcsight-enterprise-security-manager)
- [IBM QRadar](/itdr/siem-configuration-guide-ibm-qradar)
- [Microsoft Azure Sentinel](/itdr/siem-configuration-guide-microsoft-azure-sentinel)
- [Netmonastery DNIF](/itdr/siem-configuration-guide-netmonastery-dnif)
- [Splunk](/itdr/siem-configuration-guide-splunk-enterprise-and-cloud-platform)
- [Syslog](/itdr/siem-configuration-guide-syslog)
- [Sumo Logic](/itdr/siem-configuration-guide-sumo-logic)
- [Edit or delete](/itdr/editing-or-deleting-siem-integration) a SIEM integration.
