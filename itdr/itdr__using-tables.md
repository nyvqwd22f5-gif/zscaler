# Using Tables
Source: https://help.zscaler.com/itdr/using-tables
PDF: https://help.zscaler.com/pdf/com/en/1531845.pdf

Data is organized and displayed in tables on some of the pages (e.g., Detailed Findings and Recommendations, Change Detection, Scan Reports, etc.) in the Zscaler ITDR Admin Portal. You can perform various actions in a table and modify table settings as necessary.
You can do the following:
- [Export table data](#itdr-export-table-csv-section)
- [Copy columns](#itdr-copy-columns-table-section)
- [Resize columns](#itdr-resize-columns-table-section)
- [Search for specific information](#itdr-filter-data-table)
[]You can export or download data from a table as a CSV or JSON file.
Select **Export as CSV**, **Download as CSV**,** or Download as JSON** from the **Actions** drop-down menu to download data from the table to your system.
[See image.](#itdr-export-table-as-csv-image)
[]You can copy specific columns from tables (up to 1,000 rows) and paste them into CSV, JSON, Notepad, or other compatible files.
To copy columns:
-
Click **Copy Table **from the **Actions** drop-down menu.
[See image.](#itdr-copy-table-option-image)
-
In the **Select Columns To Copy** window, select the columns that you want to copy.
[See image.](#itdr-copy-table-select-columns-image)
-
Click **Copy**.
The selected columns are copied to the clipboard. You can paste the copied data into any compatible file.
[]When the data in a column takes up more space than the viewing area allows, it gets clipped.
To increase the column width:
-
Hover over the right border of the column.
A **Resize** icon appears.
-
Drag and drop the icon to the desired column width.
[See image.](#itdr-copy-table-resize-columns-image)
[]You can use the search icon to filter and view specific data.
To search and view specific data:
- Select a column.
-
In the column header, click inside the field, and enter the text you want to search.
[See image.](#itdr-search-data-table)
