# Applying the AD Threat Detection Filter
Source: https://help.zscaler.com/itdr/applying-ad-threat-detection-filter
PDF: https://help.zscaler.com/pdf/com/en/1531795.pdf

You can apply the Active Directory (AD) Threat Detection filter to view events specific to AD Threat Detection on the Investigate dashboard.
To apply the Active Directory Threat Detection filter, click **AD Threat Detection** from the **ITDR **menu.
[See image.](#ad-threat-detection-filter)
You are redirected to the Investigate dashboard with a [filter applied](/itdr/understanding-ad-threat-detection-filter) for viewing events specific to AD Threat Detection.
