# Deleting an Issue from the Entra ID Change Detection Issue Safelist
Source: https://help.zscaler.com/itdr/deleting-issue-entra-id-change-detection-issue-safelist
PDF: https://help.zscaler.com/pdf/com/en/1531770.pdf

You can delete an Entra ID change detection issue that is marked safe from the [Entra ID Change Detection Issue Safelist](/itdr/about-entra-id-change-detection-safelist).
To delete an Entra ID change detection issue from the safelist:
- Go to **ITDR** > **Settings** > **Entra ID** **Change Detection Issue Safelist**.
- Select a tenant from the **Result for** drop-down menu.
-
Under **Actions**, click the **Delete** icon to remove the object from the safelist.
[See image.](#itdr-object-delete-safelist)
-
In the confirmation window, click **OK**.
When you remove an issue from the safelist, it gets added to the [Entra ID Change Detection dashboard](/itdr/about-entra-id-change-detection-dashboard).
