# Editing or Deleting an OpenID IdP Configuration
Source: https://help.zscaler.com/itdr/editing-or-deleting-openid-idp-configuration
PDF: https://help.zscaler.com/pdf/com/en/1531869.pdf

You can edit or delete an OpenID identity provider (IdP) configuration in the Zscaler ITDR Admin Portal.
Editing an OpenID IdP Configuration
To edit an OpenID IdP configuration:
- Go to **Settings** > **Users & Roles** > **IdP Providers**.
-
In the table, locate an OpenID IdP configuration you want to modify, and then click the **Edit** icon.
[See image.](#itdr-edit-openid-idp)
-
In the **OpenID Provider Details** window, modify the required fields.
[See image.](#itdr-modify-openid-idp-details)
- Click **Save**.
Deleting an OpenID IdP Configuration
To delete an OpenID IdP Configuration:
- Go to **Settings** > **Users & Roles** > **IdP Providers**.
-
In the table, locate an OpenID IdP configuration you want to delete, and then click the **Delete** icon.
[See image.](#itdr-openid-delete-idp)
- In the confirmation window, click **OK**.
