# Managing Agent Update Groups
Source: https://help.zscaler.com/itdr/managing-agent-update-groups
PDF: https://help.zscaler.com/pdf/com/en/1531658.pdf

You can manage agent update groups using the following options:
- [Enabling Phased Updates](#enabling-phased-updates)
- [Enabling or Disabling Individual Update](#enabling-or-disabling-updates)
- [Downloading Status Details of Agent Update Groups](#Downloading-Status)
- [Editing an Agent Update Group](#edit-agent)
- [Exporting Hostnames](#export-agent)
- [Deleting an Agent Update Group](#delete-agent)
[]
You can enable phased updates for the endpoint agents to selectively roll out the updates using agent update groups.
To enable phased updates:
- Go to **Settings **> **Endpoint Settings **> **Agent Update Groups**.
-
Click **Enable Phased Updates**.
[See image.](#itdr-phased-updates-option)
-
In the confirmation window, click **OK**.
[See image.](#itdr-confirmation-phased-updates)
Enabling this feature stops any automatic updates for endpoint agents that are not a part of the groups.
[]
To enable or disable updates for a specific agent update group:
- Go to **Settings **> **Endpoint Settings **> **Agent Update Groups**.
-
Locate the agent update group for which you want to enable or disable updates. Under the **Actions **column, click the **X **icon to disable phased updates, or click the **Checkmark **icon to enable phased updates.
[See image.](#itdr-enable-disable-icons)
You can see the status of each group and how many endpoints have been updated in the table.
[]
You can download the update status details of an agent update group as a CSV file.
To download update status details:
- Go to **Settings **> **Endpoint Settings **> **Agent Update Groups**.
-
Locate the agent update group for which you want to download the update details, and click the **Download **icon under the **Actions **column.
[See image.](#itdr-download-icon-agent-group)
A CSV file with the agent group update status is downloaded to your system.
[See image.](#itdr-csv-file-agent-update-group)
[]
To edit an agent update group:
- Go to **Settings **> **Endpoint Settings **> **Agent Update Groups**.
-
Locate the agent update group that you want to edit, and click the **Edit **icon under the **Actions **column.
[See image.](#itdr-edit-agent-group)
-
In the **Agent Update Group Details** window, make the necessary changes.
[See image.](#itdr-add-group-window)
- Click **Save**.
[]
You can export the list of hostnames associated with an agent update group as a TXT file.
To export hostnames from an agent update group:
- Go to **Settings **> **Endpoint Settings **> **Agent Update Groups**.
-
Locate the agent update group that you want to export, and click the **Edit **icon under the **Actions **column.
[See image.](#itdr-edit-agent-group-icon)
-
In the **Agent Update Group Details** window, click **Export hostnames**.
[See image.](#itdr-export-hostnames-agent-group)
A TXT file containing the list of hostnames is downloaded to your system.
[]
To delete an agent update group:
- Go to **Settings **> **Endpoint Settings **> **Agent Update Groups**.
-
Locate the agent update group that you want to delete, and click the **Delete **icon under the **Actions **column.
[See image.](#itdr-delete-agent-update-groups)
-
In the confirmation window, click **OK**.
The agent update group is deleted.
