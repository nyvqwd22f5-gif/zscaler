# Editing or Deleting an Object from the AD Change Detection Object Safelist
Source: https://help.zscaler.com/itdr/editing-or-deleting-object-ad-change-detection-object-safelist
PDF: https://help.zscaler.com/pdf/com/en/1531801.pdf

You can edit or delete an Active Directory (AD) change detection object that is marked safe from the [AD Change Detection Object Safelist](/itdr/about-ad-change-detection-object-safelist).
Editing an AD Change Detection Object from the Safelist
To edit an AD change detection object from the safelist:
- Go to **ITDR** > **Settings** > **AD** **Change Detection Object Safelist**.
- Select a tenant from the **Result for** drop-down menu.
-
Locate the change detection object you want to edit, and under **Actions**, click the **Edit **icon.
[See image.](#edit-icon-object)
-
Make changes to the **Reason **or the **Expiry **field as required.
[See image.](#edit-window-object)
- Click **Save**.
Deleting an AD Change Detection Object from the Safelist
To delete an AD change detection object from the safelist:
- Go to **ITDR** > **Settings** > **AD** **Change Detection Object Safelist**.
- Select a tenant from the **Result for** drop-down menu.
-
Locate the change detection object you want to remove, and under **Actions**, click the **Delete **icon.
[See image.](#itdr-object-delete-safelist)
-
In the confirmation window, click **OK**.
When you remove an object from the safelist, it gets added to the [AD Change Detection dashboard](/itdr/about-change-detection-dashboard).
