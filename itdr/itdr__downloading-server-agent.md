# Downloading a Server Agent
Source: https://help.zscaler.com/itdr/downloading-server-agent
PDF: https://help.zscaler.com/pdf/com/en/1531858.pdf

Server agents are installed on domain controllers and continuously monitor Active Directory activity for identity threat detection. You can download the server agent directly from the ITDR Admin Portal.
To download a server agent:
- Go to **Settings **> **Server Agent Settings **> **Agents**.
-
Click **Download Agent**.
[See image.](#deception-download-landmine-agent-installer-option)
-
In the **Download Agent** window, choose an OS installation type:
- **Windows (EXE)**
- **Windows (MSI)**
[See image.](#deception-download-landmine-agent-installer)
-
Click **Download Installer**.
The latest version of the server agent is downloaded to your system. You can [install a server agent on a domain controller](/itdr/installing-server-agent).
