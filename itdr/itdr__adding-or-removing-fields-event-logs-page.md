# Adding or Removing Fields On the Event Logs Page
Source: https://help.zscaler.com/itdr/adding-or-removing-fields-event-logs-page
PDF: https://help.zscaler.com/pdf/com/en/1531827.pdf

You can add or remove fields on the Event Logs page. You can add a maximum of 50 fields.
To add or remove fields:
-
On the **Event Logs** page, click the **Expand** icon.
[See image.](#event-logs-icon)
-
In the panel that lists the fields, select or unselect fields to add or remove columns in the **Event Logs** table.
[See image.](#column-chooser)
After you modify the table, you can [export](/itdr/exporting-event-logs) the event logs.
