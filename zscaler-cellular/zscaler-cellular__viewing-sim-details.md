# Viewing SIM Details
Source: https://help.zscaler.com/zscaler-cellular/viewing-sim-details
PDF: https://help.zscaler.com/pdf/com/en/1518126.pdf

You can access in-depth information about each SIM—both physical SIMs and eSIMs—provisioned to your organization. The SIM details page provides a comprehensive overview of a selected SIM's session activity, including its current status, historical data usage, and key connectivity details.
To view the details of a SIM card:
-
In the left-side navigation, select **SIMs**.
The **SIMs **page appears.
-
In the table, click the **ICCID **link for the SIM you are interested in.
The **SIM details **page opens.
-
On the **SIM details **page, you can view:
- **SIM Info**: Displays the basic details about the SIM:
- **Connection**: Indicates whether the SIM is online or offline.
- **APN**: The Access Point Name used for the SIM's network connectivity.
- **ICCID**: The Integrated Circuit Card Identifier associated with the SIM card.
- **MSISDN**: The Mobile Station International Subscriber Directory Number is the associated mobile number of the SIM, if applicable.
- **Activation Date**: The date and time the SIM was activated.
- **Country**: The location where the SIM card is located.
- **Last Usage Updated At**: The time when the last change was recorded.
- **EID**: The Embedded Identity Document for eSIMs.
- **Operator**: The telecom operator providing the network service.
- **IP Address**: The IP address currently assigned to the SIM.
- **IMEI**: The International Mobile Equipment Identity, the unique identifier for the device using the SIM.
- **IMSI**: The International Mobile Subscriber Identity associated with the SIM.
- **Tags**: The tags associated with the SIM card. You can click a specific tag to filter the SIMs page to display only the SIMs that have the tag associated with them.
-
**Device Info**: Displays the basic details about the device associated with the SIM:
- **Device Manufacturer**: The name of the company that produces the device associated with the SIM card.
- **Device Type**: The category of the device associated with the SIM card (e.g., router, modem, and IoT gateway).
- **Operating System**: The operating system running on the device associated with the SIM card.
- **Device Model**: The full official identifier of the type of device associated with the SIM card.
- **Model Name**: The specific code of the type of device associated with the SIM card.
- **Band Details**: The frequency bands supported by the device associated with the SIM card.
[See image.](#basic-information)
-
**Current Data Usage**: A dynamic widget showing the cumulative data usage for the date range selected in the **Usage Chart**.
[See image.](#current-usage-data)
-
**E-SIM Info**: A widget showing to whom an eSIM was assigned along with options for managing it. This widget appears only for eSIMs.
[See image.](#e-sim-info)
- **Usage Chart**: A widget that provides a detailed visual representation of data usage over time and allows you to identify usage trends, peak usage periods, and potential anomalies. The chart includes the following customization options:
-
**Date Range**: Select a default date range (**Today**, **Yesterday**, **Last 7 days**, **Last 30 Days**, **This Month**, or **Last Month**) or choose **Custom Range **and select start and end dates to display usage details for the specific period.
[See image.](#usage-chart-date-range)
-
Option to download the usage details in different formats (**SVG**, **PNG**, or **CSV**).
[See image.](#usage-chart-download-option)

Option to view network events associated with the SIM.
