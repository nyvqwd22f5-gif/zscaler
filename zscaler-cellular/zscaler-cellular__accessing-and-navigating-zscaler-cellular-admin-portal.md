# Accessing and Navigating the Zscaler Cellular Admin Portal
Source: https://help.zscaler.com/zscaler-cellular/accessing-and-navigating-zscaler-cellular-admin-portal
PDF: https://help.zscaler.com/pdf/com/en/1518141.pdf

The Zscaler Cellular Admin Portal allows you to manage Zscaler SIMs provisioned to your organization. This article covers the following topics on how to access and navigate the Zscaler Cellular Admin Portal.
- [Signing in to the Zscaler Cellular Admin Portal](#signing-in-to-the-portal)
- [Accepting the End User Subscription Agreement (EUSA)](#accepting-eusa)
- [Navigating within the Zscaler Cellular Admin Portal](#navigating-within-the-portal)
[]When a Zscaler Cellular service is enabled for your organization, a tenant is provisioned in Zscaler Internet Access (ZIA), Zscaler Private Access (ZPA), Branch Connector, Zscaler Cellular, and ZIdentity. You can access the Zscaler Cellular Admin Portal via ZIdentity. Zscaler sends you an email with details such as the URL to access and set up ZIdentity. To learn more, see the [Step-by-Step Configuration Guide for ZIdentity](/zidentity/step-step-configuration-guide-zidentity).
After completing the ZIdentity setup, follow the steps below:
- [Log in to ZIdentity](/zidentity/accessing-and-navigating-zidentity-landing-page).
- On the ZIdentity Landing Page, click the **Cellular **tile.
[]When you log in to the Zscaler Cellular Admin Portal for the first time, you need to accept the End User Subscription Agreement (EUSA). To learn more, see [Accepting the End User Subscription Agreement (EUSA)](/zscaler-cellular/accepting-end-user-subscription-agreement-eusa).
[]The Zscaler Cellular Admin Portal has the following items in the left-side navigation:
- [Dashboard](#dashboard)
- [Deployment Regions](#deployment)
- [Configuration](#config)
- [SIMs](#sims)
- [Account Details](#customers)
- [Network Events](#network-events)
- [Login Tracker](#logins)
- [Support Ticket](#support-ticket-page)
[See image.](#admin-portal-full-view)
[]The Zscaler Cellular Dashboard provides a centralized view of your organization's SIM card usage, network activity, and connectivity metrics. It offers a graphical representation of key metrics and real-time insights into your deployed SIM cards and their connection to the Zscaler cloud.
To learn more, see [Understanding the Zscaler Cellular Dashboard](/zscaler-cellular/understanding-zscaler-cellular-dashboard).
[]You can monitor the deployment and service status or request additional deployment of Cellular Edges directly from the Zscaler Cellular Admin Portal.
To learn more, see [Viewing and Monitoring Cellular Edge Deployments](/zscaler-cellular/viewing-and-monitoring-cellular-edge-deployments).
[]You can view the Branch Connector credentials and update them to be able to deploy Cellular Edges in different regions.
To learn more, see [Configuring and Updating Credentials for Cellular Edge Deployment](/zscaler-cellular/configuring-and-updating-credentials-cellular-edge-deployment).
[]The SIMs page allows you to view essential details and statuses, and manage your Zscaler SIMs.
To learn more, see [About SIMs](/zscaler-cellular/about-sims).
[]From the Account Details page, you can view your account details and configure remote access for Zscaler Support.
To learn more, see [Viewing Account Details](/zscaler-cellular/viewing-account-details).
[]The Network Events page offers a centralized interface for monitoring and analyzing network activity across all Zscaler SIMs provisioned to your organization. It provides detailed insights into network events for each SIM, capturing critical connectivity information such as SIM transitions to online or offline states, as well as authorization events.
To learn more, see [About Network Events](/zscaler-cellular/about-network-events).
[]The Login Tracker allows you to gain insights into user authentication activities. It serves as a vital tool for monitoring login events, offering administrators a streamlined method to track user access patterns and investigate login-related events.
To learn more, see [Viewing Login Activity](/zscaler-cellular/viewing-login-activity).
[]Click Support Ticket to quickly access the Zscaler Support Portal, where you can submit new support requests, track the status of existing tickets, communicate with the support team, and find information, guides, and announcements. This feature ensures seamless assistance for resolving technical issues, addressing inquiries, and managing your organization's support interactions effectively.
