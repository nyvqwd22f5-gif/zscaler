# Cyera Assets
Source: https://help.zscaler.com/uvm/cyera-assets
PDF: https://help.zscaler.com/pdf/com/en/1528301.pdf

![The Cyera Assets tile](/downloads/uvm/configure/sources/cyera-assets/mceclip0.png)
Cyera is a Data Security Posture Management tool that uses native APIs to discover, classify, and secure sensitive data across IaaS, PaaS, SaaS, and on-premise environments. The Cyera Assets connector retrieves open and closed issues.
Required Parameters
- Client ID
- Client Secret
Generating a Client ID and Secret
- Log in to the Cyera platform.
- From the bottom left of the page, select the settings icon.
- Click **API Tokens**.
**![Selecting API Tokens from the settings menu in the Cyera platform](/downloads/uvm/configure/sources/cyera-assets/0887a4d9-d390-4fb7-8405-7c5cd374cc2d.png)
**
- Click **Generate** **Token**.
- Enter a name.
- Select the **Viewer** role or higher.
- Click **Create**.
- Copy the **Client ID** and **Client Secret**. They cannot be accessed again.
