# AttackForge
Source: https://help.zscaler.com/uvm/attackforge
PDF: https://help.zscaler.com/pdf/com/en/1528246.pdf

![The AttackForge tile](/downloads/uvm/configure/sources/attackforge/mceclip0.png)
AttackForge is a penetration management platform. The AttackForge source retrieves the vulnerabilities data available to the user.
Required Parameters
- API Key
- Host: Your host is the beginning of your URL (e.g., `demo.attackforge.com`).
Permissions
- The user must be provided access to RESTful API methods by an administrator.
- The scope of data available to you in the API is restricted to the data available within the AttackForge application. Ensure that you adjust the access to data according to your needs.
