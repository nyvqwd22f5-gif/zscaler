# Nessus
Source: https://help.zscaler.com/uvm/nessus
PDF: https://help.zscaler.com/pdf/com/en/1528381.pdf

![The Nesses tile](/downloads/uvm/configure/sources/nessus/402ff83c-7af5-402d-b522-670f7aa74c20.png)
Nessus scans for security vulnerabilities in devices, applications, operating systems, cloud services, and other network resources.
Required Parameters
- Access Key
- Secret Key
Retrieving the Access Key and Secret Key
An API key consists of an access key and a secret key.
To create an API key:
- Log in to Tenable Nessus.
- From the top navigation bar, click **Settings**. This redirects you to the About page.
- From the left-side navigation, click** My Account.**
- Select the **API Keys** tab.
- Click **Generate**. A dialog box appears, confirming your selection to generate a new API key.
- Click **Generate** again. Your new API key appears.
