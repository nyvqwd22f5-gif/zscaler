# JFrog XRay
Source: https://help.zscaler.com/uvm/jfrog-xray
PDF: https://help.zscaler.com/pdf/com/en/1528401.pdf

![The JFrog XRay tile](/downloads/uvm/configure/sources/jfrog-xray/mceclip0.png)
JFrog Xray is a universal software composition analysis (SCA) solution that integrates with Artifactory and identifies vulnerabilities on open source and license compliance violations.
Parameters
- Server URL: The URL of your organization's JFrog Xray account. The syntax should be `https://[ADDRESS]` and not include a trailing slash.
- Username and Password: The user must have the required permissions.
Prerequisites and User Permissions
- Jfrog version 3.80 or higher
- Valid user with the roles **Manage Watches** and **Manage Reports**
- Optionally, you can disable the **UI Access** and** Can Update Profile** options
