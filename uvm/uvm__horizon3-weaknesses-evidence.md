# Horizon3 Weaknesses Evidence
Source: https://help.zscaler.com/uvm/horizon3-weaknesses-evidence
PDF: https://help.zscaler.com/pdf/com/en/1528176.pdf

Horizon3 provides an autonomous penetration testing solution that helps organizations find, fix, and verify your exploitable attack surface.
Required Parameters
- API Key
- Choose pentest: Select whether to retrieve all pentest or just the latest pentest. This setting is applied to all source runs.
- Choose Pentest type: Add and remove pentest types from the list.
Retrieving the API Key
- In the Horizon3 platform, go to **User** > **Account Settings**
- When creating an API key, you must assign it a role that controls its permissions. For the SecOps platform source, select the **Read-Only** permission.
