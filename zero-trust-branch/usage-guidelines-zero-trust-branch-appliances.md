# Usage Guidelines for Zero Trust Branch Appliances
Source: https://help.zscaler.com/zero-trust-branch/usage-guidelines-zero-trust-branch-appliances
PDF: https://help.zscaler.com/pdf/com/en/1529029.pdf

Zero Trust Branch appliances are installed in an organization’s network and are dedicated to the organization’s traffic. The Zscaler Operations team performs the following tasks on the organization's Zero Trust Branch appliances:
- Provides the required hardware and software for the initial installation of the Zero Trust Branch appliances.
- Ensures that the Zero Trust Branch appliances are suitably packed for shipment.
- Upgrades or replaces theZero Trust Branch appliances to maintain the operational standards of Zscaler. Zscaler will ship the hardware to the organization. It is the organization's responsibility to set up the replacement hardware.
- Maintains the technical refresh status of the Zero Trust Branch appliances.
- Provides remote upgrades of software on the Zero Trust Branch appliances on a schedule that conforms with Zscaler's standard release and maintenance practices.
Zero Trust Branch appliances remain the property of Zscaler during the applicable subscription term.
Organization's Responsibilities
For Zscaler Operations to service, manage, and maintain the Zero Trust Branch appliances, Zscaler requires the organization to:
- Provide a local point of contact for delivery acceptance and assist Zscaler with any specific requirements for delivery.
- Ensure that the Zero Trust Branch appliances are not moved from one physical location to another without prior approval from Zscaler.
- Provide a continuous, uninterrupted, and suitable power supply.
- Implement and maintain temperature, humidity, and other environmental conditions that Zscaler recommends.
- Implement suitable surge protection devices.
- Ensure that each Zero Trust Branch appliance has internet connectivity.
- Ensure that theZero Trust Branch appliances are used to provide Zscaler service only and are not used for any other purpose.
- Ensure that only Zscaler staff access or service the Zero Trust Branch appliances. Neither the organization nor any third party can make any repair attempts or other changes to theZero Trust Branch appliances unless they have explicit approval from Zscaler.
- Manage all network-related issues.
- Use the Zero Trust Branch appliances only in accordance with instructions prescribed by Zscaler and only with hardware and software provided by Zscaler. Also, ensure that the Zero Trust Branch appliances are installed and operated in accordance with the specifications and recommendations provided by Zscaler.
- Comply with the firewall access requirements, if you use a firewall to control access to the products on the Zero Trust Branch appliances.
- Connect the Zero Trust Branch appliances to the appropriate network resources recommended by Zscaler and provide the network resources and the necessary IP addresses to run the service.
If there are any issues, you can receive Zero Trust Branch appliance support services through [Zscaler Support](/contact-support). Zscaler Support works with you to troubleshoot and diagnose the issues remotely. If the issue cannot be resolved remotely or if it is a hardware fault, Zscaler Support might decide to replace the Zero Trust Branch appliance equipment.
Zscaler offers comprehensive return and advanced replacement services for Zero Trust Branch appliances. For details and for information about initiating hardware replacement, see [Return Merchandise Authorization Process and Policy](/zero-trust-branch/return-merchandise-authorization-process-and-policy).
To learn more about Zero Trust Branch, see the detailed [device requirements](https://www.zscaler.com/legal/zero-trust-branch).
