# Deployment Overview
Source: https://help.zscaler.com/zero-trust-branch/deployment-overview
PDF: https://help.zscaler.com/pdf/com/en/1509391.pdf

Zscaler Zero Trust Branch has two components:
- Zero Trust Branch appliances: Provide forwarding and network enforcement points deployed in the customer environments. Depending on the throughput and scale requirements, there are different physical and virtual options available.
- Zero Trust Branch Admin Portal: A centralized Software as a Service (SaaS) management portal used to manage Zero Trust Branch appliances.
There are several ways to deploy Zero Trust Branch:
- [Deploying a Device Using Zero Trust Provisioning](/zero-trust-branch/deploying-device-using-zero-trust-provisioning)
- [Deploying on a VMware Virtual Machine](/zero-trust-branch/deploying-vmware-virtual-machine)
- [Deploying on a Zscaler ZT800](/zero-trust-branch/deploying-zscaler-zt800) (version 7.8 or earlier)
- [Deploying on a Zscaler ZTS4000 or ZTS8000](/zero-trust-branch/deploying-zts4000-zts8000)
