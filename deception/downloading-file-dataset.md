# Downloading a File Dataset
Source: https://help.zscaler.com/deception/downloading-file-dataset
PDF: https://help.zscaler.com/pdf/com/en/1531556.pdf

You can download both default and custom file datasets individually as a ZIP file.
To download a file dataset:
- Go to **Miragemaker **> **Files **> **File Datasets**.
-
Click the **Download **icon for the dataset that you want to download.
[See image.](#zd-download-file-dataset)
The file dataset is downloaded to your system as a ZIP file.
