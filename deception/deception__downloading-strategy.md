# Downloading a Strategy
Source: https://help.zscaler.com/deception/downloading-strategy
PDF: https://help.zscaler.com/pdf/com/en/1531465.pdf

You can download a single strategy as a JSON file. You also have the option to [export all strategies to a compressed file (.zip)](/deception/exporting-strategies).
To download a strategy:
- Go to **Miragemaker **>** Strategy Builder** >** Deception Strategy**.
- Click the **Download** icon for the strategy that you want to download.
[See image.](#deception-download-strategy)
The strategy configuration details are downloaded in the JSON format.
