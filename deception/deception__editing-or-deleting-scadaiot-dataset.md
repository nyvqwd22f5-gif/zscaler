# Editing or Deleting a SCADA/IoT Dataset
Source: https://help.zscaler.com/deception/editing-or-deleting-scadaiot-dataset
PDF: https://help.zscaler.com/pdf/com/en/1531521.pdf

You can edit or delete SCADA/IoT datasets that you configured in the Zscaler Deception Admin Portal.
Editing a SCADA/IoT Dataset
To edit a SCADA/IoT dataset:
- Go to **Miragemaker** > **SCADA/IoT Datasets**.
-
Under **Actions**, click the **Edit** icon for the dataset you want to modify.
[See image.](#zd-edit-option-scada-iot)
-
In the **SCADA/IoT Dataset Details** window, modify the **Name **or upload a different file as required.
[See image.](#zd-edit-window-scada-iot-new)
- Click **Submit**.
The default SCADA/IoT datasets cannot be edited.
Deleting a SCADA/IoT Dataset
To delete a SCADA/IoT dataset:
- Go to **Miragemaker** > **SCADA/IoT Datasets**.
-
Under **Actions**, click the **Delete** icon for the dataset you want to delete.
[See image.](#zd-delete-option-scada-iot)
- In the confirmation window, click **OK**.
The default SCADA/IoT datasets cannot be deleted.
