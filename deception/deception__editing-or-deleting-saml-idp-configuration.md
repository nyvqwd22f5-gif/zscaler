# Editing or Deleting a SAML IdP Configuration
Source: https://help.zscaler.com/deception/editing-or-deleting-saml-idp-configuration
PDF: https://help.zscaler.com/pdf/com/en/1531590.pdf

You can edit or delete a SAML identity provider (IdP) configuration in the Zscaler Deception Admin Portal.
Editing a SAML IdP Configuration
To edit a SAML IdP configuration:
- Go to **Settings** > **Users & Roles** > **IdP Providers**.
-
In the table, locate a SAML IdP configuration you want to modify, and then click the **Edit** icon.
[See image.](#deception-edit-saml-idp)
-
In the **SAML Provider Details** window, modify the required fields.
[See image.](#deception-edit-saml-idp-details)
- Click **Save**.
Deleting a SAML IdP Configuration
To delete a SAML IdP Configuration:
- Go to **Settings** > **Users & Roles** > **IdP Providers**.
-
In the table, locate a SAML IdP configuration you want to delete, and then click the **Delete** icon.
[See image.](#deception-delete-saml-idp)
- In the confirmation window, click **OK**.
