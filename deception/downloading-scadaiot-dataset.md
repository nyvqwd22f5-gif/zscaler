# Downloading a SCADA/IoT Dataset
Source: https://help.zscaler.com/deception/downloading-scadaiot-dataset
PDF: https://help.zscaler.com/pdf/com/en/1531520.pdf

You can download both default and custom SCADA/IoT datasets individually as a ZIP file.
To download a SCADA/IoT dataset:
- Go to **Miragemaker **> **SCADA/IoT Datasets**.
-
Click the **Download **icon for the dataset that you want to download.
[See image.](#zd-download-scada-iot-dataset)
The SCADA/IoT dataset is downloaded to your system as a ZIP file.
