# Exporting Network Decoy Personalities
Source: https://help.zscaler.com/deception/exporting-network-decoy-personalities
PDF: https://help.zscaler.com/pdf/com/en/1531526.pdf

You can export a ZIP file containing all network decoy personalities, each in its own JSON file.
To export network decoy personalities:
- Go to **Miragemaker **> **Strategy Builder **> **Network Decoy**.
- Click **Export**.
[See image.](#zd-export-network-decoy-personalities)
All network decoy personalities are downloaded to your system in a compressed (.zip) file.
