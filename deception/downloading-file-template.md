# Downloading a File Template
Source: https://help.zscaler.com/deception/downloading-file-template
PDF: https://help.zscaler.com/pdf/com/en/1531557.pdf

You can download both default and custom file templates individually as a ZIP file.
To download a file template:
- Go to **Miragemaker **> **Files **> **File Templates**.
-
Click the **Download **icon for the template that you want to download.
[See image.](#zd-download-file-dataset)
The file template is downloaded to your system as a ZIP file.
