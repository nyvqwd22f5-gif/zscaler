# Exporting Active Directory Decoy Personalities
Source: https://help.zscaler.com/deception/exporting-active-directory-decoy-personalities
PDF: https://help.zscaler.com/pdf/com/en/1531540.pdf

You can export a ZIP file containing all Active Directory (AD) decoy personalities, each in its own JSON file.
To export AD decoy personalities:
- Go to **Miragemaker **> **Strategy Builder **> **Active Directory Decoy**.
- Click **Export**.
[See image.](#zd-export-ad-decoy-personalities)
All AD decoy personalities are downloaded to your system in a compressed (.zip) file.
