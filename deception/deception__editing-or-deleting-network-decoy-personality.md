# Editing or Deleting a Network Decoy Personality
Source: https://help.zscaler.com/deception/editing-or-deleting-network-decoy-personality
PDF: https://help.zscaler.com/pdf/com/en/1531525.pdf

You can edit or delete network decoy personalities that you configured in the Zscaler Deception Admin Portal.
Editing a Network Decoy Personality
To edit a network decoy personality:
- Go to **Miragemaker** > **Strategy Builder **> **Network Decoy**.
- Under **Actions**, click the **Edit** icon for the decoy personality you want to modify.
[See image.](#zd-edit-option-ndp)
- In the **Personality Details** window, modify the fields [under each tab](/deception/configuring-network-decoy-personality) (**General**, **Services**, and **Active Directory**) as required.
[See image.](#zd-edit-window-ndp)
- Click **Submit**.
The default network decoy personalities cannot be edited.
Deleting a Network Decoy Personality
To delete a network decoy personality:
- Go to **Miragemaker** > **Strategy Builder **> **Network Decoy**.
- Under **Actions**, click the **Delete** icon for the decoy personality you want to delete.
[See image.](#zd-delete-option-ndp)
- In the confirmation window, click **OK**.
The default network decoy personalities cannot be deleted.
