# Downloading a Sandbox Report
Source: https://help.zscaler.com/deception/downloading-sandbox-report
PDF: https://help.zscaler.com/pdf/com/en/1531394.pdf

You can send evidence files to Zscaler Sandbox or a third-party sandbox. To learn more, see [About Sandbox](/zia/about-sandbox). After the evidence file is sent, you can download an analysis report. This report can be used by analysts to enrich threat intelligence and security operation workflows in order to strengthen defenses.
Downloading a Zscaler Sandbox Report
To download a Zscaler Sandbox report:
- On the **Evidence** tab, select an Indicators of Compromise (IOC) evidence file that you have sent to the Zscaler Sandbox.
-
Click the **Download** icon under the **Report** column.
[See image.](#download-sandbox-report-icon)
-
In the **Download Report** window, select **Zscaler Sandbox**.
[See image.](#download-zscaler-sandbox-report)
-
Click **Done**. The **Zscaler Sandbox Report** page appears.
[See image.](#zscaler-sandbox-report-page)
To learn more, see [Understanding the Zscaler Sandbox Report](/deception/understanding-zscaler-sandbox-report).
-
Click **Download Report**.
The report is downloaded to your system in the JSON format.
Downloading a Third-Party Sandbox Report
To download a third-party sandbox report:
- On the **Evidence** tab, select an IOC evidence file that you have sent to a third-party sandbox.
-
Click the **Download** icon under the **Report** column.
[See image.](#download-third-party-sandbox-report-icon)
-
In the **Download Report** window, select a third-party sandbox.
[See image.](#download-third-party-sandbox-report)
-
Click **Done**.
The report is downloaded to your system in either a PDF or JSON format.
