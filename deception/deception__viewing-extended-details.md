# Viewing Extended Details
Source: https://help.zscaler.com/deception/viewing-extended-details
PDF: https://help.zscaler.com/pdf/com/en/1531376.pdf

The extended details page shows the in-depth analysis of attack activities. On this page, you can view:
- [ThreatParse](/deception/about-threatparse) that consists of easy-to-understand attack details.
- [Attack chronology](/deception/about-attack-chronology) that displays a temporal overview and heatmap of the attacker’s activities.
- [Attacker’s network activity](/deception/about-network) that displays a temporal overview and heatmap of the attacker’s activities.
- [Event logs](/deception/about-event-logs) that show details of alerts created by an attacker for a decoy.
- [Evidence files](/deception/about-evidence) in Packet Capture (PCAP), Remote Desktop Protocol (RDP), and Indicators of Compromise (IoC) formats.
To view the extended details page:
-
On the **Investigate** page, click an element in the alert graph. For example, click an attacker icon or a Decoy Connector.
The details pane opens.
-
In the details pane, click the **View Extended Details **button.
[See image.](#extended-details-button)
The **Extended Details** window appears.
-
To view in-depth details, click one of the tabs:
- **ThreatParse**: Provides information about the attack based MITRE ATT&CK framework along with a risk score in plain English. To learn more, see [Viewing the ThreatParse Details](/deception/viewing-threatparse-details).
- **Chronology**: Provides a temporal overview and heatmap of the attacker's activities. To learn more, see [Viewing the Attack Chronology Details](/deception/viewing-attack-chronology-details).
- **Network**: Provides information about the attacker's activity on your network based on the telemetry generated by network decoys. To learn more, see [Viewing the Network Details](/deception/viewing-network-details).
- **Event Logs**: Provides detailed log information the events associated with an activity. To learn more, see [About Event Logs](/deception/about-event-logs).
- **Evidence**: Provides details of evidence of an attack in the PCAP, RDP, or IoC files that help gain more insights into an attack. To learn more, see [About Evidence](/deception/about-evidence).
