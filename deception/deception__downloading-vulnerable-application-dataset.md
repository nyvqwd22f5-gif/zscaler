# Downloading a Vulnerable Application Dataset
Source: https://help.zscaler.com/deception/downloading-vulnerable-application-dataset
PDF: https://help.zscaler.com/pdf/com/en/1531555.pdf

You can download both default and custom vulnerable application datasets individually as a ZIP file.
To download a vulnerable application dataset:
- Go to **Miragemaker **> **Vulnerable Application Datasets (CVE Datasets)**.
-
Click the **Download **icon for the dataset that you want to download.
[See image.](#zd-download-vulnerable-application-dataset)
The vulnerable application dataset is downloaded to your system as a ZIP file.
