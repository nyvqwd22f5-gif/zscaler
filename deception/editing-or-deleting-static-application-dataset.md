# Editing or Deleting a Static Application Dataset
Source: https://help.zscaler.com/deception/editing-or-deleting-static-application-dataset
PDF: https://help.zscaler.com/pdf/com/en/1531553.pdf

You can edit or delete static application datasets that you configured in the Zscaler Deception Admin Portal.
Editing a Static Application Dataset
To edit a static application dataset:
- Go to **Miragemaker** > **Static Application Datasets**.
-
Under **Actions**, click the **Edit** icon for the dataset you want to modify.
[See image.](#zd-edit-option-stad)
-
In the **Static Application Dataset Details** window, modify the [required fields](/deception/adding-static-application-dataset).
[See image.](#zd-edit-window-stad)
- Click **Submit**.
The default static application datasets cannot be edited.
Deleting a Static Application Dataset
To delete a static application dataset:
- Go to **Miragemaker** > **Static Application Datasets**.
-
Under **Actions**, click the **Delete** icon for the dataset you want to delete.
[See image.](#zd-delete-option-stad)
- In the confirmation window, click **OK**.
The default static application datasets cannot be deleted.
