# About Active Directory Decoy Personalities
Source: https://help.zscaler.com/deception/about-active-directory-decoy-personalities
PDF: https://help.zscaler.com/pdf/com/en/1531537.pdf

Active Directory (AD) decoy personalities are templates that can be used to create [AD decoys](/deception/about-active-directory-decoys) via [deception strategies](/deception/about-deception-strategy). Zscaler Deception provides a list of preconfigured AD decoy personalities for various business use cases. You can also create custom personalities based on your business requirements.
AD decoy personalities provide the following benefits and enable you to:
- Use templates to deploy AD decoys via a deception strategy.
- Leverage preconfigured templates or create new templates to define AD decoys for various business use cases.
About the Active Directory Decoy Personalities Page
On the Active Directory Decoy Personalities page (Miragemaker > Strategy Builder > Active Directory Decoy), you can do the following:
- View the list of preconfigured and custom AD decoy personalities, and search for a specific entry by name. For each decoy personality, you can see:
- **Name**: The name of the AD decoy personality.
- **Description**: The description of the AD decoy personality.
- [Configure an AD decoy personality](/deception/configuring-active-directory-decoy-personality).
- [Download an AD decoy personality](/deception/downloading-active-directory-decoy-personality).
- [Edit or delete an AD decoy personality](/deception/editing-or-deleting-active-directory-decoy-personality).
- [Export all AD decoy personalities](/deception/exporting-active-directory-decoy-personalities).
