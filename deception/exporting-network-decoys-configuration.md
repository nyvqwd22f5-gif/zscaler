# Exporting Network Decoys Configuration
Source: https://help.zscaler.com/deception/exporting-network-decoys-configuration
PDF: https://help.zscaler.com/pdf/com/en/1531278.pdf

You can export the network decoy configuration details as a CSV file.
To export the network decoy configuration details:
- Go to **Deceive** > **Network Decoys**.
- Select the **Internal **or **Zero Trust Network **tab.
- Click **Export Decoy Configuration**.
![Export decoy configuration](/downloads/deception/deceive/network-decoys/network-decoy-management/exporting-network-decoys-configuration/zscaler-deception-export-decoy-config-1.png)
A CSV file with the decoy configuration details is downloaded to your local system.
