# Adding a Custom File Template
Source: https://help.zscaler.com/deception/adding-custom-file-template
PDF: https://help.zscaler.com/pdf/com/en/1531449.pdf

You can add a custom file template and use it as a baseline when [adding dynamic file datasets](/deception/adding-dynamic-file-dataset). Before you can add a template, you must create a compressed (.zip) file that includes the template's files.
To add a custom file template:
- Go to **Miragemaker** > **Files** > **File Templates**.
-
Click **Add Template**.
[See image.](#deception-add-file-template-option)
-
In the **File Template Details** window:
- Enter a **Name** for the template.
- Click **Upload** and browse to upload the compressed (.zip) file.
[See image.](#deception-add-file-template-details)
-
Click **Submit**.
The custom file template is added.
[See image.](#deception-custom-file-template-added)
