# Enabling Active Directory Decoy Deployment on a Server Agent
Source: https://help.zscaler.com/deception/enabling-active-directory-decoy-deployment-server-agent
PDF: https://help.zscaler.com/pdf/com/en/1531613.pdf

You can enable Active Directory (AD) decoy deployments on a server agent.
To enable AD decoy deployment on a server agent:
- Go to **Settings** > **Server Agent Settings** > **Domains**.
-
Locate the domain you want, and click the **Edit **icon under the** Actions** column.
[See image.](#edit_icon_for_domains_tab)
-
In the dialog window:
- **Enabled**: Select to manage AD decoys deployment using a server agent.
- **Agent**: Select a server agent from the drop-down menu.
[See image.](#policy_configuring_for_server_agents)
- Click **Save**.
The domain is added, and you can view the AD domains managed by server agents on the Domains page (Deceive > Active Directory Decoys > Domains).
