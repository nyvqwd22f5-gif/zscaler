# Downloading Landmine Agents
Source: https://help.zscaler.com/deception/downloading-landmine-agents
PDF: https://help.zscaler.com/pdf/com/en/1531310.pdf

Landmine agents are persistent agents installed on endpoints, such as desktops or laptops on your network. These agents deploy decoy credentials, files, processes, and lures on the endpoints.
To download a landmine agent:
- Go to **Settings **> **Endpoint Settings **> **Agents**.
-
Click **Download Agent**, and select** Landmine Agent** from the drop-down menu.
[See image.](#deception-download-landmine-agent-installer-option)
-
In the **Download Landmine Agent** window, choose an OS installation type:
- **Windows (EXE)**
- **Windows (MSI)**
- **macOS Intel (pkg)**
- **macOS Apple Silicon (pkg)**
- **Linux (rpm)**
- **Linux (deb)**
[See image.](#deception-download-landmine-agent-installer)
If endpoint deception with Zscaler Client Connector is enabled for Windows, download the landmine agent from Zscaler Client Connector. To learn more, see [Deploying Endpoint Deception with Zscaler Client Connector for Windows](/deception/deploying-endpoint-deception-zscaler-client-connector-windows).
-
Click **Download Installer**.
The latest version of the landmine agent is downloaded to your system. The version number is displayed on the title bar of the **Download Landmine Agent** window.
