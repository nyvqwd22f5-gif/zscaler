# Downloading a Network Decoy Personality
Source: https://help.zscaler.com/deception/downloading-network-decoy-personality
PDF: https://help.zscaler.com/pdf/com/en/1531524.pdf

You can download both default and custom network decoy personalities individually as a JSON file.
To download a network decoy personality:
- Go to **Miragemaker **> **Strategy Builder **> **Network Decoy**.
- Click the **Download **icon for the decoy personality that you want to download.
[See image.](#zd-download-ndp)
The network decoy personality is downloaded to your system as a JSON file.
