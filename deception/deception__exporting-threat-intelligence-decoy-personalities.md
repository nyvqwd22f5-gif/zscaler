# Exporting Threat Intelligence Decoy Personalities
Source: https://help.zscaler.com/deception/exporting-threat-intelligence-decoy-personalities
PDF: https://help.zscaler.com/pdf/com/en/1531531.pdf

You can export a ZIP file containing all Threat Intelligence (TI) decoy personalities, each in its own JSON file.
To export TI decoy personalities:
- Go to **Miragemaker **> **Strategy Builder **> **Threat Intelligence Decoy**.
- Click **Export**.
[See image.](#zd-export-ti-decoy-personalities)
All TI decoy personalities are downloaded to your system in a compressed (.zip) file.
