# Downloading and Uploading the Subnet Configuration
Source: https://help.zscaler.com/deception/downloading-and-uploading-subnet-configuration
PDF: https://help.zscaler.com/pdf/com/en/1531287.pdf

Downloading the Subnet Configuration
[]You can download subnet configurations added to the Zscaler Deception Admin Portal.
To download a CSV file with the subnet configuration:
- Log in to the Deception Admin Portal.
- On the left-side navigation, go to **Settings** > **Interfaces**.
-
Click **Config**, and then select **Download Config** from the drop-down menu.
[See image.](#download-subnet-config)
A CSV file with the subnet configuration details is downloaded.
[See image.](#downloaded-csv-file)
Uploading the Subnet Configuration
Uploading a CSV file with subnet configuration enables you to add and configure more than one subnet at a time.
To upload a CSV file with subnet configuration:
- [Download the CSV file with the subnet configuration](#zscaler-deception-download-subnet).
- Open and update the CSV file with your subnet information.
- Log in to the Deception Admin Portal.
- On the left-side navigation, go to **Settings** > **Interfaces**.
-
Click **Config**, and then select **Upload Config** from the drop-down menu.
[See image.](#upload-subnet-config)
- Browse and select the CSV file, then click **Open**.
-
In the confirmation window, click **OK**.
[See image.](#upload-csv-subnet-confirmation)
The new subnets are added to the **Interfaces** table.
