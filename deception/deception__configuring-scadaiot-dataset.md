# Configuring a SCADA/IoT Dataset
Source: https://help.zscaler.com/deception/configuring-scadaiot-dataset
PDF: https://help.zscaler.com/pdf/com/en/1531519.pdf

You can create a custom dataset for Supervisory Control and Data Acquisition (SCADA) and Internet of Things (IoT) devices, and use them to create SCADA/IoT decoys.
To create a SCADA/IoT dataset:
- Go to **Miragemaker **> **SCADA/IoT Datasets**.
-
Locate an existing SCADA/IoT dataset based on the new custom dataset you want to create, and click the **Download **icon.
[See image.](#zd-scada-iot-download-option)
The SCADA/IoT dataset (.zip) is downloaded to your system.
- Make the necessary changes to the SCADA/IoT dataset that you downloaded to meet your requirements.
-
Click **Add Dataset**.
[See image.](#zd-scada-iot-add-option)
- In the **SCADA/IoT Dataset Details** window:
- **Name**: Enter the name for the SCADA/IoT dataset.
-
**Dataset/Templates (Zip File)**: Click **Upload** to upload the custom dataset you saved to your system.
[See image.](#zd-scada-iot-configuration)
-
Click **Submit**.
The SCADA/IoT dataset is uploaded to the Zscaler Deception Admin Portal.
You can use the dataset in network decoys to deploy SCADA/IoT decoys. To learn more, see [Configuring Services on a Network Decoy](/deception/configuring-services-network-decoy#configuring-scada-iot-service).
