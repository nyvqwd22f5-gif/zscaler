# Uploading Password Keywords
Source: https://help.zscaler.com/itdr/uploading-password-keywords
PDF: https://help.zscaler.com/pdf/com/en/1531853.pdf

You can upload a text file that includes the password keywords.
To upload a text file with password keywords:
- Go to **Settings** > **Server Agent Settings** > **Configuration**.
- Click **Upload**.
[See image.](#view_keywords_upload)
- Locate and select the text file on your system, then click **Open**.
-
Click **Save**.
[See image.](#view_saving_password_keyword)
The password keywords are successfully saved.
