# Downloading a Landmine Decoy Personality
Source: https://help.zscaler.com/deception/downloading-landmine-decoy-personality
PDF: https://help.zscaler.com/pdf/com/en/1531534.pdf

You can download both default and custom landmine decoy personalities individually as a JSON file.
To download a landmine decoy personality:
- Go to **Miragemaker **> **Strategy Builder **> **Landmine**.
-
Click the **Download **icon for the decoy personality that you want to download.
[See image.](#zd-download-ldp)
The landmine decoy personality is downloaded to your system as a JSON file.
