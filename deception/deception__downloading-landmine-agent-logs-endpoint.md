# Downloading Landmine Agent Logs from an Endpoint
Source: https://help.zscaler.com/deception/downloading-landmine-agent-logs-endpoint
PDF: https://help.zscaler.com/pdf/com/en/1531395.pdf

This article provides instructions on how to trigger the downloading of landmine agent logs from an endpoint that is actively connected to the Zscaler Deception Admin Portal. After you download the logs, you can send them to Zscaler Support to troubleshoot issues.
You cannot download logs from an endpoint for a landmine agentless installer. To learn more, see [About Landmine Agent and Agentless](/deception/about-landmine-agents).
To download landmine agent logs from an endpoint:
- Go to **Settings **> **Endpoint Settings **> **Agents**.
- In the **Agents** table, select an agent for which you want to download the logs.
-
From the** Actions **drop-down menu, select **Trigger Logs**.
[See image.](#trigger-option-action-menu)
After 10 minutes, a **Download** icon appears under the **Actions** column of the selected agent.
[See image.](#download-agent-logs-icon)
To check when the agent logs were uploaded to the Deception Admin Portal, hover over the **Download **icon.
[See image.](#zd-agent-log-tooltip)
-
Click the **Download **icon.
The landmine agent logs from the endpoint are downloaded to your local system in a compressed (.zip) file.
