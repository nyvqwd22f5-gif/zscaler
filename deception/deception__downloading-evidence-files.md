# Downloading Evidence Files
Source: https://help.zscaler.com/deception/downloading-evidence-files
PDF: https://help.zscaler.com/pdf/com/en/1531369.pdf

You can download the evidence file in the following formats:
- Packet capture (PCAP)
- Remote desktop protocol (RDP) recording files
- Indicators of compromise (IOC)
To download an evidence file:
-
On the **Evidence** page, select an activity based on the reported timestamp.
You can filter the evidence files based on the file type using the drop-down menu under the **Type** column. For example, you can select **ioc** to filter IOC files only.
-
Under the **Actions** column, click the **Download** icon.![Download the evidence file](/downloads/deception/investigate/exporting-evidence-files/Download%20Evidence%20Files.jpg)
The evidence file is downloaded to your system.
PCAP files contain one minute or 5 MB of PCAP data, whichever is less. You can download a unified PCAP file for the selected time duration from the [Network](/deception/about-network) page.
