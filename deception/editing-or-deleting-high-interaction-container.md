# Editing or Deleting a High-Interaction Container
Source: https://help.zscaler.com/deception/editing-or-deleting-high-interaction-container
PDF: https://help.zscaler.com/pdf/com/en/1531554.pdf

You can edit or delete high-interaction containers that you configured in the Zscaler Deception Admin Portal.
Editing a High-Interaction Container
To edit a high-interaction container:
- Go to **Miragemaker** > **High-Interaction Containers**.
-
Under **Actions**, click the **Edit** icon for the high-interaction container you want to modify.
[See image.](#zd-edit-option-hc)
-
In the **Web Application Container Details** window or the **Other Custom Protocols Container Details** window, modify the required fields. To learn more, see [Configuring a Web Application Container](/deception/configuring-web-application-container) and [Configuring a Custom Protocol Container](/deception/configuring-custom-protocol-container).
[See image.](#zd-edit-window-hc)
- Click **Submit**.
The default high-interaction containers cannot be edited.
Deleting a High-Interaction Container
To delete a high-interaction container:
- Go to **Miragemaker** > **High-Interaction Containers**.
-
Under **Actions**, click the **Delete** icon for the high-interaction container you want to delete.
[See image.](#zd-delete-option-hc)
- In the confirmation window, click **OK**.
The default high-interaction containers cannot be deleted.
