# Exporting Threat Intelligence Decoys DNS Configuration
Source: https://help.zscaler.com/deception/exporting-threat-intelligence-decoys-dns-configuration
PDF: https://help.zscaler.com/pdf/com/en/1531457.pdf

You can export DNS configuration details for Threat Intelligence (TI) decoys as a CSV file.
To export the DNS configuration details:
- Go to **Deceive** > **Threat Intelligence Decoys**.
- Click **Export DNS**.
[]![Export TI decoys DNS configuration details](/downloads/deception/product-documentation/deceive/threat-intelligence-decoys/threat-intelligence-decoy-management/exporting-threat-intelligence-decoys-dns-configuration/zscaler-deception-threat-intelligence-export-dns-2.png)
A CSV file is downloaded to your system.
