# Viewing Decoy Connector Update Logs
Source: https://help.zscaler.com/deception/viewing-decoy-connector-update-logs
PDF: https://help.zscaler.com/pdf/com/en/1531270.pdf

Update logs provide valuable information that you can use to troubleshoot Decoy Connector update issues.
To view the update logs:
- Log in to the Zscaler Deception Admin Portal.
- Go to **Settings** > **Topology** > **Decoy Connectors**.
- Select a Decoy Connector for which you want to view the update logs, and then click **Logs**.
-
Select **View update logs** from the drop-down menu.
[See image.](#view-update-logs)
-
When the **Update Logs** window appears, view the logs, and then click **Close**.
[See image.](#update-logs)
