# Editing or Deleting a Vulnerable Application Dataset
Source: https://help.zscaler.com/deception/editing-or-deleting-vulnerable-application-dataset
PDF: https://help.zscaler.com/pdf/com/en/1531480.pdf

You can edit or delete vulnerable application datasets that you have configured in the Zscaler Deception Admin Portal.
Editing a Vulnerable Application Dataset
To edit a vulnerable application dataset that you have configured:
- Go to **Miragemaker** > **Vulnerable Application Datasets (CVE Datasets)**.
- Under **Actions**, click the **Edit** icon for the vulnerable application dataset you want to modify.
- In the **Vulnerable Application Datasets (CVE Datasets) Details** window, modify the required fields.
- Click **Submit**.
The default vulnerable application datasets cannot be edited.
Deleting a Vulnerable Application Dataset
To delete a vulnerable application dataset that you have configured:
- Go to **Miragemaker** > **Vulnerable Application Datasets (CVE Datasets)**.
- Under **Actions**, click the **Delete** icon for the vulnerable application dataset you want to delete.
- In the confirmation window, click **OK**.
The default vulnerable application datasets cannot be deleted.
