# Editing or Deleting a Dynamic Application Dataset
Source: https://help.zscaler.com/deception/editing-or-deleting-dynamic-application-dataset
PDF: https://help.zscaler.com/pdf/com/en/1531483.pdf

You can edit or delete dynamic application datasets that you have configured in the Zscaler Deception Admin Portal
Editing a Dynamic Application Dataset
To edit a dynamic application dataset that you have configured:
- Go to **Miragemaker** > **Dynamic Application Datasets**.
- Under **Actions**, click the **Edit** icon for the dynamic application dataset you want to modify.
- In the **Dynamic Application Datasets Details** window, modify the required fields.
- Click **Submit**.
The default dynamic application datasets cannot be edited.
Deleting a Dynamic Application Dataset
To delete a dynamic application dataset that you have configured:
- Go to **Miragemaker** > **Dynamic Application Datasets**.
- Under **Actions**, click the **Delete** icon for the dynamic application dataset you want to delete.
- In the confirmation window, click **OK**.
The default dynamic application datasets cannot be deleted.
