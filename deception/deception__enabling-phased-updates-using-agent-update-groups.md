# Enabling Phased Updates Using Agent Update Groups
Source: https://help.zscaler.com/deception/enabling-phased-updates-using-agent-update-groups
PDF: https://help.zscaler.com/pdf/com/en/1531321.pdf

You can enable phased updates for the landmine agents to selectively roll out the updates using agent update groups.
To enable the phased updates:
- Go to **Settings **> **Endpoint Settings **> **Agent Update Groups**.
-
Click **Enable Phased Updates**.
[See image.](#phased-updates-option)
-
In the confirmation window, click **OK**.
[See image.](#confirmation-phased-updates)
Enabling this feature stops any automatic updates for agents that are not a part of the groups.
