# Downloading an Active Directory Decoy Personality
Source: https://help.zscaler.com/deception/downloading-active-directory-decoy-personality
PDF: https://help.zscaler.com/pdf/com/en/1531539.pdf

You can download both default and custom Active Directory (AD) decoy personalities individually as a JSON file.
To download an AD decoy personality:
- Go to **Miragemaker **> **Strategy Builder **> **Active Directory Decoy**.
- Click the **Download **icon for the decoy personality that you want to download.
[See image.](#zd-download-addp)
The AD decoy personality is downloaded to your system as a JSON file.
