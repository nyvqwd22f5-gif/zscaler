# Enabling Phased Updates Using Server Agent Update Groups
Source: https://help.zscaler.com/deception/enabling-phased-updates-using-server-agent-update-groups
PDF: https://help.zscaler.com/pdf/com/en/1531607.pdf

You can update the server agents in phases to roll out updates using agent update groups.
Enabling this feature stops any automatic updates for agents that are not a part of the groups.
To enable the phased updates:
- Go to **Settings **> **Server Agent Settings **> **Agent Update Groups**.
-
Click **Enable Phased Updates**.
[See image.](#phased-updates-option)
-
In the confirmation window, click **OK**.
[See image.](#confirmation-phased-updates)
