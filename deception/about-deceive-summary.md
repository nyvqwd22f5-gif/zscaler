# About Deceive Summary
Source: https://help.zscaler.com/deception/about-deceive-summary
PDF: https://help.zscaler.com/pdf/com/en/1531346.pdf

The Deceive Summary page displays the details of all the decoys that are deployed in the Zscaler Deception Admin Portal.
Deceive Summary provides the following benefits and enables you to:
- View the summary of all the deployed decoys and review the health status of Decoy Connector and service back ends.
- Analyze decoy deployment logs to debug the components that caused an error during the deployment.
About the Deceive Summary Page
On the Summary page (Deceive > Summary), you can do the following:
- View the summary of all the decoys that are created and deployed in the Deception Admin Portal.
- View the decoy deployment status percentage.
- [View the deployment health status of the Decoy Connector and service back end](/deception/viewing-decoy-deployment-health-status).
- [Start or stop decoys](/deception/starting-and-stopping-decoys).
- [View and analyze deployment logs](/deception/viewing-decoy-deployment-logs) to troubleshoot deployment issues.
