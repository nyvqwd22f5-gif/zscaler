# Disabling Support User Access
Source: https://help.zscaler.com/deception/disabling-support-user-access
PDF: https://help.zscaler.com/pdf/com/en/1531486.pdf

You can disable support users from accessing the Zscaler Deception Admin Portal.
Disabling access to support users might cause delayed assistance from the Zscaler Support team. The Zscaler Support team might request you to enable access when you submit a help ticket for troubleshooting issues.
To disable access to support users:
- Go to **Settings** > **Users & Roles** > **Settings**.
-
Under** Support Access Settings**, select **Disable Support Access**.
[See image.](#deception-disable-access-support-user)
- Click **Save**.
-
In the **Action Required** window, click **Confirm**.
[See image.](#deception-disable-support-user-resubmit)
