# Exporting Threat Intelligence Decoys Configuration
Source: https://help.zscaler.com/deception/exporting-threat-intelligence-decoys-configuration
PDF: https://help.zscaler.com/pdf/com/en/1531290.pdf

You can export the Threat Intelligence (TI) configuration details as a CSV file.
To export the TI configuration details:
- Go to **Deceive** > **Threat Intelligence Decoys**.
- Click **Export Deployment Configuration**.
![Export TI decoy configuration details](/downloads/deception/product-documentation/deceive/threat-intelligence-decoys/threat-intelligence-decoy-management/exporting-threat-intelligence-decoys-configuration/zscaler-deception-export-ti-decoy-configurations-1.png)
A CSV file with the configuration details is downloaded to your system.
