# Editing or Deleting an Active Directory Decoy Personality
Source: https://help.zscaler.com/deception/editing-or-deleting-active-directory-decoy-personality
PDF: https://help.zscaler.com/pdf/com/en/1531541.pdf

You can edit or delete Active Directory (AD) decoy personalities configured in the Zscaler Deception Admin Portal.
Editing an Active Directory Decoy Personality
To edit an AD decoy personality:
- Go to **Miragemaker** > **Strategy Builder **> **Active Directory Decoy**.
- Under **Actions**, click the **Edit** icon for the decoy personality you want to modify.
[See image.](#zd-edit-option-addp)
- In the **Decoy Details** window, modify the [required fields](/deception/configuring-active-directory-decoy-personality).
[See image.](#zd-edit-window-addp)
- Click **Save**.
The default AD decoy personalities cannot be edited.
Deleting an Active Directory Decoy Personality
To delete an AD decoy personality:
- Go to **Miragemaker** > **Strategy Builder **> **Active Directory Decoy**.
- Under **Actions**, click the **Delete** icon for the decoy personality you want to delete.
[See image.](#zd-delete-option-addp)
- In the confirmation window, click **OK**.
The default AD decoy personalities cannot be deleted.
