# Downloading a Static Application Dataset
Source: https://help.zscaler.com/deception/downloading-static-application-dataset
PDF: https://help.zscaler.com/pdf/com/en/1531552.pdf

You can download both default and custom static application datasets individually as a ZIP file.
To download a static application dataset:
- Go to **Miragemaker **> **Static Application Datasets**.
-
Click the **Download **icon for the dataset that you want to download.
[See image.](#zd-download-static-application-dataset)
The static application dataset is downloaded to your system as a ZIP file.
