# Configuring Keyword Datasets
Source: https://help.zscaler.com/deception/configuring-keyword-datasets
PDF: https://help.zscaler.com/pdf/com/en/1531517.pdf

You can configure [keyword datasets](/deception/understanding-keyword-datasets) to autogenerate recommendations for decoy parameters, such as hostnames, file and folder names, network decoy names, etc.
To configure keywords:
- Go to **Miragemaker **> **Keyword Datasets**.
-
Enter keywords (one per line) of your choice.
[See image.](#zd-keyword-dataset)
- Click **Save**.
Zscaler recommends using keywords that can lure attackers. For example, you can use keywords specific to a customer or project, such as internal code names, abbreviations, etc.
