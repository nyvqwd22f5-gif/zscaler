# Editing or Deleting a File Dataset
Source: https://help.zscaler.com/deception/editing-or-deleting-file-dataset
PDF: https://help.zscaler.com/pdf/com/en/1531558.pdf

You can edit or delete file datasets that you configured in the Zscaler Deception Admin Portal.
Editing a File Dataset
To edit a file dataset:
- Go to **Miragemaker** > **Files **> **File Datasets**.
-
Under **Actions**, click the **Edit** icon for the dataset you want to modify.
[See image.](#zd-edit-option-file-dataset)
-
In the **File Dataset Details** window, modify the required fields depending on the type of dataset ([static](/deception/adding-static-file-dataset) or [dynamic](/deception/adding-dynamic-file-dataset)).
[See image.](#zd-edit-window-file-dataset)
- Click **Submit**.
The default file datasets cannot be edited.
Deleting a File Dataset
To delete a file dataset:
- Go to **Miragemaker** > **File **> **File Datasets**.
-
Under **Actions**, click the **Delete** icon for the dataset you want to delete.
[See image.](#zd-delete-option-file-dataset)
- In the confirmation window, click **OK**.
The default file datasets cannot be deleted.
