# Viewing Decoy Deployment Logs
Source: https://help.zscaler.com/deception/viewing-decoy-deployment-logs
PDF: https://help.zscaler.com/pdf/com/en/1531352.pdf

You can view and analyze decoy deployment logs to troubleshoot deployment issues. You can use the logs to debug the components that caused an error during the decoy deployment.
To view the decoy deployment logs:
- Go to **Deceive** > **Summary**.
- Click **View Deployment Logs**.
[See image.](#deception-view-deployment-logs)
The **Deployment Logs** window appears. The red text indicates deployment issues.
[See image.](#deception-deployment-logs)
Click **Show Only Failed **to view only the failed deployment log details.
