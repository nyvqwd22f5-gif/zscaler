# Exporting Strategies
Source: https://help.zscaler.com/deception/exporting-strategies
PDF: https://help.zscaler.com/pdf/com/en/1531464.pdf

You can export all the strategies as a compressed file (.zip). You also have the option to [download a single strategy as a JSON file](/deception/downloading-deception-strategy).
To export strategies:
- Go to **Miragemaker **>** Strategy Builder **>** Deception Strategy**.
- Click **Export**.
[See image.](#deception-export-strategy)
A .zip file is downloaded. You can extract the .zip file to see the configuration details of each strategy in the JSON format.
