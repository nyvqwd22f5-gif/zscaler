# Editing or Deleting a File Template
Source: https://help.zscaler.com/deception/editing-or-deleting-file-template
PDF: https://help.zscaler.com/pdf/com/en/1531559.pdf

You can edit or delete file templates that you configured in the Zscaler Deception Admin Portal.
Editing a File Template
To edit a file template:
- Go to **Miragemaker** > **Files **> **File Template**.
-
Under **Actions**, click the **Edit** icon for the template you want to modify.
[See image.](#zd-edit-option-file-template)
-
In the **File Template Details** window, modify the [required fields](/deception/adding-custom-file-template).
[See image.](#zd-edit-window-file-template)
- Click **Submit**.
The default file templates cannot be edited.
Deleting a File Template
To delete a file template:
- Go to **Miragemaker** > **File **> **File Templates**.
-
Under **Actions**, click the **Delete** icon for the template you want to delete.
[See image.](#zd-delete-option-file-template)
- In the confirmation window, click **OK**.
The default file templates cannot be deleted.
