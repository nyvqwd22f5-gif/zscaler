# Editing and Deleting a Network Decoy
Source: https://help.zscaler.com/deception/editing-and-deleting-network-decoy
PDF: https://help.zscaler.com/pdf/com/en/1531284.pdf

You can edit or delete network decoys that you configured in the Zscaler Deception Admin Portal.
Editing a Network Decoy
- [Stop the decoys](/deception/starting-and-stopping-decoys#stop-decoys).
- Go to **Deceive** > **Network Decoys**.
- Select the **Internal **or **Zero Trust Network **tab.
- Navigate to the network decoy you want to edit, and click the **Edit** icon.
[See image.](#edit-icon-network-decoys)
- In the **Network Decoy Detail** window, modify the decoy configuration per your requirements.
- Click **Submit**.
[See image.](#edit-network-decoy-details)
- [Start the decoys](/deception/starting-and-stopping-decoys#start-decoys).
Deleting a Network Decoy
- [Stop the decoys](/deception/starting-and-stopping-decoys#stop-decoys).
- Select the **Internal **or **Zero Trust Network **tab.
- Navigate to the network decoy you want to delete, and click the **Delete** icon.
[See image.](#delete-icon-network-decoy)
- In the confirmation window, click **OK**.
- [Start the decoys](/deception/starting-and-stopping-decoys#start-decoys).
