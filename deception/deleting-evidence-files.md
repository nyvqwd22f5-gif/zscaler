# Deleting Evidence Files
Source: https://help.zscaler.com/deception/deleting-evidence-files
PDF: https://help.zscaler.com/pdf/com/en/1531396.pdf

You can delete evidence files from the Zscaler Deception Admin Portal.
Deleting Selected Evidence Files
To delete selected evidence files:
- Open the [extended details](/deception/viewing-extended-details) page.
- Click **Evidence**.
- Select the evidence files that you want to delete.
-
Click **Actions** > **Delete**.
[See image.](#delete-selected-evidence-files)
-
In the confirmation window, click **OK**.
The selected evidence files are deleted.
Deleting All Evidence Files
To delete all evidence files:
- Open the [extended details](/deception/viewing-extended-details) page.
- Click **Evidence**.
-
Click **Actions** > **Delete All**.
[See image.](#delete-all-evidence-files)
All evidence files for the specified time period on the [Zscaler Deception Dashboard](/deception/viewing-and-managing-zscaler-deception-dashboard) are permanently deleted from the Decoy Connectors.
-
In the confirmation window, click **OK**.
All the evidence files are deleted.
