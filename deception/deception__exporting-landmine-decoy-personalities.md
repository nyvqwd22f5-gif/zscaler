# Exporting Landmine Decoy Personalities
Source: https://help.zscaler.com/deception/exporting-landmine-decoy-personalities
PDF: https://help.zscaler.com/pdf/com/en/1531536.pdf

You can export a ZIP file containing all landmine decoy personalities, each in its own JSON file.
To export landmine decoy personalities:
- Go to **Miragemaker **> **Strategy Builder **> **Landmine**.
-
Click **Export**.
[See image.](#zd-export-landmine-decoy-personalities)
All landmine decoy personalities are downloaded to your system in a compressed (.zip) file.
