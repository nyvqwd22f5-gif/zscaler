# Downloading Server Agent Logs
Source: https://help.zscaler.com/itdr/downloading-server-agent-logs
PDF: https://help.zscaler.com/pdf/com/en/1531866.pdf

This article provides instructions on how to download the server agent logs. After you download the logs, you can send them to Zscaler Support to troubleshoot issues.
To download server agent logs:
- Go to **Settings **> **Server Agent Settings **> **Agents**.
- In the **Agent details** table, select a server agent for which you want to download the logs.
-
Click **Actions** > **Trigger Logs**.
[See image.](#trigger-option-action-menu)
After 15 minutes, a **Download** icon appears under the **Actions** column of the selected server agent.
[See image.](#download-agent-logs-icon)
To check when the server agent logs were uploaded to the ITDR Admin Portal, hover over the **Download **icon.
[See image.](#zd-agent-log-tooltip)
-
Click the **Download **icon.
The server agent logs are downloaded to your local system in a compressed (.zip) file.
