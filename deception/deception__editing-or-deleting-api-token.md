# Editing or Deleting an API Token
Source: https://help.zscaler.com/deception/editing-or-deleting-api-token
PDF: https://help.zscaler.com/pdf/com/en/1531429.pdf

You can edit or delete an API token that you configured in the Zscaler Deception Admin Portal.
Editing an API Token
- Go to **Orchestrate** > **API Tokens**.
- Select the API token you want to modify, and under **Actions**, click the **Edit** icon.
[See image.](#edit-api-token-option)
- In the **API Token Details **window, modify the required fields.
[See image.](#deception-edit-api-token-details)
You cannot edit the API token expiration period.
- Click **Submit**.
Deleting an API Token
- Go to **Orchestrate** > **API Tokens**.
- Select the API token you want to delete, and under **Actions**, click the **Delete** icon.
[See image.](#deception-delete-api-icon)
- In the confirmation window, click **OK**.
