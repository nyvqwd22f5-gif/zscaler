# About File Templates
Source: https://help.zscaler.com/deception/about-file-templates
PDF: https://help.zscaler.com/pdf/com/en/1531448.pdf

Miragemaker file templates are a set of files that you can use as a baseline when creating file datasets.
File templates provide the following benefits and enable you to:
- [Add dynamic file datasets](/deception/adding-dynamic-file-dataset) quickly using the predefined file templates.
- Configure custom file templates per your requirements.
About the File Templates Page
On the File Templates page (Miragemaker > Files > File Templates), you can do the following:
- View a list of predefined and custom file templates.
- [Add a custom file template](/deception/adding-custom-file-template).
- [Download a file template](/deception/downloading-file-template).
